 ⑵ 地质时代名称

 ⑶ 寒武纪（前寒武纪）

> ① 成冰期

 ⑷古生代**：**5.8亿年前~2.25亿年前

> ①古生代时代特征

>> ○ 标准索引化石 **：** 三叶虫、腕足动物、节肢动物、有孔虫（放射虫）

>> ○ 陆生植物最早出现于古生代

>> ○ 助记符 **：** **Kam** 那里 **o**nly **de**solate，**Seok**tan **Pe**ace

> ② 寒武纪 **：** 腕足动物和三叶虫的首次出现

>> ○ 三叶虫 **:** 在 **古生代** 蓬勃发展

> ③ 奥陶纪 **：** 节肢动物的首次出现

> ④ 志留纪 **:** 菊石和角石的首次出现

>> ○ 中生代丰富的菊石，标准指标化石

> ⑤ 泥盆纪

> ⑥ 石炭纪 **:** 有孔虫的首次出现

> ⑦ 二叠纪（Cisuralian Epoch） **：** 又称“二叠纪”

>> ○ 两栖动物时代，与盘古大陆的形成有关>> ○ 二叠纪-三叠纪灭绝事件 **：** 地球历史上最大规模的物种灭绝

>>○金钱页岩**：**代表古生代结束，但代表中生代

 ⑸中生代**：**爬行动物时代，又称恐龙时代。 2.25亿年前~6500万年前

> ①中生代时代特征

>> ○ 标准索引化石 **：** 菊石、箭石、恐龙、始祖鸟

>> ○ 普遍比古生代或新生代温暖

> ② 三叠纪

> ③侏罗纪时期**：**小型爬行动物的时代。始祖鸟化石

> ④白垩纪

 ⑹新生代**：**6500万年前~现在

> ①新生代特征

>> ○ 新生代分为始新世和渐新世

>> ○ 新生代被子植物繁盛

> ②始新世（6500万年前~200万年前）

>> ○ 古新世

>> ○ 始新世

**3\.韩国地质**

  ⑴ 朝鲜半岛地质构造

> ① 由约10个隆起、沉积盆地和褶皱山脉组成

> ② 隆起 **：** 无区域

-岩石的方向分布

>> ○ 主要由寒武纪形成的变质岩组成

>> ○ 由于过去的造山运动而暴露

> ③ 沉积盆地 **：** 隆起之间的沉积岩层

>> ○ 平安盆地 **:** 由古生代沉积岩组成

>> ○ 庆尚盆地 **:** 由中生代白垩系沉积岩和火山岩组成

>> ○ 太白山盆地 **:** 由古生代和中生代沉积岩组成，未经变质作用

> ④ 复杂褶皱 **：** 岩石因褶皱或断层而发生复杂变形的区域

  ⑵ 韩国地质分布

> ① 按岩石类型分布

>> ○ **变质岩** **：**约占面积的40%

>>> ○ 多形成于寒武纪

>>> ○（注）大多数古老的岩石都是变质的，因为火成岩和沉积岩最终都会变质

>>> ○（参考）古生代形成的岩石大多是变质岩

>> ○ **火成岩** **：** 约占面积的35%

>>> ○ 大多在中生代时期侵入

>> ○ **沉积岩** **:** 约占面积的25%

>>> ○ 多形成于古生代海洋和中生代陆地

>>> ○（参考）古生代形成的岩石大多是变质岩

> ② 按地质时代分布 **:** 寒武纪 ＞ 中生代 ＞ 古生代 ＞ 新生代

>> ○ 寒武纪岩石 **：** 约43%。分布于京畿道、江原道北部。变质岩

>> ○ 古生代岩石 **：** 约11%。分布于忠清道、全罗北道、庆尚北道。沉积岩（河城组）

>> ○ 中生代岩石**：** 约40%。分布于全罗南道、庆尚南道。沉积岩（陆地形成）+火成岩（侵入岩）

>> ○ 新生代岩石 **：** 约 6%。分布于济州岛、郁陵岛、独岛、白头山。沉积岩（陆地和海洋）+火成岩（火山）

>> ○ 中生代侵入 **:** 从江原道斜向下

>> ○ **朝鲜集团** **:** 由石灰岩、砂岩、页岩组成的厚海层

>>> ○ 三叶虫、介形类、腕足类、牙形刺化石

>>> ○ 存在非煤层。包含海洋和陆地岩层

>>> ○ 发现蕨类植物化石

>> ○ **特征2.** 大部分为海相岩石 **：** 最终在晚古生代转变为陆相岩石

>>> ○ 陆岩时代 **:** 平安期晚期 ~

> ③中生代地质特征

>> ○ 松林运动

>> ○ **大同群** **：**早中生代陆地岩层。含有银杏、非煤层

>>> ○ 三叠纪晚期 ~ 侏罗纪早期

>> ○ 大波山造山运动 **:** 侏罗纪晚期的地质运动

>>> ○ 韩国在大宝山造山运动期间经历了重大的构造运动

>>> ○ 这次运动导致侏罗纪早期岩石发生剧烈变形

>>> ○ 造成地质构造东西走向

>>> ○ 大宝山造山运动使大东群变形

>>> ○ 大宝山造山运动并未使庆尚集团变形

>> ○ **庆尚群** **:** 晚白垩世庆尚盆地河流和湖泊环境中形成的沉积岩

>>> ○ 恐龙脚印、鸟类脚印、淡水蛤化石

>>> ○ (注) 因位于庆尚南道而被命名为庆尚集团

>> ○ 佛国寺造山运动

>>> ○ 与晚白垩世恐龙灭绝有关的构造运动

>> ○ **特点1.** 恐龙是代表性化石

>> ○ **特征2.** 高侵入**：** 主要发生在晚侏罗世

>>> ○ 以粘性火成岩为主，以花岗岩为主

>>> ○ 中生代是朝鲜半岛构造活动显着的时期（如大同造山运动、佛国寺运动等）

>>> ○ 韩国的煤炭大部分形成于中生代

>> ○ **特征3.** 主要是**陆地岩层**

>>> ○ 河流或湖泊中形成的岩层

>>> ○ 平安安群晚期 ~ 白垩纪晚期

>> ○ **特征4.** 庆尚道的岩层大部分为中生代

> ④ 新生代地质特征

>> ○ **功能 1.** **火山活动**活跃

>>> ○ 沿韩国东海岸分布，形成小阵型 **：** 白头山、济州岛、独岛

>>> ○ 以低粘度火山岩为主，以玄武岩为主

>> ○ 大约2500万年前 **:** 太平洋板块开始俯冲到日本列岛下方，导致东海扩张

>> ○ 新生代第三纪 **:** 白头山火山场

>>> ○ 晚新生代第三纪（新近纪） **:** 活跃沉积

>> ○ 新生代第四纪 **:** 济州岛独岛的形成

>>> ○ 独岛的形成 **:** 初次喷发 → 二次喷发 → 风化 → 海平面上升

>>> ○ 济州岛主峰山脊 **:** 由晚新生代玄武岩组成

>>> ○ 济州岛的万丈窟 **:** 当高度流动的熔岩流动时形成，留下一个空心结构>> ○ 末次冰期末 **:** 黄海形成
