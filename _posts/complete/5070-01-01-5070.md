## **1.基于能量代谢的生物体分类**

⑴ 按能源分类

> ① **第 1 类.** 光养生物：利用光作为能源的生物体。分为光合自养生物和光合异养生物。

> ② **第 2 类.** 化学营养生物：使用化学能作为能源的生物体。分为化能自养生物和化能异养生物。

>> ○ 蓝藻_O。当 H<sub>2</sub>S 不存在时，limnetica_ 使用 H<sub>2</sub>O 作为初始电子供体；当存在 H<sub>2</sub>S 时，limnetica_ 使用 H<sub>2</sub>S 作为初始电子供体。

>> ○ 对于蓝藻_O。 limnetica_，使用时能源为H<sub>2</sub>S，否则为光。

 ⑵ 按碳源分类

> ① **第 1 类.** 自养生物：以 CO<sub>2</sub> 作为碳源的生物。分为光合自养生物和化能自养生物。

> ② **2.** 异养生物：以有机化合物作为碳源的生物。分为光合异养生物和化能异养生物。

 ⑶ 摘要：按能源、碳源列出时，

> ①光合自养生物：光+CO<sub>2</sub>

> ② 光异养生物：光+有机化合物

> ③化能自养生物：无机物+CO<sub>2</sub>

> ④ 化能异养生物：有机化合物+有机化合物

## 2. ATP 和电子载体

 ⑴ ATP（三磷酸腺苷）

> ① 通过磷酸化引起蛋白质构象变化，以进行机械功、物质运输和化学反应（偶联反应分子）。

> ② 结构

>> ○ 腺嘌呤 + 核糖 + 3 个磷酸分子

>> ○ 高能磷酸键：由于带负电的磷酸盐的静电排斥作用，其能量含量高且不稳定，容易水解释放大量能量（7.3 kcal/mol）。

>> ○ ATP ⇄ ADP + P<sub>i</sub>（磷酸盐），K = 1.5 × 10<sup>5</sup>

> ③ 磷酸化的类型

>> ○ **类型 1.** 底物水平磷酸化：[糖酵解](https://jb243.github.io/pages/1410) 

>> ○ **2型.** 光磷酸化：[细菌视紫红质](https://jb243.github.io/pages/69) 

>> ○ **类型 3.** 氧化磷酸化：通过电子传递链进行磷酸化

 ⑵ NAD<sup>+</sup>（烟酰胺腺嘌呤二核苷酸）：氧化反应中的辅酶，参与细胞呼吸。

> ① 腺嘌呤 + 2 个核糖 + 烟酰胺基团

>> ○ 源自烟酸（维生素 B<sub>2</sub> 衍生物）

> ② NAD<sup>+</sup> + H<sub>2</sub> ⇄ NADH + H<sup>+</sup>

>> ○ 1 个 NAD<sup>+</sup> 分子携带 1 个氢原子和 1 个电子。

>> ○ 因此，它携带 1 个氢离子和 2 个电子。

> ③机理：亲核芳香加成反应和氢化物移位

## 3. 细胞呼吸⑴ 细胞呼吸概述

> ① 外呼吸：通过肺部吸入O<sub>2</sub>并释放CO<sub>2</sub>。

> ② 内部呼吸：组织细胞利用O<sub>2</sub>进行细胞呼吸，并释放CO<sub>2</sub>。

> ③ 在细胞呼吸中，葡萄糖氧化成CO<sub>2</sub>，O<sub>2</sub>还原成H<sub>2</sub>O，合成ATP。

⑵ 有氧呼吸：O<sub>2</sub> 用作电子传递链中的**最终电子受体**。

>> ○ 电子传递链的氧化还原电位

>>> ○ 还原能力：还原其他物质的倾向。低还原能力意味着有自行还原的倾向。

>>> ○ 还原电位：对于趋于还原的化学物质，较高的标准还原电位值。

>>> ○ 吉布斯自由能变化：化学物质越喜欢电子，自由能变化就越负。

>>> ○ 结论：随着反应过程的进行，还原力降低，还原电位增大，自由能变化ΔG变得更负。

>>> ○ 要发生化学反应，ΔG ＜ 0，因此随着过程的进行，ΔG 必须变得更负： ΔG<sub>earlier_step</sub> + ΔG<sub>latter_step</sub> ＜ 0

>>> ○ 通过使用ΔG = -nFE，可以很容易地理解还原电位和ΔG 之间的关系。

> ④ 细胞呼吸计算

>> ○ **产量 1.** 大约 1 NADH → 3 ATP, 1 FADH<sub>2</sub> → 2 ATP

>> ○ **产量2.** 对于原核生物、真菌、藻类等，产量较低：1 NADH → 2.5 ATP，1 FADH<sub>2</sub> → 1.5 ATP

>>> ○ 当电子穿过配合物 I 和 III 时释放 4H<sup>+</sup>，当电子穿过配合物 IV 时释放 2H<sup>+</sup>。

>>>> ○ 复合体IV更接近ATP合酶，结构变化困难，因此产量较低。

>>> ○ 合成 1 个 ATP 需要 4H<sup>+</sup>。

>>>> ○ 3H<sup>+</sup> 移动时 ATP 合成酶转一圈。

>>>> ○ 当丙酮酸进入内膜时，H<sup>+</sup> 被消耗。

>>> ○ NADH 产量 = 移动的总 H<sup>+</sup> ÷ (4 H<sup>+</sup> / 1 ATP) = (4 × 2 + 2) ÷ 4 = 2.5

>>> ○ FADH<sub>2</sub> 产量 = H+ 移动总量 ÷ (4 H<sup>+</sup> / 1 ATP) = (4 × 1 + 2) ÷ 4 = 1.5

>> ○ 1分子葡萄糖通过糖酵解转化为2分子丙酮酸，产生2 ATP和2 NADH。

>> ○ 2丙酮酸进入线粒体，进行氧化和柠檬酸循环，产生2 ATP + 8 NADH + 2 FADH<sub>2</sub>。

>>> ○ **提示：** CO<sub>2</sub> 释放会产生 NADH。

>>> ○ **提示：** 除了苹果酸转化为 OAA 之外，NADH 的形成还伴随着 CO<sub>2</sub> 的产生。

>> ○ 胞质 NADH 必须穿过线粒体内膜的穿梭机，并可能转化为 FADH<sub>2</sub>。

>> ○ 肝、心、肾：共产生**32 ATP**（38 ATP，按产量1计算）。与苹果酸-天冬氨酸穿梭相关。

>> ○ 骨骼肌、大脑等：总共产生**30 ATP**（36 ATP，以产量1计算）。与磷酸甘油穿梭有关。

>>> ○ 之前计算为 32 ATP。

>>> ○ 尽管效率较低，但骨骼肌和大脑利用磷酸甘油穿梭来快速获取能量。>> ○ 实际上，当游离磷酸盐穿过线粒体内膜时，也会使用 1 H<sup>+</sup>。

⑸ 其他葡萄糖分解代谢途径

> ① EMP（Embden-Meyerhof-Parnas）途径：[糖酵解](https://jb243.github.io/pages/1410)

>> ○ 生成 ATP 的反应。

>> ○ 化学反应式：C<sub>6</sub>H<sub>12</sub>O<sub>6</sub> + 2ADP + 2P<sub>i</sub> + 2NAD<sup>+</sup> → 2C<sub>3</sub>H<sub>4</sub>O<sub>3</sub> + 2ATP + 2NADH + 2H<sup>+</sup>> ② TCA循环

>> ○ 生成 ATP 的反应。

>> ○ 化学反应式：C<sub>3</sub>H<sub>4</sub>O<sub>3</sub> + 3H<sub>2</sub>O + ADP + P<sub>i</sub> + 4NAD<sup>+</sup> + FAD → 3CO<sub>2</sub> + 4(NADH + H<sup>+</sup>) + FADH<sub>2</sub> + ATP

> ③ HMP（己糖单磷酸途径）

>> ○ 又称PPP（磷酸戊糖途径）、戊糖分流、Warburg-Dickens 途径。

>> ○ 产生合成代谢过程（例如脂肪酸合成）所需的 NADPH 的反应。

>> ○ 化学反应式：C<sub>6</sub>H<sub>12</sub>O<sub>6</sub> + 12NADP<sup>+</sup> + 3O<sub>2</sub> + ATP → 6CO<sub>2</sub> + 12NADPH + ADP + P<sub>i</sub>

>>> ○ 6-磷酸葡萄糖可完全还原为二氧化碳，生成12分子NADPH。

>>> ○ 该反应可以合成不同碳数的糖。

>>> ○ 从摄入的核酸分解中获得的戊糖也通过 HMP 途径进行代谢。

>>> ○ PEP 羧化酶比 Rubisco 具有更高的 CO<sub>2</sub> 亲和力。

>>> ○ 束鞘细胞中不存在 PEP 羧化酶。

>> ○ 第三<sup>第</sup>。还原的苹果酸 (4C) 移动到内部（**束鞘细胞**），释放 CO<sub>2</sub>。

>> ○ 第 4<sup></sup>。束鞘细胞中的 Rubisco 参与卡尔文循环，并在基质中提供二氧化碳和草酰乙酸。

>>> ○ Rubisco 不存在于叶肉细胞中。>> ○ 第五<sup>th</sup>。苹果酸释放CO<sub>2</sub>再次生成NADPH并产生丙酮酸(3C)：不需要光系统II的原因。

>> ○ 第六<sup>th</sup>。丙酮酸利用 ATP 产生 PEP。

>> ○ 7<sup>th</sup>。 PEP 转移至叶肉细胞。

>> ○ 替代途径：草酰乙酸通过天冬氨酸而不是苹果酸。

> ④ CAM植物：极端炎热干燥环境下的植物、多肉植物（如景天、仙人掌、菠萝等沙漠植物）

>> ○ 光系统 II 存在，但在容易发生光呼吸的条件下其活性会降低 ([ref](https://www.frontiersin.org/articles/10.3389/fpls.2019.01012/full))。

>> ○ 仅循环光磷酸化，低补偿点，碳固定**暂时**分离。

>> ○ 第一<sup>第一</sup>。白天，气孔完全关闭以保存水分。发生暗反应但效率较低。

>> ○ 第二<sup>第二</sup>。夜间，气孔打开，在 C<sub>4</sub> 循环中利用 CO<sub>2</sub> 产生苹果酸。

>>> ○ 第二<sup>第二</sup> - 第一<sup>第一</sup>。 PEP 羧化酶将 CO<sub>2</sub> 与 PEP 结合，产生草酰乙酸。 **初始碳固定产品**。

>>> ○ 第二<sup>第二</sup> - 第二<sup>第二</sup>。草酰乙酸被还原为苹果酸。

>> ○ 第三<sup>第</sup>。产生的苹果酸储存在液泡中。

>> ○ 第 4<sup></sup>。白天，卡尔文循环使用从苹果酸分解中获得的 CO<sub>2</sub> 进行运行。

>>> ○ 第 4<sup></sup> - 第 1<sup></sup>。苹果酸 → CO<sub>2</sub> + NADPH + 丙酮酸 (3C)

>>> ○ 第 4<sup></sup> - 第 2<sup></sup>。丙酮酸 (3C) 使用 ATP 转化为 PEP 或在 TCA 循环中利用以产生 ATP。

>>> ○ 第 4<sup></sup> - 第 3<sup></sup>。 CO<sub>2</sub> 移动到基质，与草酰乙酸和 Rubisco 一起参与卡尔文循环。
