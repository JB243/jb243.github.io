## 1.循环系统

⑴ 循环系统的类型

> ① 开放式循环系统

>> ○ **生物**：节肢动物和软体动物

>> ○ **简单**：无毛细管；血淋巴和间质液之间没有明显区别

>> ○ **效率**：低液压节省能源；无需形成毛细血管网络，更容易构建和维护循环系统

> ② **闭式循环系统**

>> ○ **生物**：环节动物、头足类动物、脊椎动物和其他大型或高度流动的生物体

>> ○ **复杂性**：毛细血管的存在区分了血液和间质液

>> ○ **效率**：高液压可有效输送氧气和营养物质；渗透压由血液中的大分子维持，有助于调节血压

⑵ 脊椎动物的循环系统（心血管系统）——闭式循环系统

> ① 代谢率高的动物比代谢率低的动物拥有更复杂的血管和更强大的心脏

> ② 生物体内血管的复杂程度和分布与各器官的代谢活动成正比

> ③ **鱼（1 个心房，1 个心室）** – *单循环*

>> ○ 体循环与肺循环不分离；血液在返回心脏之前必须经过两个毛细血管床，从而限制了血流速度。

>> ○ 通过骨骼肌活动维持必要的血流。

>> ○ 只有脱氧的血液流经心脏。

> ④ **两栖动物（2 个心房，1 个心室）** – *双循环*

>> ○ 体循环和肺循环分开，可以让更多的血液供应到大脑和肌肉；然而，含氧血液和脱氧血液的混合会降低效率

>> ○ 为了补偿较低的氧气效率，开发了皮肤循环系统

> ⑤ **爬行动物（2 个心房，部分分开 2 个心室）** – *双循环*

>> ○ 含氧血液和脱氧血液混合较少，提高物质交换效率

> ⑥ **鸟类和哺乳动物（2 个心房，完全分开 2 个心室）** – *双循环*

>> ○ 含氧血液和脱氧血液完全分离，实现高效的物质交换

>> ○ 该系统支持恒温动物，其消耗的能量比相同体型的变温动物多约 10 倍

> ⑦ **人体循环系统**

>> ○ **体循环**：左心室→主动脉→全身（毛细血管）→腔静脉→右心房

>> ○ **肺循环**：右心室→肺动脉→肺（毛细血管）→肺静脉→左心房

## **2\.血液成分和功能**

⑴ 血液离心：按成分分离血液

> ①抗凝离心：从下到上分离红细胞、软层、血浆

>> ○ 软层（血沉棕黄层）：白细胞+血小板

>> ○ 血浆：血液的液体成分

> ② 不加抗凝剂离心：从下到上分离为血凝块和血清

>> ○ 血栓：凝固血成分，红细胞+白细胞+血小板+凝血因子>> ○ 血清：血液凝固后剩余的液体成分，从血浆中除去纤维蛋白原和其他成分

>> ○ 细胞成分被血浆中的纤维蛋白原缠绕

> ③红细胞体积分数（血细胞比容）：红细胞体积与总血容量的比值

>> ○ 正常成年男性：0.41至0.51

>> ○ 正常成年女性：0.36 至 0.45

>> ○ 为什么男性比女性更多地使用权力

>> ○ 贫血者血细胞比容小

>> ○ 粘度与红细胞体积分数成正比

⑷ 细胞成分：白细胞（白细胞）

> ① 源自骨髓干细胞，每1毫米5,000至10,000个<sup>3</sup>

> ②寿命：100~200天

> ③ 特点

>> ○ 与红细胞不同，它们也存在于细胞内和淋巴液中

>> ○ 存在细胞核并用于核型分析

>> ○ 形状不恒定和阿米巴运动

>> ○ 蜂群：组织内自由运动

> ④ 产生：由内皮细胞、骨髓成纤维细胞等白细胞产生的集落刺激因子（CSF）产生

> ⑤ 粒细胞：吞噬作用、化脓、过敏反应、炎症反应

>> ○ 主要特点：多形核和颗粒多，密度高

>> ○ 根据粒细胞染色染料分为中性粒细胞、嗜酸性粒细胞、嗜碱性粒细胞

>> ○ 中性粒细胞 

>>> ○ 功能：抗体包被的病原体吞噬作用

>>> ○ 分布：占白细胞总数的60-65%。 12-14微米

>>> ○ 颗粒形状：形成三块

>> ○ 嗜酸性粒细胞

>>> ○ 功能：抗体包被的寄生虫死亡和过敏性超敏反应

>>> ○ 分布：占白细胞总数的1%

>>> ○ 形状：形成2块。颗粒比较大。 12 至 17 微米

>>> ○ 被曙红染成红色。

>> ○ 嗜中性粒细胞（嗜碱性粒细胞）

>>> ○ 功能：组胺和肝素释放促进T淋巴细胞发育

>>> ○ 分布：占白细胞总数的0.2%。

>>> ○ 形状：分散颗粒、大颗粒、14 ~ 16 ㎛

>>> ○ 亚甲蓝染成深紫色

> ⑥ 肥大细胞：因损伤或抗原结合而释放组胺、白三烯等

>> ○ 指作用于组织细胞的嗜碱性白细胞

> ⑦ 单核细胞：吞噬细胞。它们约占所有白细胞的 4%。尺寸：~20μm

>> ○ 单核细胞起源于骨髓中的造血干细胞。

>> ○ **1 型：** 巨噬细胞

>> ○ **2型：**异物巨细胞

>> ○ 单核细胞分化为巨噬细胞和树突状细胞。

> ⑧ 巨噬细胞

>> ○ 吞噬消化微生物，呈递抗原，激活T淋巴细胞

>> ○ 大脑中的小胶质细胞也是巨噬细胞的一种

>> ○ **分类1：**根据功能

>> ○ **1-1.** M1 巨噬细胞（经典激活或炎症巨噬细胞）

>>> ○ **促炎细胞**：参与细胞死亡和抗肿瘤活性。

>>> ○ **伸长系数**：长轴与短轴之比接近1。

>>> ○ **诱导 M1 型的细胞因子**：TLR、TNF-α、IFN-γ、CSF2、LPS、STAT1、IRF5、IL-17A>>> ○ **M1 型分泌的细胞因子**：IL-6、IL-8、IL-23p40、TNF-α、IL-1β、IL-12p70、IL-12p40、IFN-γ

>> ○ **1-2。 M2 巨噬细胞（选择性激活或抗炎巨噬细胞）**

>>> ○ **抗炎细胞**：参与组织修复和促肿瘤活性。

>>> ○ **伸长系数**：长轴与短轴的比率高。

>>> ○ **诱导 M2 型的细胞因子**：IL-4、IL-10、IL-13、TGF-β、PGE2、STAT3、STAT6、IRF4

>>> ○ **M2 型分泌的细胞因子**：IL-10

>>> ○ **M2型基因标记**：CD68、CD163、CD204、CD206、VEGF、cMAF、ARG1、YM1、CCL20、CCL22、IDO1

>>> ○ 大多数肿瘤相关巨噬细胞 (TAM) 属于 M2 型。

>> ○ **分类 2：** 基于组织位置

>>> ○ **库普弗细胞**：位于肝毛细血管中的巨噬细胞。

>>>> ○ 先天免疫系统的一部分，称为网状内皮系统 (RES)。

>>>> ○ 唯一位于血管内的巨噬细胞。

>>> ○ **脾细胞**：位于脾脏中。

>>> ○ **BMDM（骨髓源性巨噬细胞）**：位于骨骼中。

⑵ 心肌

> ① 特点

>> ○ 毛细血管和线粒体密度高：有氧呼吸 

>> ○ 因肌红蛋白含量高而呈红色

>> ○ 营养：脂肪酸 ≫ 葡萄糖、乳酸（∴ 仅有氧呼吸）

>> ○ 冠状动脉：供应心肌的血液，心肌供血不足会导致心脏骤停、心脏病发作、心肌梗塞

> ② 自律细胞的起搏电位

>> ○ 自律细胞：窦房结（SA）（位于右心房）、房室（AV）结（位于右心房）、浦肯野纤维

>> ○ SA结在副交感神经刺激下延长舒张期并降低心率

## **4\.血管系统**

⑴血管

<br>

<img src="https://img1.daumcdn.net/thumb/R1280x0/?scode=mtistory2&fname=https://blog.kakaocdn.net/dn/pjYzh/btrz3O4LZmT/0xfpIqzuog5pCKbYbR7USK/img.jpg" alt="绘图" />

<b>图10.</b>血管的结构

<br>

> ①动脉：由三层（内膜、中膜、外膜）、许多弹力纤维和肌纤维（平滑肌）组成，主动脉（含11%血液）

>> ○ 弹性组织比例相对高于静脉 → 弹性↑

>> ○ 血管壁厚，可承受高血压

>> ○ 动脉壁的弹性可以让动脉在心脏放松时恢复到原来的状态并维持高血压

>> ○ 主动脉自行调节收缩和舒张

>> ○ 小动脉的收缩和扩张受自主神经系统的调节。

> ② 静脉：由三层组成（内膜、中膜、外膜）。存在瓣膜和肌纤维（平滑肌）。腔静脉和小静脉（61% 血液）

>> ○ 比动脉更粗，直径更大，容纳循环系统一半以上的血液

>> ○ 血管中半径最大，血流阻力最小

>> ○ 壁比动脉更薄且弹性更小，并且由于血压较低，因此存在瓣膜以防止回流。

>> ○ 除平滑肌外，邻近的骨骼肌也有调节作用

> ③ 毛细血管

>> ○ 毛细血管壁由一层非常薄的上皮细胞组成。无平滑肌

>> ○ 毛细孔（开口）：内皮细胞管中的孔。不在严格控制BBB的大脑中

>> ○ 基底层：被内皮细胞包围，易于渗透，有利于物质交换。>> ○ 疏水性物质和小的亲水性物质：通过简单扩散穿过内皮细胞膜。例子包括二氧化碳、水、葡萄糖和氨基酸。

>> ○ 大亲水物质：通过转胞吞作用进行交叉。在肝脏和肠道中，内皮细胞之间的间隙足以让蛋白质通过。

>> ○ 距离毛细血管100μm以上的细胞会发生细胞死亡。

>> ○ 肺泡毛细血管的直径约为8μm。
