## **1.肌肉**

 ⑴组成部分：动物肌肉的组成如下：

> ① 水：70%

> ② 肌球蛋白、肌动蛋白、肌红蛋白等蛋白质：20%

 ⑵ 肌肉与骨骼的相互作用

> ① 肌肉系统的功能是运动，它与骨骼相互作用

> ② 肌肉通过肌腱通过关节与骨骼相连。

> ③ 肌肉只能收缩 **→** 合作肌（同向收缩）和拮抗肌（相反方向收缩）共同作用

>> ○ “肌肉只能收缩”的说法是指没有机制在放松方向产生张力

>> ○ 收缩的肌肉自然会放松到平衡状态

 ⑶ 分类

> ① 骨骼肌（如舌头、括约肌、膈肌）

>> ○ 结构：圆柱形**多核**细胞，交叉条纹（横条纹，发达的肌节）

>>> ○ 多核细胞是骨骼肌的独特特征

>> ○ 特点：无分级、肌浆网丰富、钙肌钙蛋白结合

>> ○ **弗兰克-斯塔林定律**：该定律指出肌肉张力受肌肉长度影响

>>> ○ 当肌动蛋白和肌球蛋白丝最大程度重叠时，可获得最大张力

>>> ○ 肌节长度长时：肌肉收缩时，肌球蛋白跨桥数量增加，导致总张力增加

>>> ○ 当肌节长度较短时：随着肌节长度变得比肌肉的有效长度短，肌动蛋白和肌球蛋白丝之间的重叠减少，导致张力降低。如果它们不再重叠，就无法产生张力

>> ○ **破伤风 可能：** 骨骼肌是三种肌肉中唯一具有破伤风能力的肌肉。

>>> ○ 抽搐：一种具有恒定大小和形状的一次性肌肉收缩模式，类似于动作电位

>>> ○ 破伤风：短时间重复电刺激导致骨骼肌持续收缩而不松弛

>>> ○ 破伤风 可能：骨骼肌不应期短 → 高频刺激 → 细胞内 Ca<sup>2+</sup> 积聚 → 重叠收缩

>>> ○ 未融合破伤风：收缩时相对低频刺激，导致肌肉收缩不均匀

>>> ○ 融合破伤风：收缩时相对高频的刺激，导致平滑肌和相连肌收缩

>> ○ 肌肉收缩调节：自愿调节（自愿控制）

>>> ○ 受躯体运动神经系统调节，与自主神经系统不同，不涉及神经节>>> ○ 仅收缩和兴奋

>>> ○ 神经肌肉接头处的乙酰胆碱分泌导致钙与肌钙蛋白结合，引发肌肉收缩

>> ○ 肌生长抑制素

>>> ○ 抑制骨骼肌的发育和生长

>>> ○ 参与韧带和肌腱的形成和发育

> ② 心肌（心肌）

>> ○ 结构：**分枝**形状、闰盘、交叉条纹（发达的肌节）、肌肉细胞之间的间隙连接和粘附连接

>>> ○ 闰盘包含间隙连接，可实现同步信号传输。

>>> ○ 闰盘也含有桥粒。

>>> ○ 由于细胞与细胞粘附连接，收缩同时发生→所有细胞充当单个功能单元。

>> ○ 特点：分级、大量线粒体、钙肌钙蛋白结合、高静息电位，小刺激即可启动动作电位

>> ○ 不可能破伤风：在心肌的平台期，会出现类似于绝对不应期的现象，使得重叠的肌肉收缩不可能。

>> ○ **收缩调节**：非自愿调节（非自愿控制）

>>> ○ 在起搏器中，窦房结 (SA) 受自主神经系统调节。

>>> ○ 交感神经系统增加Ca²⁺通透性，提高SA结的起搏速率，而副交感神经系统增加K⁺通透性，降低起搏速率。

>>> ○ 心肌细胞仅受交感神经系统调节（关于每搏输出量是否增加）。

> ③ 平滑肌（内脏肌）（例如消化道、血管壁、子宫）

>> ○ 结构

>>> ○ 肌肉细胞之间存在间隙连接

>>> ○ 钙不直接与肌钙蛋白结合；钙调蛋白参与

>>> ○ 纺锤形单核细胞

>>> ○ 无肌节

>> ○ 特征：分级、不发达的T型管

>>> ○ 由于缺乏发育良好的 T 管，收缩和舒张缓慢

>> ○ 单单位平滑肌和多单位平滑肌存在：单单位平滑肌较常见

>> ○ **收缩调节**：非自愿调节（非自愿控制）

>>> ○ 交感神经系统：内脏肌松弛，小动脉收缩，气道收缩

>>> ○ 副交感神经系统：内脏肌收缩、小动脉舒张、气道舒张

>>> ○ 肌球蛋白磷酸化（激活收缩）和去磷酸化（抑制收缩）

>>> ○ 主动脉血管平滑肌通过自身的牵张感受器进行自我调节

## **2.骨骼肌收缩模型**

 ⑴ 肌肉收缩的调节

> ① 骨骼肌受躯体运动神经系统而非自主神经系统调节

> ② 当躯体运动神经系统分泌乙酰胆碱时，钙与肌钙蛋白结合，引发肌肉收缩

>> ○ 请注意，肌钙蛋白既不是肌动蛋白也不是肌球蛋白。

> ③滞后时间：大脑反应开始与肌肉反应开始之间的时间差

 ⑵ 肌球蛋白的酶促作用> ① 肌球蛋白由两条重链和两条轻链组成

> ② 重链

>> ○ 重链尾部（150 nm）盘绕成超螺旋，形成定向肌球蛋白丝。

>> ○ 重链的头部（17 nm）包含肌动蛋白结合位点和 ATP 酶位点，提供滑动的驱动力。

> ③ 轻链

>> ○ 附着在重链的头部，调节 ATP 酶活性。

 ⑶ 肌肉收缩的滑动丝模型：ATP分解导致两种机械作用

<br>

> ① 第一<sup>第一</sup>。静息状态下的肌肉（僵硬）：肌球蛋白头与肌动蛋白丝结合，无需 ATP 或 ADP

> ② 第二<sup>第二</sup>。 ATP 结合：肌球蛋白头与 ATP 结合，与肌动蛋白丝分离

> ③ 第三<sup>第</sup>。肌球蛋白头具有 ATP 酶活性，将 ATP 分解为 ADP 和无机磷酸盐

> ④ 第 4<sup></sup>。肌球蛋白头具有高能状态并采用多种构象。

> ⑤ 第 5<sup></sup>。跨桥形成：在采用各种构象的同时，肌球蛋白头部直立并与肌动蛋白丝结合。

>> ○ **[第一次机械动作]**

>> ○ 仍处于高能状态

>> ○ 如果肌球蛋白头不与肌动蛋白结合，则该过程无法继续进行下一步

> ⑥ 第六<sup>th</sup>。肌节收缩：肌球蛋白头最初处于高能状态，随着 ADP 和磷酸盐的释放而向下倾斜，转变为低能状态。在此过程中，肌动蛋白丝被拉向中心。

>> ○ **[第二机械动作]**

>> ○ 尽管处于低能构象，肌球蛋白仍然与肌动蛋白丝结合。

 ⑷钙介导的肌肉收缩控制：人体运动控制在5至10赫兹的范围内

> ① 第一<sup>第一</sup>。神经肌肉接头突触末端释放的乙酰胆碱触发动作电位

>> ○ 1<sup>st</sup> \- 1<sup>st</sup>。释放的乙酰胆碱扩散到突触间隙

>> ○ 第一<sup>第一</sup> \- 第二<sup>第二</sup>。在电机端板上感应出分级电势。

>> ○ 第一<sup>第一</sup> \- 第三<sup>第三</sup>。分级电位使肌纤维去极化，触发肌纤维中的动作电位。

> ② 第二<sup>第二</sup>。动作电位沿着细胞膜和 T 管传播。

>> ○ 横小管（T 型小管）：由肌细胞膜内陷形成的管状结构，将膜兴奋传递到细胞内部。

>> ○ T 管膜包含产生动作电位所需的电压门控 K<sup>+</sup> 通道。

> ③ 第三<sup>第</sup>。动作电位诱导肌浆网（一种光滑内质网）释放 Ca<sup>2+</sup>

>> ○ 第三<sup>rd</sup> – 1<sup>st</sup>。当动作电位到达 T 管膜（细胞膜）时，电压门控 Ca<sup>2+</sup> 通道被激活。

>> ○ 第三<sup>第三</sup> – 第二<sup>第二</sup>。在Ca<sup>2+</sup>浓度梯度的驱动下，Ca<sup>2+</sup>从细胞外移动到细胞质中。

>> ○ 第三<sup>rd</sup> – 第三<sup>rd</sup>。进入细胞质的 Ca<sup>2+</sup> 会打开肌浆网膜中 Ca<sup>2+</sup> 依赖性 Ca<sup>2+</sup> 释放通道。

>>> ○ 第三<sup>位</sup> – 第三<sup>位</sup> – 第一<sup>位</sup>。动作电位引起 T 管上二氢吡啶 (DHP) 受体的构象变化

>>> ○ 第三<sup>次</sup> – 第三<sup>次</sup> – 第二<sup>次</sup>。位于 T 管附近的肌浆网膜中的钙通道（兰尼碱受体）打开。>> ○ 第 3<sup></sup> - 第 4<sup></sup>。由于Ca<sup>2+</sup>浓度梯度，Ca<sup>2+</sup>从肌浆网膜移动到细胞质

>> ○ 由于钙是由肌浆网供给的，因此骨骼肌收缩不需要外部钙供给

> ④ 第 4<sup></sup>。当钙离子与细丝和肌钙蛋白结合时，肌球蛋白结合位点就会暴露。

>> ○ 钙离子不与肌动蛋白或肌球蛋白结合

>> ○ 肌钙蛋白复合物与原肌球蛋白和肌动蛋白丝结合
 
>> ○ 第 4<sup></sup> - 第 1<sup></sup>。 Ca<sup>2+</sup> 与肌钙蛋白复合物 C 亚基的结合

>> ○ 第 4<sup></sup> - 第 2<sup></sup>。原肌球蛋白位置移动

>> ○ 第 4<sup></sup> – 第 3<sup></sup>。肌动蛋白丝上肌球蛋白结合位点的暴露

⑸ 抽动肌肉纤维

> ①运动单位：单个运动神经元及其控制的所有肌纤维。

>> ○ LMN（下运动神经元）：支配骨骼肌的脊髓运动神经元。

>> ○ 抽搐：动作电位通过 T 管系统传播时发生的短暂收缩（收缩的最小单位）；遵循全有或全无法则。

>> ○ 破伤风：当动作电位达到足够高的频率时，纤维没有时间放松，抽搐最终产生连续的、持续的收缩；仅发生在骨骼肌中。

>> ○ 如果发射频率较低，则会出现松弛期，并且张力图会显示锯齿状振荡（未融合/不完全破伤风）。如果抽搐之间的间隔很短，肌肉就会持续收缩（融合/完全破伤风）。

>> ○ 张力受收缩纤维数量和收缩速度（率）的影响。

> ②（反义词）强直肌纤维：姿势肌；不产生动作电位；不遵循全有或全无的单位。

> ③ 抽动纤维的类型

>> ○ 慢肌纤维（红色纤维）：氧化/需氧； ↑ 线粒体和肌红蛋白；血管化良好。>>> ○ 氧化纤维特性：主要依靠有氧代谢——较慢但持续时间较长（抗疲劳）。

>>> ○ 与快肌相比：肌浆网不发达，Ca<sup>2+</sup>泵送较慢，胞质Ca<sup>2+</sup>停留时间较长；抽搐持续时间 ≈ 五倍长。

>> ○ 快肌纤维（白色纤维）：稀疏的线粒体和毛细血管；产生快速、有力的收缩，但无法维持； IIa 型和 IIb 型可以相互转换。

>>> ○ 糖酵解纤维特性：主要依靠糖酵解；直径较大，肌红蛋白较低，容易疲劳；疲劳通常归因于乳酸。

 ⑹ 肌肉收缩时的能量供给：根据肌纤维类型的不同供给方式

> ① 一般依次为ATP、磷酸肌酸、乳酸发酵、糖原、脂肪、蛋白质

> ② **第一<sup>st</sup>优先级。** ATP

>> ○ ~ 5 mM (约5秒)

>> ○ 尸僵：由于 ATP 耗尽，肌肉无法放松的状态

> ③ **第二<sup>第二</sup>优先级。** 磷酸肌酸

>> ○ 功能：再生 ATP 以对抗 ATP 消耗

>> ○ 磷酸肌酸 + ADP **→** 肌酸 + ATP

>>> ○ ATP分解时释放的能量：7.3 kcal

>>> ○ 磷酸肌酸键断裂时释放的能量：10.3 kcal

>> ○ 20 ~ 40 mM，快速能量源（约20秒）

> ④ **第三<sup>rd</sup>优先级。** 乳酸发酵：人体不进行酒精发酵或醋酸发酵

>> ○ 当能源耗尽时：葡萄糖→丙酮酸→乳酸，通过糖酵解获得ATP（葡萄糖→丙酮酸）。

>> ○ 积累的乳酸在一小时内返回血液并在肝脏中分解（科里循环），因此与肌肉酸痛的相关性较小

>> ○ 在肌肉中：丙酮酸的形成速度比乳酸快得多，因此产生的 NADH 多于 NAD+

>> ○ 在肝脏中：由于丙酮酸已重新合成为葡萄糖，因此存在的 NAD+ 多于 NADH

>>> ○ 当乳酸到达时，它与丰富的 NAD+ 反应形成丙酮酸

>> ○ 乳酸发酵经常发生在快肌纤维中，而快肌纤维是糖酵解依赖性的。

>> ○ 番茄作为恢复辅助剂

>>> ○ 西红柿50%以上是谷氨酸

>>> ○ 谷氨酸将丙酮酸转化为丙氨酸，减少乳酸产生和疲劳

> ⑤ **第 4<sup></sup> 优先级。** 糖原

>> ○ 在肝脏和肌肉中储存一天

>> ○ 肝脏中储存的糖原：为整个身体提供能量

>> ○ 肌肉中储存的糖原：只为肌肉提供能量

>> ○ 可以通过糖异生参与无氧和有氧呼吸

> ⑥ **第 5<sup></sup> 优先级。** 脂肪

>> ○ 碳水化合物 (50 ~ 60%) **→** 脂肪 (20 ~ 35%) **→** 蛋白质 (5 ~ 15%) 按能源使用顺序

>> ○ 有氧呼吸的能量来源（ **∴** 在缺氧条件下无效）

> ⑥ 第六<sup>th</sup>。 MLCP 使肌球蛋白轻链去磷酸化

> ⑦7<sup>th</sup>。平滑肌松弛：去磷酸化肌球蛋白失去ATP酶活性，导致肌动蛋白-肌球蛋白跨桥脱离→肌肉松弛

> ② 发生率约为万分之一

> ① 定义：剧烈运动后长时间出现的肌肉酸痛

> ② 肌肉收缩类型

>> ○ 几何收缩：在 ATP 消耗过程中，肌肉既不收缩也不拉长

>> ○向心收缩：ATP消耗过程中肌肉收缩，与之前学过的类似

>>> ○ 例：爬上坡，弯曲膝盖对地面施力

>> ○ 离心收缩：肌肉在 ATP 消耗过程中拉长

>>> ○ 示例：下降时，伸展膝盖的肌肉受到地面冲击力，导致它们拉长

> ③ 机制

>> ○ 第一<sup>第一</sup>。离心收缩使用 ATP，但肌肉在工作时通常会拉长

>> ○ 第二<sup>第二</sup>。即使受到外力拉动，肌肉也会收缩，导致一些肌动蛋白纤维弹出

>> ○ 第三<sup>第</sup>。一些肌动蛋白纤维断裂，因为一些肌动蛋白纤维可以承受负载>> ○ 第 4<sup></sup>。发生炎症等生物反应以保护伤口

>> ○ 第五<sup>th</sup>。出现发热或暂时性肌肉肿胀（肿块），肌肉显得更大

>> ○ 第六<sup>th</sup>。出现肌肉酸痛

>> ○ 7<sup>th</sup>。随后进行同类运动可减轻肌肉酸痛的严重程度
