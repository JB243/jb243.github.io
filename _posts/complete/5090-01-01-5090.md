## **1.概述**

 ⑴ 功能

> ① 身体支撑

> ② 保护内脏器官

> ③ 提供运动和肌肉附着的链接

> ④ 矿物质储存：钙、磷储存

> ⑤ 骨髓中的血细胞生成

 ⑵ 骨骼的分类

> ① 静水骨骼：当肌肉压缩充满不可压缩液体的腔体时在体内形成。

>> ○ 示例：腔肠动物（海葵）、鱿鱼和章鱼（喷射推进）、蚯蚓运动

>> ○ **蚯蚓运动：**

>>> ○ 第一<sup>第一</sup>。头尾节的纵向肌收缩，环行肌松弛，使头、尾节短而粗；它们的刚毛固定在地面上。其余的节段，纵向肌放松，环肌收缩，变得又长又细。

>>> ○ 第二<sup>第二</sup>。头部圆形肌肉的收缩将头部向前推。头部后面和后面的节段变厚，接触地面并固定，防止蠕虫向后滑动。

>>> ○ 第三<sup>第</sup>。前节再次增厚并锚定在新位置，而后节从地面升起并向前拉。

> ② 外骨骼：形成身体外部的硬壳，附有肌肉。

>> ○ 例如：软体动物、节肢动物。

> ③ 内骨骼：支撑身体的内部结构，肌肉附着并作用于其上。

>> ○ 示例：人体骨骼

 ⑶ 人体骨骼（内骨骼）

> ① 组成

>> ○ 总共206块骨头

>> ○ 主要骨骼：脊柱、头骨、胸腔。

>> ○ 附肢骨骼：关节、肩部、四肢骨。

## **2.骨骼的结构和形成**

 ⑴ 骨骼的组成部分

> ① 与正常组织相比，无机矿物质含量丰富。

> ② 水占骨重的5-8%。

>> ○ 85%的水分布在有机成分、胶原纤维、基质、骨晶体的水化层中。

>> ○ 15%的水分布在骨管和骨腔中。

> ③有机成分占干骨重量的33%，占骨体积的50%。

>> ○ 功能：赋予骨骼柔韧性和弹性。

>> ○ 主要由胶原蛋白组成。

>> ○ 胶原蛋白（胶原纤维、I型胶原蛋白）：90%。抗张力能力较差。

>> ○ 非晶质基质：1​​0%。

>> ○ 成骨不全症（脆骨综合症）：胶原蛋白突变和缺乏。

> ④ 无机矿物质占干骨重量的67%，占骨体积的25%。

>> ○ 功能：提供骨骼硬度。

>> ○ 无机矿物质中85%是磷酸钙。 15%是碳酸钙。

>> ○ 钙占干骨重量的39%：体内钙的99%（其余部分在牙齿中）。

>> ○ 磷酸盐占干骨重量的17%：体内磷的90%。

>> ○ 碳酸盐占干骨重量的9.7%：体内碳酸盐的80%。

>> ○ 钠占干骨重的0.7%：体内钠的35%。

>> ○ 镁占干骨重量的0.4%：占体内镁的50%。

>> ○ 钾占干骨重量的0.2%：体内钾的4%。

> ⑤ 补钙

>> ○ 99%的钙分布在牙齿和骨骼中。

>> ○ 骨骼中的钙含量在出生后第一年就被替换了100%，儿童时期约为10%，成年后每年替换2-4%。

>> ○ 成年期钙的合成和分解增加。

>> ○ 40-50 岁期间，由于分解比合成增加，骨质流失加速。

 ⑵ 骨骼结构

> ① 骨骼由致密骨和松质骨组成。

>> ○ 致密骨（皮质骨、致密骨）：骨的外层。

>>> ○ 形成致密、坚韧的骨骼外表面。

>>> ○ 强度：100-150 MPa。

>> ○ 松质骨（小梁骨）：骨头的内部部分。

>>> ○ 多孔，内部结构呈蜂窝状。

>>> ○ 强度：8-50 MPa。

>> ○ 骨膜

>>> ○ 致密纤维膜。由与血管相连的福克曼运河穿过。作为神经纤维的通道。>>> ○ 内层：又称形成层或成骨层。成骨细胞排列松散。血管丰富。发生活跃的骨形成。

>>> ○ 中层：含有未分化骨祖细胞的区域。

>>> ○ 外层：由不规则排列的胶原蛋白和弹性纤维组成的厚层致密结缔组织。也称为纤维层。

>>> ○ 夏皮纤维：从骨膜深入延伸到皮质骨的纤维束。

>> ○ 骨内膜

>>> ○ 覆盖骨腔壁和松质骨表面的薄层细胞结缔组织。

>>> ○ 主要由骨祖细胞（骨内细胞）组成，是成骨细胞的前体细胞。

>>> ○ 仅存在于长骨中。

>>> ○ 围绕髓腔，即骨骼的中心部分。

>> ○ 营养物质和气体交换：通过血管和福克曼管运输。

>> ○ **成分 1：** 破骨细胞

>>> ○ 分解骨骼（骨细胞）并将钙释放到血液中。

>>> ○ 直径：20-100 μm。

>>> ○ 白细胞内的一种单核巨噬细胞。

>>> ○ 单核破骨细胞转变为多核破骨细胞（约50个巨核）并粘附在骨表面，分泌溶酶体酶以促进骨吸收。

>> ○ **成分 2：** 成骨细胞

>>> ○ 形成骨骼（骨细胞）并从血液中去除钙。

>>> ○ 也称为骨形成细胞。

>>> ○ 合成并分泌骨基质，并将钙、镁等矿物质沉积到基质上。骨组织发生矿化。

>> ○ **成分 3：** 骨细胞（骨细胞）：分裂能力有限 ×

>>> ○ 位于骨腔内。

>>> ○ 骨细胞通过小管连接，并通过间隙连接交换小分子。

>> ○ **成分 4：** 成骨细胞：分裂能力 ○>> ○ **成分 5：** 血管、神经纤维

>> ○ **成分 6：** 甲状旁腺激素：刺激骨吸收。

>> ○ **成分 7：** 来自甲状腺的降钙素：促进骨形成。

>> ○ **成分 8：** 血钙：参与血液凝固、肌肉收缩、产生动作电位、酶活性。

> ⑥ 软骨

>> ○ 软骨没有血管、淋巴管或神经供应，细胞密度低。

>> ○ 细胞成分中水占30%。

>> ○ 对于关节润滑很重要。

>> ○ 由于 II 型胶原蛋白的分泌，软骨会被番红-O 染色。

>> ○ 软骨组织中硫酸盐的吸收率被用作骨骼生长的指标。

>> ○ 生长因子：

>>> ○ 胰岛素样生长因子 (IGF)

>>> ○ 转化生长因子-β (TGF-β)

>>> ○ 碱性成纤维细胞生长因子 (b-FGF)

>>> ○ 骨形态发生蛋白 (BMP)

>>> ○ 血小板衍生生长因子 (PDGF)

> ⑦ 骨骼重塑
 
>> ○ 定义：随着破骨细胞响应外部条件分解骨骼，以及成骨细胞创建骨骼，骨骼的结构不断变化。

>> ○ 骨骼经历持续的重塑过程。

>> ○ **沃尔夫定律**（也称为沃尔夫定律）

>>> ○ 定义：指骨重塑过程中的力学变化。

>>> ○ 由朱利叶斯·沃尔夫 (Julius Wolff) (1836-1902) 提出。

>>> ○ 摘要：骨骼越用越强。

>>> ○ 对骨骼的强烈物理影响会导致强度和刚度增加，作为一种补偿机制。

>>> ○ 由于身体新陈代谢降低，对骨骼的物理影响较弱，会导致力量和硬度下降。

>>> ○ 强度和刚度与骨密度、厚度等有关。

⑶ 骨骼生长

> ① 骨骼发育阶段的分类

>> ○ 编织骨

>>> ○ 见于肿瘤、佩吉特骨病、胎儿骨骼、生长中的骨骼、骨折的骨骼以及骨折后的骨形成区域。

>>> ○ 佩吉特骨病：骨质疏松症后常见的骨病，由于破骨细胞过度活动导致骨骼生长异常和疼痛。

>> ○ 板层骨

>>> ○ 出生第一年后开始形成。

>>> ○ 代替编织骨作为成熟的骨组织。

> ② 第一<sup>第一</sup>。胚胎发育阶段：所有骨骼均由软骨组成。

> ③ 第二<sup>第二</sup>。骨形成阶段

>> ○ 初级骨化中心在软骨中心内发育，开始骨化过程。

>> ○ 骨分化标记物：碱性磷酸盐（_体外_；7 ~ 10 天）→ 骨桥蛋白 → 骨连接蛋白 → 骨钙素（_体外_；3 周）

> ④ 第三<sup>第</sup>。二次骨化中心形成

>> ○ 当初级骨化中心进行足够的骨化时，骨的末端会出现新的骨形成中心（次级骨化中心）。

>> ○ 骨传导：将骨细胞传导到邻近区域。

>> ○ 骨诱导：自身诱导骨组织形成的能力。

> ⑤ 第 4<sup></sup>。骨骺（生长）板闭合

>> ○ 在激素影响下，成骨细胞前体细胞的增殖速度比成软骨细胞快，导致两个骨化中心之间的骨骺生长板闭合→纵向生长停止。

>> ○ 激素：生长激素、甲状腺激素、性激素。

>> ○ 骨骺生长板：两个骨化中心之间的软骨区域。

>> ○ 骨骺（生长）板的结构

>>> ○ **结构1.** 静止区：静止的干状软骨细胞。

>>> ○ **结构2.** 增殖区：分裂细胞柱扩大骨长度。

>>> ○ **结构3.** 肥大区：软骨细胞增大，分泌基质，然后死亡。

>>> ○ **结构4.** 骨化：钙化的骨组织取代软骨。>> ○ 形态发生素和生长板

>>> ○ PTHrP（甲状旁腺激素相关蛋白）：促进静息区的形成；赋予软骨细胞增殖能力；防止过早肥大（通过抑制 GDF5 和 BMP 的过度激活）。

>>> ○ IHH（印度刺猬）：由肥大软骨细胞产生；增加静息区 PTHrP 表达；有助于平衡增殖和分化。

>>> ○ GDF5（生长分化因子 5）：有助于软骨形成； BMP 家族的成员；在软骨膜和周围生长板中表达；作用于增殖的软骨细胞以促进生长。

>>> ○ BMP（骨形态发生蛋白）：过度激活时，会导致过早肥大。

## **3。骨骼的机械连接**

 ⑴ 关节：骨骼之间的连接

> ① 不可动关节

>> ○ 纤维接头

>>> ○ 由胶原蛋白和弹性纤维组成；三种类型：韧带联合（韧带关节）、缝合和嵌合（钉和插座）。

>>> ○ 缝合线的代表性例子是颅骨（头骨）缝合线。

>> ○ 软骨关节

>>> ○ **1 型：** 软骨联合（原发性软骨关节）— 联合由透明软骨形成。

>>> ○ **2 型：** 联合 — 纤维软骨结合。

>> ○ 由于脊柱活动受限，有时被归类为双关节病。

> ② 可动关节：典型关节

>> ○ 细分为滑膜关节、平面关节、铰链关节、枢轴关节、椭球关节（髁状关节）、鞍状关节、球窝关节。

>> ○ 滑膜关节：对向关节软骨+滑液+关节囊。

>> ○ 球窝关节：绕三个轴运动的关节。

>>> ○ 示例：肩部（肱骨与肩胛骨相交处）、髋部（股骨与骨盆相交处）；也称为球状/子叶状关节。

>> ○ 铰链关节：绕一个轴运动的关节。

>>> ○ 示例：肘部的肱尺部分；膝关节。

>> ○ 枢轴关节：可以绕其长轴旋转的关节。

>>> ○ 示例：肘部的近端桡尺关节；寰枢关节处的颈部旋转。

> ③ **滑膜关节**（关节软骨）：代表可动关节

>> ○ 被认为是异质材料：间质流体形式（不可压缩、不可混溶）+固体形式（多孔、可渗透）。

>> ○ 相对关节面之间的间隔约为 100 nm。

>>> ○ 软骨细胞呈球形并呈放射状排列。

>> ○ **结构 4：** 钙化区

>>> ○ 最深层

>>> ○ 分为软骨下骨和透明软骨。

>>> ○ 用 H&E（苏木精和伊红）染色显示为深蓝色线。

>> ○ **成分 1.** 胶原蛋白

>>> ○ 人体内含量最丰富的蛋白质。

>>> ○ 原胶原蛋白：胶原蛋白的基本单位。

>>> ○ 原胶原α链 → 三条组装成三螺旋 → 螺旋聚集成带状原纤维。

>>> ○ 高抗拉力。

>>> ○ 低抗压性：蛋白聚糖提供大部分抗压性。

>> ○ **成分 2.** [蛋白聚糖](https://jb243.github.io/pages/1389)

>>> ○ 由核心蛋白和附加的多条糖胺聚糖 (GAG) 链组成的大糖蛋白。

>>> ○ 压缩 → 蛋白多糖变形并排出间质液 → 固定电荷密度上升，阻力增加。

>>> ○ 唐南渗透膨胀压：压缩→聚集蛋白聚糖聚集，负电荷密度增加→水分被吸入，组织膨胀（水肿）。

 ⑵ 骨骼和肌肉连接

> ① 韧带：连接骨头与骨头。

>> ○ 胶原纤维并非完全平行，对各个方向的负载有一定的耐受力。

>> ○ 弹性蛋白几乎不存在。

> ② 肌腱：连接肌肉和骨骼，实现运动。

>> ○ 起始于中轴骨骼，附着于周边骨骼。

>> ○ 胶原纤维排列有序、平行，可承受单向拉伸载荷。拉伸强度50兆帕。

>> ○ 胶原蛋白含量为80%。弹性蛋白几乎不存在。血管相对稀疏。

>> ○ 腱旁组织：血管化。形成保护套。

>> ○ Epitenon：无血管。含有肌腱细胞并促进肌腱在滑液中滑动。

> ③韧带和肌腱的血液供应很少，导致伤口愈合缓慢。
