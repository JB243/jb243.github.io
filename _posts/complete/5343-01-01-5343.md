## **1\.天球和坐标系**

 ⑴ 天球

> ① 以地球上的观察者为中心的想象的无限球形天空。

> ② 地球中心是天球的中心。

 ⑵ 在天球上的位置

> ① 天顶和天底 **：** 观察者的垂直线与天球相交的点。

>> ○ 垂直圆 **:** 经过天顶和最低点的圆。

> ② 地平线及其他

>> ○ 地平线 **：** 地平线平面延伸与天球相交所形成的圆。

>> ○ 高度圆 **：** 所有与地平线平行的圆。

>> ○ 子午线**：** 天球的赤道，又称垂圈。

> ③ 极数

>> ○ 天球的北极和南极 **：** 地球自转轴与天球的交点。

>> ○ 天球赤道 **：** 地球赤道延伸与天球相交所形成的圆。

>> ○ 小时圈 **:** 穿过天球北极和南极的圈。

> ④ 子午线 **：** 穿过天顶、天底和天极的独特圆。

> ⑤ 当考虑到地球中心另一侧的观察者时，只需将观察者的方位反转到同一天球即可。

>> ○ 这对于将解释从北半球扩展到南半球很有用。

 ⑶ 天体坐标

> ① 赤道坐标系

>> ○ 纬度

>> ○ 经度

>> ○ 示例 **:** 当地点 A（北纬 34.2°，东经 135°）1 月 26 日上午 00:20 时，地点 B（北纬 34.2°，西经 120°）现在几点？

> ② 水平坐标系 **：** 以天顶、天底、水平面为中心解释。

>> ○ 基本点

>>> ○ 北点 **：** 在地平线与子午线交点处最接近天极的点。

>>> ○ 南点 **：** 地平线与子午线交点处最靠近南天极的点。

>>> ○ 东点 **：** 从北点沿地平线向东 90° 的点。

>>> ○ 西点 **：** 从北点沿地平线向西 90° 的点。

>> ○ **水平坐标** **：** 使用相对于地平线和北点的方位角和高度来指示天体位置的坐标系。

>>> ○ 方位角 (A) **：** 将星星投射到地平线上时，从北点顺时针旋转的角度。

>>> ○ 高度 (h) **:** 从地平线到天体垂直测量的角度。

>> ○ 天顶距离 (Z) **:** 天顶到天体的角度。

<中心>Z = 90° \- h</中心>

>> ○ 从不同纬度的两点观测同一颗恒星时，纬度越高，恒星停留在地平线以上的时间越长。

>>> ○ 原因**：** 小时圈与地平线之间的角度减小，导致能见度更长。

>>> ○ 随着星星在地平线以上的时间增加，星星升起和落下时的方位角差异也会增加。

>>> ○（注）考虑北半球的绕极恒星可以帮助理解这种现象。

>> ○ 当观察两颗恒星升起时，恒星的赤纬越高，它在地平线上方停留的时间就越长。

>>> ○（注）考虑北半球的绕极恒星可以帮助理解这种现象。>> ○ 纬度越高，两颗恒星升起时的方位角差异越大。

>>> ○ 结合以上两个现象就可以理解了。

 ③赤道坐标系**：**以天极、赤道为中心解释。

>> ○ 赤经 (α) **:** 春分点与天体时圈之间的角度。 0 小时至 24 小时，1 小时 = 15°。

>>> ○ 从春分沿天赤道至天体。西→东。

>>> ○ 春分赤经定义为 0h。

>> ○ 赤纬 (δ) **:** 恒星位于正南时的高度。 -90° ± 90°（北 (+)、南 (-)）。

>>> ○ 从天赤道沿时圈至天体。向上或向下。

>>> ○ 春分、秋分 **:** 赤纬 = 0°。

>>> ○ 夏至 **:** 赤纬 = 23.5°。

>>> ○ 冬至 **:** 赤纬 = -23.5°。

>> ○ **赤道坐标** **:** 使用赤经和赤纬表示天体位置的坐标系。

>>> ○ 特征**：** 无论时间和地点如何，坐标都保持不变。

> ④ 子午线高度

>> ○ 定义**：** 天体位于南子午线时的高度。

>> ○ 子午线高度 = 90° \- 纬度 + 赤纬

>> ○ 90° \- 纬度 **:** 水平坐标系中天赤道赤纬与南经线交点的高度。

>> ○（注）子午线高度应相对于水平坐标系考虑。

> ⑤ 黄道坐标系

>> ○ 黄道坐标 **:** 以黄道面为基准表示恒星位置的坐标。

>> ○ 黄道纬度、黄道经度

>> ○ 由于太阳绕黄道逆时针旋转，黄道纬度变为零，但黄道经度随着时间的推移而增加。

  ⑷ 星星的名字

> ① 北半球赤道星

> ② 黄道附近的赤道星

> ③ 南半球赤道星

> ④ 12 黄道星座 **:** 以春分点为中心，将黄道分为 12 等分，每等分 30°，并选择距离这些分分点最近的恒星，形成 12 星座。
