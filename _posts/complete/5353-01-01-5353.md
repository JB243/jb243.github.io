## **3\. **海水的运动**

⑴ **波速**

**图。 5.** 深海波浪和浅海波浪

> ① **船上水粒子的运动方向：**波浪传播方向

> ② **波速一般方程**

>> ○ **含义1：**速度随深度增加（显而易见）

>> ○ **含义2：** 波长越长，速度越高 **：** 想象海啸时很容易理解

> ③ **情况1：** **深海波浪** **：**当深度> 0.5 λ时。也称为表面波。

>> ○ 水粒子形成**圆形轨道**。

>> ○ 半径随着深度的增加而减小。

>> ○ 深海波浪的速度 **：** （注）这并不意味着波浪的速度随深度无限增加。

>> ○ 随着深度的增加，水粒子圆周运动的尺寸减小。

> ④ **情况2：** **浅水波浪** **：** 当深度< 0.05 λ时。又称长波。

>> ○ 随着深度的增加，只有波长会缩短。

>> ○ 浅水波速

>> ○ 随着深度减小，**:** 传播速度减小，波长缩短，波高增加。

> ⑤ **情况3：** 0.05 λ ＜ 深度＜ 0.5 λ **：** 波浪开始受到海底影响。

> ⑥ **潮汐波（海啸）**

>> ○ **类型 1：** 风暴海啸 **：** 由低压（台风）产生

>>> ○ 到达海岸 → 深度减小 → 速度减小 → 波长减小 → 波高增加

>> ○ **类型 2：** 地震海啸（海啸）**：** 由地震、火山和山体滑坡等垂直运动产生

>>> ○ 波长很长，与**浅水波**具有相同的特性

>>> ○ 垂直分层比水平分层更容易发生地震海啸

## **5\.潮间带**

⑴ 了解潮间带

> ① 沿岸宽阔平坦的沙子或卵石区域，涨潮时被淹没，退潮时暴露在外

> ② 组成部分：河口生态系统、潮间带生态系统、潮下生态系统

>> ○ 河口生态系统 = 部分沼泽植被 + α

>> ○ 潮间带生态系统 = 沼泽植被其他部分 + 潮间带

>> ○ 潮下生态系统 = 高潮带 + 低潮带

> ③ 形成原因

>> ○ 波浪作用

>>> ○ 波浪 **:** 随着风暴到达海岸而减弱

>>> ○ 波浪作用 **：** 由波浪引起的侵蚀，导致土壤、沙子和砾石的产生、运输和沉积，形成滩涂

>> ○ 【潮滩现象】(https://jb243.github.io/pages/344)

>> ○ 沉积现象 **:** 河流沉积物堆积，含有大量有机物，形成沉积物

> ④ 功能

>> ○ 经济价值**：** 提供约50%的鱼类，以及近100%的甲壳类和软体动物

>> ○ 自然净化功能 **：** 通过沼泽植被和滩涂过滤污染物，防止富营养化

>> ○ 自然灾害减缓和气候调节 **：** 作为陆地与海洋交汇处的缓冲区，缓解快速变化

>> ○ 生态栖息地 **：** 主要是海洋生物和鸟类的舒适繁殖地

>>> 示例 **:** 拉姆萨尔公约

>> ○ 文化教育价值

 ⑵ 潮间带生物的环境条件和分布

> ① 底栖生物的定义和分类

>> ○ **第 1 类：** 浮游生物 **：** 由于游泳能力弱或无游泳能力而沿水流漂流

>>> ○ 底栖动物 **:** 根据栖息地、生物谱系、食物类型和大小分类

>>> ○ 饮食不仅包括草食性和肉食性，还包括碎食性、滤食性和食腐性

> ② 底栖生物的环境因素

>> ○ 物理因素**：**波浪、潮汐、海水、基质、光照、水温等。

>>> ○ 波浪

>>>> ○ 形成泡沫 → 阻挡光线穿透
