## **1\.概述**

 ⑴ 细胞分裂

> ① 有丝分裂：体细胞分裂

> ② 减数分裂：生殖细胞分裂（二倍体 → 单倍体）

>> ○ 发生于性腺

>> ○ 防止每一代遗传物质加倍

 ⑵ 配子和受精卵

> ① 配子：单倍体细胞（n）

> ② 合子：由两个配子融合而成，二倍体（2n）

> ③ 受精：精子与卵子的融合

>> ○ 精子：活动配子

>> ○ 卵：不动配子

>> ○ 雌性：产生较大配子（卵）的生物体

>> ○ 雄性：产生较小配子（精子）的生物体

> ④ 性腺：产生配子的器官，即睾丸和卵巢

> ⑤ 施肥方法

>> ○ 体内受精：通过交配在生殖道内受精。哺乳动物、鸟类、爬行动物、鲨鱼等

>> ○ 体外受精：雌性将卵子释放到水中，雄性释放精子。鱼类、两栖类等

 ⑶ 繁殖

> ① 无性繁殖：仅涉及有丝分裂，产生基因相同的后代

>> ○ 分裂：通过有丝分裂分成两个相同的细胞；例如，海葵。

>> ○ 出芽：体细胞的一部分脱离，发育成新个体；例如，九头蛇。

>> ○ 再生：在全身再生的情况下，它被认为是繁殖的一种形式；例如，海星、扁虫。

>> ○ 孤雌生殖：未受精卵的发育；例如，蚜虫（雌性和雄性）、蜜蜂（雄性）。

> ② 有性生殖：涉及减数分裂，确保进化的遗传多样性

>> ○ 配子发生 → 交配 → 受精

>> ○ 染色体重组创造出多样化的后代以适应不同的环境

>> ○ 雌雄同体和雌雄同体

## **2\.性腺激素**

 ⑴ 概述

> ① 除LH、FSH外，大多数性腺激素均为类固醇。

 ⑵ GnRH（促性腺激素释放激素）

> ① **功能1.** 刺激垂体前叶分泌LH和FSH

 ⑶ 卵泡刺激素（FSH）：肽类激素

> ① 男

>> ○ **功能 1.** 刺激支持细胞以及睾酮生成 → 精子生成

> ② 女

>> ○ **功能1.** 刺激卵泡内的颗粒细胞，促进芳香酶分泌

>> ○ **功能2.** **早熟**：卵泡成熟

 ⑷ 黄体生成素（LH）：来自垂体前叶的肽类激素 

> ① 男

>> ○ **功能1.** 刺激间质细胞分泌睾酮

>> ○ **功能 2.** 睾酮与 FSH 一起刺激支持细胞

> ② 女

>> ○ **功能1.** 刺激颗粒细胞分泌睾酮

>> ○ **功能2.晚熟：诱导排卵

>> ○ **功能3.** 促进其他雄激素的分泌

>> ○ 睾酮通过颗粒细胞分泌的芳香酶转化为雌激素

 ⑸ 雄激素：睾酮、雄酮、DHA的统称。

> ① 功能：男性生殖重要的类固醇激素

>> ○ **功能 1.** 精子产生>> ○ **功能2.** 男性第二性征：男性生殖器官发育、毛发粗细及分布、肌肉量增大、声音低沉

>> ○ **功能3.** 通过雌激素抑制乳房生长

>> ○ **功能 4.** 通过支持细胞启动并维持精子形成

>> ○ **功能5.** LH负反馈，抑制下丘脑GnRH释放和垂体前叶分泌LH

>> ○ **功能6.** 诱导和维持辅助生殖器官的分化和功能

>> ○ **功能 7.** 促进骨骼生长（青春期）和阻止骨骼生长（青春期后期）

>> ○ **功能8.** 性欲和攻击行为

>> ○ **功能9.** 刺激肾脏促红细胞生成素分泌

> ② 女性的卵巢和肾上腺皮质也产生少量

> ③ **1 型：** 睾酮

>> ○ 与雌酮和雌二醇非常相似，主要区别在于甲基化、双键和酮基

>> ○ 男性约95%的睾酮由睾丸分泌，5%由肾上腺皮质分泌

> ④ **2 型：** 雄酮

> ⑤ **类型3：** DHEA（脱氢表雄酮）：DHT最具代表性

> ⑥ **类型 3-1:** DHT（脱氢睾酮）

>> ○ 睾酮在外周组织中被 5α-还原酶转化为 DHT

>> ○ 前列腺癌和前列腺增生：癌细胞受到DHT的刺激。治疗包括抑制 5α-还原酶以抑制 DHT。

>> ○ 先天性肾上腺增生症（CAH）、肾上腺生殖综合症

>>> ○ 由于21-α-羟化酶缺乏，胆固醇转化为性激素，增强DHT功能

>>> ○ 皮质酮、皮质醇、醛固酮缺乏导致女性男性化

>>> ○ 皮质醇和醛固酮处方可缓解CAH

> ⑦ 大脑：睾酮通过芳香酶转化为雌激素

> ⑧ 太监：切除睾丸。所有附属生殖管退化。朝鲜时代使用

⑹雌激素：雌酮、雌二醇、雌三醇的统称。

> ① 结构

>> ○ **功能 10.** 下丘脑和垂体反馈。

>> ○ **功能11.** 更年期调节。

>> ○ **功能12.** 轻微降低体温。

> ④ 合成：睾酮产生后，在芳香酶的作用下转化为雌激素。

>> ○ 龟的芳香酶在25°C时活性较低，导致生出雄性。

>> ○ 龟的芳香酶在32°C时更加活跃，导致雌性分娩。

 ⑻抗苗勒氏管激素：糖蛋白。

 ⑼ 抑制素：肽类激素。

## **3。生殖系统的发育：胚胎管**

 ⑴ 双电位（无差异）性腺

> ① 生殖结构直到发育第7周才开始分化。> ② 当发育信号传递时，髓质分化为睾丸，形成男性生殖系统。

> ③在没有发育信号的情况下，皮质分化为卵巢，形成女性生殖系统。

> ④ 双能内生殖器：源自胚胎肾脏，由沃尔夫管和苗勒管组成。

>> ○ 随着发育的进行，一对导管发育，而另一对导管退化。

> ⑤ 双电位外生殖器：由生殖结节、泌尿生殖皱襞、尿道沟、阴茎/阴蒂、阴囊/阴唇组成。

⑵ SRY（Y染色体性别决定区）：男性Y染色体上的性别决定位点

> ① SRY 诱导性腺髓质分化为睾丸并刺激 Leydig 和 Sertoli 细胞。

> ② 间质细胞：在SRY影响下，分泌睾酮及其衍生物DHT（二氢睾酮）

>> ○ 睾酮：促进沃尔夫管分化，有助于睾丸从腹部下降到阴囊

>> ○ DHT：调节外生殖器和其他男性第二性征的分化

> ③ 支持细胞：在SRY影响下，分泌抗苗勒氏管激素（AMH）

⑶ 男性：沃尔夫管→输精管、附睾、精囊

> ①间质细胞分泌的睾酮积极促进沃尔夫管的发育。

> ② 支持细胞分泌的 AMH 主动引起苗勒管退化。

> ③ 假两性畸形

>> ○ 男性继承了 5α-还原酶，这种酶可将睾酮转化为 DHT。

>> ○ 缺乏这种酶的患者不会产生双氢睾酮，因此出生时外生殖器表现为女性。

>> ○ 例如，男性外生殖器和前列腺在胎儿发育过程中未能充分发育。

>> ○ 在青春期，睾丸睾酮分泌的更新通常会导致自发的男性化。

> ④ 隐睾

>> ○ 如果睾丸留在腹腔内，温度升高会损害生精功能，导致不孕。

⑷ 女性：苗勒氏管→阴道上部、输卵管/输卵管、子宫、子宫颈、阴道

> ① 由于间质细胞不分泌显着的睾酮，沃尔夫管退化。

> ② 在缺乏DHT的情况下，外生殖器发育出女性特征。

> ③ 由于支持细胞不分泌AMH，苗勒氏管自然发育。

## **4。男性生殖系统**

 ⑴ 生殖器官

> ① 附睾和精囊由沃尔夫管（中肾管）形成。

> ②睾丸（睾丸）：产生精子和激素；正常的精子发生需要相对较低的温度。

>> ○ 胎儿第七个月左右，睾丸从腹腔下降到阴囊。

>> ○ 1-3% 的新生儿有不完全血统。

>> ○ 约80%随着生长而自发下降。

>> ○ 精子发生需要低于核心体温 4–5 °C 的温度； ≥35°C 时平滑肌松弛。

> ③附睾：储存和浓缩精子；赋予活力（即精子成熟）；射精时通过蠕动输送精子。

> ④前列腺：分泌一种稀薄的液体，含有前列腺素、营养物质和碱性物质，增强蠕动能力。

>> ○ 前列腺素，占精液的60%，促进精子活力和女性生殖道的活力。

>> ○ 良性前列腺增生（BPH）：引起尿频、排空不完全感；通过前列腺特异性抗原（PSA）测试进行评估。> ⑤ 精囊：分泌精液的大部分成分。

>> ○ 产生含有葡萄糖和果糖的粘稠分泌物。

>> ○ 碱性，保护精子免受女性阴道酸性分泌物的侵害。

>> ○ 精子使用的大部分能量由果糖提供。

> ⑥ 考珀氏（尿道球）腺：分泌透明、粘液、碱性的射精前液体 → 中和尿液酸度并在射精前清洁尿道。

>> ○ 还分泌少量润滑性粘液。

> ⑦ 尿道

>> ○ 生殖道（射精管）和泌尿道共用尿道。

>> ○ 当精囊的导管与穿过前列腺的导管连接，然后连接到尿道时，就形成射精管。

>> ○ 来自尿道球（考珀）腺的导管单独连接。

> ⑧ 龟头（龟头）

>> ○ 包皮环切术：切除包皮。与阴茎癌、某些性传播感染和宫颈感染的风险降低相关；与割礼的宗教仪式有关。
 
⑵ 睾丸

> ①基膜：防止异物侵入；形成血睾屏障的一部分来调节分子运输。

> ② 生精小管

>> ○ 精子发生部位。

>> ○ 包括支持细胞和间质细胞。

>> ○ 总长度约250m。

> ③ 间质细胞

>> ○ 位于生精小管之间的间隙。

>> ○ 激素刺激：在LH刺激下，分泌睾酮等雄激素。

>> ○ 激素分泌：睾酮。

> ④ 支持细胞

>> ○ 嵌入曲细精管壁；为精子细胞的产生和发育提供广泛的支持。

>> ○ 荷尔蒙刺激：响应睾酮和 FSH，分泌促进精子发生的旁分泌因子。

>> ○ 激素分泌：抗苗勒氏管激素（AMH）、雄激素结合蛋白（ABP）、抑制素。

>> ○ 提供营养支持。

>> ○ 形成血睾屏障，保护精子细胞免受免疫攻击。

>> ○ 吞噬发育中精子细胞残留的细胞质。

⑶ 精子

> ① 单倍体核。

> ②顶体：来源于高尔基体；含有用于渗透卵子的水解酶；高尔基体参与形成顶体反应所需的顶体。

> ③鞭毛（中段）：含有大量线粒体；产生 ATP。

> ④ 尾巴：鞭毛运动；速度 1–4 毫米/分钟。

> ⑤ 必须暴露在女性生殖道的化学环境中才能获得受精能力（获能）。
