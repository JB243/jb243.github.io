## **1\.概述**

⑴刺激

> ①定义：周围环境的变化

> ② 特征：形态、强度、地点、时间

⑵ 感觉：感觉感受器受到刺激而激活时引发的主观体验

> ①机制：感觉转化→放大→传递→整合→意识

>> ○ 感觉感受器：当物理刺激超过阈值时产生动作电位

>> ○ 感觉传导：直接或通过 GPCR 介导将光能、化学能、机械能、热能转换为电能

> ② 感觉编码：区分刺激的机制

>> ○ 刺激方式：特定感觉受体标记的线感觉通道

>>> ○ 特定神经能量定律：感觉的本质是由感觉受体的类型决定的，而不是由刺激的类型决定的

>> ○ 刺激强度：感觉感受器电压大小、动作电位频率、重复次数

>> ○ 刺激位置：地形组织

>>> ○ 示例：体感小人、Retinotopic 地图、Tonotopic 地图

> ③ 感觉感受器的类型

>> ○ 基于刺激的感觉感受器类型

>>> ○ 电磁接收器

>>> ○ 化学受体

>>> ○ 机械接收器

>>> ○ 伤害感受器

>>> ○ 温度感受器

>> ○ 基于结构的感觉感受器类型

>>> ○ 简单受体：具有游离神经末梢的神经元

>>> ○ 复杂的神经受体：包裹在结缔组织鞘中的神经末梢

>>> ○ 专门的感觉受体：释放神经递质以启动感觉神经元的动作电位

>> ○ 基于原理的感觉感受器类型

>>> ○ 强直受体：缓慢适应受体

>>>> ○ 即使在持续刺激期间也能保持去极化

>>>> ○ 传达刺激**强度**（例如本体感受器、伤害感受器）

>>> ○ 相位受体：快速适应受体

>>>> ○ 如果刺激强度保持恒定，则停止放电（例如嗅觉感受器、触觉感受器）

>>>> ○ 主要检测刺激的开始、偏移和**变化**

> ④ 当神经元是感觉细胞时

>> ○ 嗅觉感受器

>> ○ 皮肤受体

>> ○ Pacinian 受体

>> ○ 温度感受器

>> ○ 牵张感受器（例如，肌梭[小脑、无意识感觉]）

⑵ 感觉系统的特点

> ① **交叉**（交叉）：大脑中的感觉束和运动束交叉，控制身体的相对侧（对侧）区域

>> ○ 原因尚不清楚

> ② 突触整合：由于连接结构而产生的感觉信息交互

>> ○ 收敛

>> ○ 发散

>> ○ 协助

>> ○ 遮挡

> ③横向抑制：感觉受体兴奋抑制附近受体活性

>> ○ 两点辨别：区分两个相邻刺激的能力

>>> ○ 示例：识别食物中的尖峰

⑷ 心理物理学

> ①原理：物理现实≠心理现实，主动的，进化的

> ② 心理物理测量功能

>> ○ 不同人的反应不同

>> ○ 测量在没有先前刺激的情况下突然引入刺激时的反应

>> ○ 概念：检出率、理论阈值曲线、低于刺激阈值、绝对阈值、获得阈值曲线

> ③ 显着差异（JND）

>> ○ 韦伯定律：ΔI/Ii = 常数

>>> ○ 光：ΔIlight/Ii，光×100=8（%）

>>> ○ 重量：ΔI重量/Ii，重量=2（%）

>>> ○ 声音：ΔIsound / Ii，声音=0.3（%）

⑸ 【味觉感受器：舌头】(https://jb243.github.io/pages/1293)

⑹ [皮肤感受器](https://jb243.github.io/pages/511)

⑺ [伤害感受器](https://jb243.github.io/pages/1295)

⑻ 化学受体

> ①化学感

>> ○ 最古老、分布最广的感觉器官

>> ○ 检测我们内部和外部环境中的化学信号

>> ○ 所有物种和细胞都能检测到各种化学物质

> ② 脊椎动物的嗅觉感受器：鼻子

> ③ 脊椎动物的味觉感受器：舌头

> ④ 飞蛾的化学受体：触角

>> ○ 在成年男性中，受体甚至可以对女性分泌的单分子安眠酮产生反应。

>> ○ Bombykol → 雄性触角 → 感觉器（感觉毛） → 嗅觉感受器 → 溶解在液体中 → 作用于感受器膜 → 溶解。

⑼ 电磁接收器

> ① 洛伦齐尼器官：存在于鲨鱼、蝾螈中，可以感知电信号

⑽ 内感受

> ① 示例：即使闭着眼睛也能知道手臂的位置

⑾ 热感受器

> ① 示例：蛇有窝器官、热传感器
