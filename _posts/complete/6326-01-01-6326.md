## **1\.什么是化学？**

⑴ 化学：物质的研究

> ①物质的组成、性质、结构

> ② 事项变更

⑵ 物质状态

> ① 固体

>> ○ 有一定的形状和固定的体积

> ② 液体

>> ○ 比固体更不规则

>> ○ 由于其运动比较自由，因此具有流动性

>> ○ 它没有固定的形状，采用容器的形状，但体积保持不变

> ③ 燃气

>> ○ 比液体更无序

>> ○ 分子相距较远，分子相互作用可以忽略不计

>> ○ 比同体积的固体或液体轻得多，可压缩，占用空间

⑶ 物质的分类

> ①纯物质：区分元素和化合物

> ② 混合物：分为均质混合物和非均质混合物

> ③要素

>> ○ 仅由一种原子组成的物质

>> ○ 示例：铜 (Cu)、氮 (N<sub>2</sub>)、铁 (Fe)、金刚石 (C)、铝 (Al)

> ④ 化合物

>> ○ 由两种或两种以上不同元素按一定比例组成的物质

>> ○ 示例：二氧化碳 (CO<sub>2</sub>)、硫酸铜 (CuSO<sub>4</sub>)、水 (H<sub>2</sub>O)

> ⑤ 均质混合物（溶液）

>> ○ 多种纯物质均匀分布的混合物，成分全部相同

>> ○ 示例：空气、糖溶液、盐水、合金

> ⑥ 非均相混合物

>> ○ 多种纯物质分布不均匀、不同部位成分不同的混合物

>> ○ 示例：泥水、牛奶、花岗岩、血、烟

⑷ 物质的性质

> ① 物理性能

>> ○ 物质的状态或形式发生变化而不改变其化学成分

>> ○ 示例：颜色、熔点、沸点、密度、溶解度等。

> ② 化学性质

>> ○ 通过化学反应将物质转变为新物质

## **2.化学式语法**

⑴ 原子、元素、分子、离子

> ① 原子：大多数化学反应的基本单位。 1900 年代后观察到小于原子的单位

> ② 元素：具有相似特征的原子的集合

> ③ 分子：由原子键合而成的功能单元

> ④ 离子：因获得或失去电子而带电荷的原子

> ⑤ 原子或离子的表示

>> ○ a: 质子数 + 中子数

>> ○ b: 质子数

>> ○ c：表示X的充电状态。-2、-、+、+2等。

>> ○ d：表示X存在的个数

> ⑥ 同位素：原子序数相同但中子数不同，因而质量数不同的元素

⑵ 痣

> ① 摩尔：计数原子或分子的单位

> ② 阿伏伽德罗数（N<sub>A</sub>）：一摩尔中的原子数，定义为12克碳12中的原子数

>> ○ N<sub>A</sub> = 6.02 × 10<sup>23</sup>

> ③ 1摩尔气体体积对于所有气体都是恒定的，在STP（0℃，1atm）下为22.4L

>> ○ 源自理想气体定律

> ④ 人类由大约 10,000 颗鼹鼠组成

⑶ 化学式、化学反应方程式和反应产率

> ① 化学式：物质的表示

>> ○ 经验公式：最简单的原子整数比

>>> ○ 示例：葡萄糖的经验式为 CH<sub>2</sub>O>> ○ 分子式：分子中实际原子数

>>> ○ 例：葡萄糖的分子式为C<sub>6</sub>H<sub>12</sub>O<sub>6</sub>

>> ○ 经验式确定分子式：质量百分比组成分析→经验式确定→分子式确定（使用质谱法）

> ② 【结构式】(https://jb243.github.io/pages/1355)

⑷ 化学计量

> ① 弥撒

>> ○ 原子量：一个原子的相对质量。以<sup>12</sup>C = 12.00为标准（无单位）

>> ○ 分子量：一个分子的相对质量。分子中原子的原子量之和（无单位）

>> ○ 实验量：实验式中原子量之和

>> ○ 平均原子量：考虑同位素分布的加权平均值。显示碳平均原子量的计算

> ② 浓度

>> ○ 摩尔浓度：溶质摩尔数 (mol) ÷ 溶液体积 (L)

>> ○ 摩尔浓度：溶质摩尔数 (mol) ÷ 溶剂质量 (kg)

>>> ○ 沸点升高、冰点降低等。

>> ○ 摩尔分数：特定物质的摩尔数 (mol) ÷ 混合物中的总摩尔数 (mol)，无因次量

>> ○ 质量百分比：溶质质量（kg）÷溶液质量（kg）×100（%）

>> ○ ppm（百万分之一）、ppb（十亿分之一）

>>> ○ 1 ppm X = 10<sup>-6</sup> kg X / 1 kg 溶液

>>> ○ 1 ppb X = 10<sup>-9</sup> kg X / 1 kg 溶液

>> ○ ppmv（体积百万分之一）、ppbv（体积十亿分之一）

>>> ○ 1 ppmv X = 10<sup>-6</sup> L X / 1 L 溶液

>>> ○ 1 ppbv X = 10<sup>-9</sup> L X / 1 L 溶液

> ③ 克当量

>> ○ 质量或分子量除以当量

>> ○ 表示1摩尔当量对应的克数

## **3\.化学反应方程式**

⑴ 化学反应方程：用化学式表示化学反应

> ① 反应物

> ② 产品

> ③ 试剂：可供实验室使用的化学物质

> ④ 观察者离子：方程式中所示的不参与反应但存在的离子

>> ○ 排除旁观离子的方程称为净离子方程

> ⑤ 化学反应方程式的书写方法

>> ○ **步骤1：** 用化学式表示反应物和产物

>> ○ **步骤 2：** 骨架方程：写下哪些化学物质发生反应

>> ○ **步骤3：**平衡化学方程式：计算骨架方程中的系数

>> ○ **步骤 4：** 详细化学方程式：包括每种化学物质的状态（例如，_g_、_l_、_s_、_aq_）和条件（_例如_、Δ、hν）

>>> ○ 气态：(_g_)

>>> ○ 液态：(_l_)

>>> ○ 固态：(_s_)

>>> ○ 水溶液状态：(_aq_)

 ⑵ 化学反应方程式定律

> ① 质量守恒定律：除核反应外，所有原子的原子数都必须守恒

> ② 电荷守恒定律：反应物电荷之和等于生成物电荷之和

> ③ 恒定合成定律：元素以固定比例组合

> ④ 多重比例定律：不同元素以整数比结合成一种元素的固定质量

>> ○ 两种元素形成多于一种化合物时除外

⑶ 化学反应方程中的系数计算

> ① 对于氧化还原反应，氧化和还原物质之间氧化数变化的倒数比给出了系数比

>> ○ 通常，氧化还原反应涉及确定氧化数变化的物质的系数，然后平衡 H2O 和 H+ 的系数

> ② 示例>> ○ C<sub>6</sub>H<sub>14</sub> + O<sub>2</sub> → CO<sub>2</sub> + H<sub>2</sub>O

>> ○ 1C<sub>6</sub>H<sub>14</sub> + O<sub>2</sub> → CO<sub>2</sub> + H<sub>2</sub>O

>> ○ 1C<sub>6</sub>H<sub>14</sub> + O<sub>2</sub> → 6CO<sub>2</sub> + H<sub>2</sub>O (**∵** C)

>> ○ 1C<sub>6</sub>H<sub>14</sub> + O<sub>2</sub> → 6CO<sub>2</sub> + 7H<sub>2</sub>O (**∵** H)

>> ○ 1C<sub>6</sub>H<sub>14</sub> + 19/2O<sub>2</sub> → 6CO<sub>2</sub> + 7H<sub>2</sub>O (**∵** O)

>> ○ 2C<sub>6</sub>H<sub>14</sub> + 19O<sub>2</sub> → 12CO<sub>2</sub> + 14H<sub>2</sub>O

⑷ 限制反应物

> ① 在反应中完全消耗的物质

> ② 生成物的量与限制反应物的量成正比

 ⑸ 反应收率百分比（%）=实际收率/理论收率×100

> ① 由于化学平衡和逆反应，反应并不总是完全

 ⑹ 反应的分类

> ① 所有反应都涉及电子运动、断裂和形成键

> ② 电子移动和氧化数变化的反应：[氧化还原反应](https://jb243.github.io/pages/1346)

> ③ 有电子移动但氧化数不变化的反应：[酸碱反应](https://jb243.github.io/pages/1352)。又分为沉淀反应和非沉淀反应
