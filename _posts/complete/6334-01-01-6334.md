## **1\.轨道原子模型**

⑴ Orbital（轨道）：原子轨道、分子轨道等。

> ① 广义：描述原子和分子中电子可能出现的位置的概率分布。也称为波函数。

> ②狭义：图中描绘的边界面是存在概率为90%时的边界面。

> ③ 概率密度用波函数ψ<sup>2</sup>的平方表示。

> ④ 波函数由薛定谔方程得到： 轨道之所以具有特定的形状，是为了满足方程。

⑵ 量子数（quantum number）：表示量子化轨道的状态。

> ①主量子数：用_n_ = 1, 2, ···表示。表示轨道大小、电子能级、电子壳层数等。

>> ○ K（_n_ = 1 的电子层）、L（_n_ = 2 的电子层）、M（_n_ = 3 的电子层）、···

> ② 方位角量子数（角动量量子数）：用ℓ = 0(s), 1(p), 2(d), 3(f),…表示，最多n-1。代表轨道的三维形状。

>> ○ 也代表非简并电子的能态。

> ③ 磁量子数：用mℓ = -ℓ, ···, 0, ···, ℓ 表示。代表围绕原子核的轨道的方向。

> ④ 自旋量子数：用m<sub><i>s</i></sub> = ±1/2表示。表示电子自旋角动量的方向，即自旋方向。

>> ○ 类似轨道相位的概念，即构成轨道的电子旋转方向。

> ⑤ n对应的电子层共有2n<sup>2</sup>个量子态。

⑶ 节点：电子存在概率为0的部分。

> ① 径向节点：与原子核距离恒定的表面，无论方向如何，电子存在的概率都变为 0。

> ② 角动量节点：无论距离如何，沿着特定方向电子存在概率变为0的表面。

>> ○ 角节点的形状和数量因轨道类型而异。

> ③ **节点数量**

>> ○ 径向节点：**_n_ - ℓ __ - 1**

>> ○ 角动量节点：**ℓ**。 s 轨道有 0 个节点，p 轨道有 1 个节点，d 轨道有 2 个节点。

>> ○ 节点总数：**_n_ - 1**

> ④ 波函数ψ(r)与概率密度函数P(r)的关系

>> ○ P(r) = |ψ(r)|<sup>2</sup> = ψ(r)ψ*(r)

> ⑤ 波函数ψ(r)与径向概率分布函数f(r)的关系

⑷ 原子轨道的形状（原子轨道、波函数）

> ① s轨道（ℓ=0）：具有球形电子分布。无角节点。有1个轨道。

>> ○ 原点：夏普轨道>> ○ 波函数的绝对值在原子核附近（r = 0）处**最大**（≠ ∞），概率密度函数为 0。

>> ○ 示例：2s轨道薛定谔方程的解（其中a0为玻尔半径，r为距原点的距离，c为常数）

<br>

> ② p 轨道 (ℓ = 1)：类似于有 2 个叶的哑铃形状。有 3 个相同能级的轨道 (mℓ = -1, 0, 1)。

>> ○ 原点：主轨道

>> ○ 一个叶有正号 (+)，另一叶有负号 (-)。

>> ○ 符号表示相位，并不表示找到电子的概率大小。

>> ○ 相位的符号与分子键合中的相长干涉和相消干涉概念有关。

>> ○ 三个 p 轨道沿 x、y、z 轴取向，表示为 p<sub>x</sub>、p<sub>y</sub>、p<sub>z</sub>。

>> ○ 在原子核附近找到电子的概率收敛于 0。

> ③ d 轨道 (ℓ = 2)：有 4 个波瓣和 5 个相同能级的轨道 (m<sub>ℓ</sub> = -2, -1, 0, 1, 2)。

>> ○ 原点：漫轨道

> ④ f 轨道 (ℓ = 3)

>> ○ 原点：基本轨道

> ⑤ f 以外的轨道按字母顺序命名：g (ℓ = 4) → h (ℓ = 5) → i (ℓ = 6)，依此类推。

⑸ 价电子（最外层电子）

> ① 原子最外层电子数。

> ② 参与化学反应的电子。

⑹ 选择规则：电子跃迁限制

> ①主量子数的变化：Δn也可以为负值

> ② 角量子数变化：Δℓ = ±1

>> ○ 示例：允许从 1s 过渡到 2p，但不允许从 1s 过渡到 2s

> ③ 磁量子数变化：Δm<sub>ℓ</sub> = 0, ±1

> ④ 自旋量子数的变化：跃迁过程中，电子的自旋不发生变化（自旋守恒）

> ⑤ 一般情况下，Δj = 0, ±1，禁止 j = 0 → j = 0 的转变；在特殊情况下，也可能发生 Δj = ±2 的转变。

## **2\.多电子原子** 

⑴ 简并轨道的击穿

> ① 在类氢原子中，能级仅由主量子数决定。然而，对于多电子原子，方位角量子数也有贡献。

> ② **原因1：** 屏蔽效应：随着电子数量的增加，每个电子的有效核电荷发生变化。

> ③ **原因2：** 穿透效应：p电子由于轨道角动量→能量增加而无法接近原子核。

⑵ Aufbau 原理：当轨道充满电子时，是从能量最低的轨道开始填充的。

> ② **注1：**根据Aufbau原理，4s轨道在3d轨道之前就充满了电子，因为它的能量较低。然而，一旦轨道形成，3d 轨道的能量就会低于 4s 轨道，导致电子从 4s 轨道中被移除。>> ○ **示例：** Fe 的基态电子排布为 1s<sup>2</sup>2s<sup>2</sup>2p<sup>6</sup>3s<sup>2</sup>3p<sup>6</sup>3d<sup>6</sup>4s<sup>2</sup>，Fe<sup>2+</sup> 的基态电子排布为1s<sup>2</sup>2s<sup>2</sup>2p<sup>6</sup>3s<sup>2</sup>3p<sup>6</sup>3d<sup>6</sup>。

>> ○ **原因 1：** 4s 和 3d 轨道具有相似的能量。

>> ○ **原因 2：** 4s 轨道中的电子-电子斥力不稳定 → 将电子划分到 d 轨道提供了稳定性。

⑶ 泡利不相容原理

> ① 对于单个原子中的多个电子，同一组四个量子数不能相同。

>> ○ 四个量子数：主量子数、方位量子数、磁量子数、自旋量子数。

> ② 最多2个电子可以占据一个轨道：如果有2个电子，它们的自旋必须相反。

⑷ 洪德法则：当存在多个简并轨道时，不成对电子越多，稳定性越高。

> ① 简并性：能量水平相等。

> ② 顺磁性：一个或多个不成对电子。与外部磁场发生反应。

>> ○ 原因：电子的循环构成电流，在外磁场作用下产生[洛伦兹力](https://jb243.github.io/pages/751#footnote_link_67_52)。

> ③ 抗磁性：没有不成对的电子。不与外部磁场发生反应。

> ④ 如果不考虑原子的基态，洪德法则可能不满足。

> ⑤ 对于类氢原子，简并性仅由主量子数_n_决定。

> ⑥ 对于 sp<sup>3</sup>d<sup>2</sup> 轨道和高自旋情况，类似的洪德规则适用。

## **3。原子键理论（VBT、价键理论）**

⑴ 概述：路易斯结构-价层电子对排斥（VSEPR）理论-**价键理论（VBT）**-分子轨道理论

> ① 定义：将轨道应用于路易斯结构，VSEPR。

> ②近似量子论。

⑵ 假设：一个原子的不同轨道杂化，形成杂化轨道成键。

> ①补充：原子内的原子轨道由于其波状特性而可以混合。

⑶ σ键、π键

> ① σ键和π键的概念从原子键理论开始出现。

> ② σ 键：没有包含键合轴的节面的键。

>> ○ 示例：s-s、s-p、pz-pz

> ③ π 键：与包含键合轴的节面键合。

>> ○ 示例：p<sub>x</sub>-p<sub>x</sub>、p<sub>y</sub>-p<sub>y</sub>

> ④ σ、π键的键强度

>> ○ 没有节面的σ键为强键合，有节面的π键为弱键合。

>> ○ 单键：1 个σ 键。键合一定会带来好处，所以总是σ键。

>> ○ 双键：1个σ键、1个π键。键合强度小于单键的两倍。

>> ○ 三键：1个σ键，2个π键。粘合强度小于单键的三倍。

>> ○ π 键需要较弱且紧密相连 → 从第三个时期开始，很少观察到多重键。

>>> ○ C、N、O易形成多重键。

⑷ sp<sup>3</sup> 混合轨道：四面体（例如乙烷）

> ① 3 p 轨道杂化形成四面体取向。

> ② h1 ~ h4 轨道的形状可以直接用公式验证。

>> ○ h<sub>1</sub> 轨道： ½ (s + p<sub>x</sub> + p<sub>y</sub> + p<sub>z</sub>)

>> ○ h<sub>2</sub> 轨道： ½ (s + p<sub>x</sub> - p<sub>y</sub> - p<sub>z</sub>)

>> ○ h<sub>3</sub> 轨道： ½ (s - p<sub>x</sub> + p<sub>y</sub> - p<sub>z</sub>)>> ○ h<sub>4</sub> 轨道： ½ (s - p<sub>x</sub> - p<sub>y</sub> + p<sub>z</sub>)

> ③ 示例：**CH<sub>4</sub>**

>> ○ (_initial_) C原子轨道: 2s<sup>2</sup> 2px<sup>1</sup> 2py<sup>1</sup> 2pz<sup>0</sup> → 不适合有2个键合位点

>> ○ (_promotion_) C 原子轨道： 2s<sup>1</sup> 2px<sup>1</sup> 2py<sup>1</sup> 2pz<sup>1</sup> → 不适合沿轴的轨道

>> ○ (_hybridization_) C 原子轨道： 2h<sub>1</sub><sup>1</sup> 2h<sub>2</sub><sup>1</sup> 2h<sub>3</sub><sup>1</sup> 2h<sub>4</sub><sup>1</sup> → 4 个键合位点。适合预测形状

⑸ sp<sup>2</sup> 混合轨道：三角平面（例如乙烯）

> ① 双键中一个是σ键，另一个是π键。

> ② 两个 p 轨道杂化，形成三角平面几何。

⑹ sp杂化轨道：线性（例如乙炔、二氧化碳）

> ① 三键中，一个是σ键，另外两个是π键。

> ② 一个 p 轨道被杂化，形成线性几何。

> ③ 示例：乙炔

## **4。分子轨道理论 (MOT)**

⑴ 概述：路易斯结构-VSEPR理论-价键理论-**分子轨道理论（MOT）**

> ①价键理论的局限性

>> ○ 假设：原子电子驻留在单个原子上。

>> ○ **限制1：** 无法预测分子的顺磁性/抗磁性：路易斯结构预测O2 具有抗磁性，但它是顺磁性的。

>>> ○ 氧具有顺磁性，MOT 预测由于不成对电子而产生的顺磁性。

>> ○ **限制 2：** 无法预测键合行为：例如，为什么惰性气体不形成键合？

> ② 特点

>> ○ 全量子理论

>> ○ 所有原子电子都对整个分子做出贡献，而不仅仅是单个原子。

⑵ **化简：** LCAO-MO（原子轨道-分子轨道的线性组合）

> ①定义：分子轨道由原子轨道重叠的线性组合形成的近似。

> ② 最知名的分子轨道近似方法。

> ③ 将分子轨道近似表示为构成分子的原子轨道的组合（加法、减法）。

> ④ **分子轨道数** = 形成分子的原子轨道总和。

> ⑤ **原子电子数** = 填充分子轨道的电子数。

⑶ 分子轨道的一般概念

> ① 基于对成键贡献的分子轨道类型：

>> ○ 类型 1：键合分子轨道 (BO) - 能量低于原子轨道

>>> ○ 由同相原子轨道的增强干涉构成。

>>> ○ 与键合相互作用有关

>>> ○ 示例： ψ<sub>σ</sub> = ψ<sub>A1s</sub> + ψ<sub>B1s</sub>

>> ○ 类型 2：反键分子轨道 (ABO) - 比原子轨道能量更高

>>> ○ 消除不同相位原子轨道的干扰而构建。

>>> ○ 与粘合时的斥力有关

>>> ○ 含有阻碍粘合的节点

>>> ○ 示例： ψ<sub>σ*</sub> = ψ<sub>A1s</sub> - ψ<sub>B1s</sub>

>> ○ 类型 3：非键合分子轨道 (NBO) - 与原子轨道能级相同

> ② 基于成键的分子轨道类型：

>> ○ σ 轨道（西格玛键）：没有包含键合轴的节面

>>> ○ 示例：1s<sub>A</sub> + 1s<sub>B</sub> → σ<sub>1s</sub> + σ<sub>1s</sub>*

>>> ○ 示例：2s<sub>A</sub> + 2s<sub>B</sub> → σ<sub>2s</sub> + σ<sub>2s</sub>*

>> ○ π 轨道（pi 键）：包含键合轴的节面

>>> ○ 示例：2p<sub>x</sub> + 2p<sub>x</sub> → π<sub>2px</sub> + π*<sub>2px</sub>>>> ○ Pi 键合通常涉及未杂化 p 轨道的重叠。

>>> ○ Pi 键总是与 Sigma 键形成，因为键合良好。

>>> ○ 由于电子排斥，Pi 键合优于双西格玛键的形成。

>>> ○ 如果 pi 键断裂，分子就会变得太不稳定，因此多个键不会旋转。

>>> ○ 对于第三周期及以后的元素，距离越大，π 键合越困难，导致双键较少。

> ③ LUMO和HOMO

>> ○ LUMO（最低未占据分子轨道）：最低能量未占据分子轨道

>> ○ HOMO（最高占据分子轨道）：最高能量占据分子轨道

>> ○ 带隙能量：LUMO 和 HOMO 之间的能量差

>>> ○ 与该能量跃迁相对应的光在基态下发射

>>> ○ LUMO 的能量比 HOMO 高

> ④ 保证金订单

>> ○ 键阶 = (键合电子数 - 反键电子数) / 2

>> ○ 非键轨道中的电子不影响键序

>> ○ 较高的键序导致较短的键长和较高的键能

>> ○ 比较键长和强度时比较键阶

> ⑤ 分子轨道节点数：**提示**可以直接画出来

>> ○ σ<sub>2s</sub> 有 0 个节点，σ<sub>2s</sub>* 有 1 个节点

>> ○ σ<sub>2p</sub> 有 2 个节点，σ<sub>2p</sub>* 有 3 个节点

>> ○ π<sub>2p</sub> 有 1 个节点，π<sub>2p</sub>* 有 2 个节点

>> ○ 根据泡利不相容原理，每个轨道最多可容纳 2 个电子

> ③ **情况2：** 第二周期双原子分子：X<sub>2</sub> 型分子

>> ○ 轨道能级随着原子序数和的增加而降低

>> ○ **sp轨道混合**

⑸ [休克尔近似](https://www.youtube.com/watch?v=oT71KYCPSJE)（休克尔方法）

> ①定义：预测环状或无环多烯MO能量和共振能量的方法。> ② 基本方程：ㅣH<sub>ij</sub> - E S<sub>ij</sub>ㅣ<sub>n×n</sub> = 0（其中ㅣ·ㅣ表示行列式）

>> ○ n: MO 数量

>> ○ 为简化起见，假设 i ≠ j 时 S<sub>ii</sub>=1 且 S<sub>ij</sub>=0

>> ○ H<sub>ij</sub>：第i个碳对第j个碳位置的影响

>> ○ H<sub>ii</sub> = α（库仑积分）。 α 是碳 2p 轨道的能量（大约 -11 eV 或 -1100 kJ/mol）

>> ○ H<sub>ij</sub> = β（i ≠ j）（共振积分）。对于多烯，β 约为 α 的 10%

>> ○ α, β < 0 : 代表负势能

> ③ 能量计算

>> ○ 总能量：从基本方程得到的最低能量开始填充MO电子，然后求和。

>> ○ π-电子能量 = (双键数) × (2α + 2β)

>> ○ 离域（共振）能量 = 总能量 − π 电子能量 −（孤立电子数）× α
