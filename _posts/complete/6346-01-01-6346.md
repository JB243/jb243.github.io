## **1.** **氧化值** 

 ⑴ 定义：假设电负性较高的元素夺走了所有共享电子对，则该元素失去所有共享电子对时所具有的假设电荷。

 ⑵ 各原子的氧化数=该原子的电荷+该原子的价电子-键数-未共享电子数

> ① 分数氧化数代表平均氧化态

 ⑶ 自然界中，原子的氧化数是标准化的：优先顺序为①＞···＞⑦

> ① 构成中性化合物的原子氧化数之和为0。

> ② 组成离子的原子氧化数之和等于离子的电荷。

> ③ 氟的氧化数始终为-1。

> ④ 碱金属为+1，碱土金属为+2，铝为+3。

> ⑤ 氢为+1。

>> ○ 例外：在 LiH 中，H 的氧化数为 -1。

> ⑥ 氧气为-2。

>> ○ 例外 1. 在 OF2 中，O 的氧化数为 +2。

>> ○ 例外 2。H2O2 (-1)、O2- (-1/2)、O3- (-1/3)。

> ⑦ 除氟以外的卤素元素为-1。

>> ○ 例外：与氧气（例如 OCl2）结合时，其值为 +1。

## **2.氧化还原反应**

 ⑴氧化：原子、分子、离子等失去高能电子。

> ① 表达式 1. 氧化数增加

> ② 表达式2.获得氧气

>> ○ 反应前：O2 中每个 O 原子的氧化数均为 0。

>> ○ 反应后：分子中O的氧化数变为-2，其他物质的氧化数增加。

> ③式3.失去氢

>> ○ 反应前：一般分子中H的氧化数为+1。

>> ○ 反应后：产生每个原子氧化数为0的H2，导致其他物种氧化数增加。

> ④ 表达式4. 损失能量的反应：由于高能电子的损失。

 ⑵ 还原：原子、分子、离子等获得高能电子。

> ① 表达式 1. 氧化数减少

> ② 表达式2. 失去氧气

>> ○ 反应前：一般分子中O的氧化数为-2。

>> ○ 反应后：产生每个原子氧化数为0的O2，导致其他物种氧化数减少。

> ③式3.获得氢

>> ○ 反应前：H2 中每个 H 原子的氧化数均为 0。

>> ○ 反应后：分子中H的氧化数变为+1，其他物质的氧化数减少。

> ④ 表达式4. 获得能量的反应：由于高能电子的获得。

 ⑶同时性：氧化和还原总是同时发生，氧化数的变化量等于转移电子数。

 ⑷ 与酸碱反应不同，氧化还原反应涉及电子的直接运动。

> ① [电子活性](https://jb243.github.io/pages/1506)（电子活性）：电子的浓度。

## **3。氧化剂和还原剂**

 ⑴ 定义

> ① 氧化剂：自身被还原同时又氧化另一种物质的物质。

> ② 还原剂：自身被氧化同时还原另一种物质的物质。

 ⑵ 氧化剂的相对强度> ① 获得电子的倾向越强（第17族），氧化剂越强。

> ② 含有高氧化值金属或非金属的化合物

>> ○ 示例：在K2Cr2O7中，Cr的氧化数为+6，使其成为强氧化剂。

> ③ 如果一种元素可以有多个氧化数，氧化数越大，氧化剂越强。

>> ○ KMnO4 > MnO2 > Mn2O3 > MnCl2

>> ○ KMnO4：由于Mn的氧化数为+7，因此它会氧化其他物质以达到较低的氧化态。

> ④ 如果氧化态比平常低，则该物质可以起到氧化剂的作用。

>> ○ 过氧化氢：H2O2中O的氧化数为-1，因此氧化其他物质达到-2。

 ⑶ 还原剂的相对强度

> ①失去电子的倾向越强（元素周期表左下），还原剂越强。

> ② 含有低氧化值金属或非金属的化合物

> ③ 如果一种元素可以有多个氧化数，则氧化数越低，还原剂越强。

>> ○ 示例：H2S (-2) > S (0) > SO2 (+4) > SO3 (+6)

 ⑷ 氧化剂和还原剂的相对性质：根据反应的不同，物质可以充当氧化剂或还原剂。

> ① **实施例1：** 歧化反应：A → A' + A"，其中A'和A"具有不同的氧化态。

>> ○ 4SO2 + 3H2O → S2O32- + 2HSO4- + 4H+

## **4。配平化学方程式**

 ⑴ **氧化数法**

> ①定义：利用氧化数的增加等于氧化数的减少这一事实。

> ② 第一<sup>第一</sup>。写出骨架方程

MnO4-(aq) + H2C2O4(aq) → Mn2+(aq) + CO2(g)

> ② 第二<sup>第二</sup>。计算氧化骨架方程的氧化数 (x) 变化

>> [ **·** ]表示的碳氧化数：H2C2O4(aq) [+6] → 2CO2(g) [+8]

>> x = +2

> ③ 第三<sup>第</sup>。计算还原骨架方程的氧化数 (y) 变化

>> [ **·** ]表示的锰的氧化值： MnO4-(aq) [+7] → Mn2+(aq) [+2]

>> y = -5

> ④ 第 4<sup></sup>。将氧化和还原骨架方程乘以适当的因子，使两个方程的氧化数变化等于 x 和 y (10) 的最小公倍数，然后将它们相加

>> 2MnO4-(aq) + 5H2C2O4(aq) → 2Mn2+(aq) + 10CO2(g)

> ⑤ 第 5<sup></sup>。在酸性环境中，用H2O平衡O原子数，用H+平衡H原子数

>> ○ 第 5<sup>位</sup> - 第 1<sup>位</sup>。

>>> 2MnO4-(aq) + 5H2C2O4(aq) → 2Mn2+(aq) + 10CO2(g) + 8H2O(ℓ)

>>> 2MnO4-(aq) + 5H2C2O4(aq) + 6H+(aq) → 2Mn2+(aq) + 10CO2(g) + 8H2O(ℓ)

>> ○ 第 5<sup>第</sup> - 第 2<sup>第</sup>。检查：检查是否保持费用平衡

>> ○ 第 5<sup> </sup> - 第 3<sup></sup>。电荷平衡自然是通过④中的过程来维持的。

> ⑥ 第六<sup>th</sup>。在碱性环境中，用H2O平衡O原子数，用(H2O - OH-)平衡H原子数

>> ○ 第 6<sup>位</sup> - 第 1<sup>位</sup>。

>>> 2MnO4-(aq) + 5H2C2O4(aq) → 2Mn2+(aq) + 10CO2(g) + 8H2O(ℓ)

>>> 2MnO4-(aq) + 5H2C2O4(aq) + 6H2O(ℓ) - 6OH-(aq) → 2Mn2+(aq) + 10CO2(g) + 8H2O(ℓ)

>>> 2MnO4-(aq) + 5H2C2O4(aq) → 2Mn2+(aq) + 10CO2(g) + 2H2O(ℓ) + 6OH-(aq)

>> ○ 第 6<sup>第</sup> - 第 2<sup>第</sup>。检查：检查是否保持费用平衡

>> ○ 第 6<sup> </sup> - 第 3<sup></sup>。电荷平衡自然是通过④中的过程来维持的。

 ⑵ 离子电子法：求出氧化半方程和还原半方程，然后将两个半方程相加。
