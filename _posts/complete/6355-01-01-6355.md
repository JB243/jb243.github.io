## **2\.有机化学结构式**

⑴ 顺序

> ① SMILES（简化分子输入行输入系统）：短 ASCII 字符串表示形式。

>> ○ 双键用“=”表示，三键用“#”表示。

>> ○ 对于环状化合物，分配诸如 1、2 等数字表示线性分子的末端连接形成环。

>>> ○ 示例：CN1C=NC2=C1C(=O)N(C(=O)N2C)C（= 咖啡因）

>> ○ C代表正碳，c代表芳香碳

>>> ○ C1CCCCC1: 环己烷 

>>> ○ c1ccccc1: 苯 

>> ○ 括号可用于表示更复杂的情况。

>>> ○ 还可以显示电荷符号，如[N+]

>> ○ @ 符号可用于表示分子中的立构中心。

>>> ○ 示例：C[C@H](O)[C@H](O)C

>> ○ / 和 \ 符号可用于表示 E/Z 异构体。

>>> ○ 示例：CCC/C(=C/C(=O)OCC)/C(=O)OCC

> ② SMARTS（SMILES任意目标规范）：用于指定分子结构模式的语言。

>> ○ 示例：[#6]-[#7]1:[#6]:[#7]:[#6]2:[#6]:1:[#6](=[#8]):[#7](:[#6](=[#8]):[#7]:2-[#6])-[#6]

>> ○ 参考：_Daylight Chem Info Systems_

> ③ SELFIES（自引用嵌入字符串）

>> ○ 示例：[C][N][C][=N][C][=C][Ring1][Ring2][C][Branch1_3][epsilon][=O][N][Branch1_3][Branch2_2][C][Branch1_3][epsilon][=O][N][Ring1][Branch1_3][C][C]

>> ○ 参考：_NIPS_，2019

> ④ InChI

>> ○ 示例：1S/C9UBO4/c1-6(10)13-8-S-3-2-4-7(8)9(11)12/h2-5H,1H3,(H,11,12)

>> ○ C9UBO4：化学式

>> ○ c1-6(10)13-8-S-3-2-4-7(8)9(11)12: 原子连接

>> ○ h2-5H,1H3,(H,11,12): 氢原子

> ⑤ InChIKey

>> ○ InChI 的缺点：可能很长

>> ○ InChIKey 将 InChI 压缩为使用 27 个大写字母的数字表示形式，旨在方便网络搜索。

> ⑥ 嵌入向量

>> ○ 根据所使用的压缩算法的类型，可以存在各种嵌入向量。

>> ○ 参考文献：_自然生物科技_，2020

## **4\.共振结构**

⑴ 概述

> ① 定义：由π电子或非成键电子对驱动的电子离域。

> ② 一般来说，每个原子的杂化结构在共振贡献者之间保持不变。

>> ○ 示例：-COOH 中每个氧的杂化轨道是 sp2 和 p 轨道。

> ③ 可分为涉及三个或三个以上原子非键合电子的共振和涉及两个原子非键合电子的共振。

> ④ 电子从富电子区移动到缺电子区。

> ⑤ 对于两个原子的非键电子的苯酰化机制很重要。

⑵ 共振结构的类型

> ①烯丙基孤对、自由基或其他双键：2个弯曲箭头

> ② 烯丙基正电荷：1个弯曲箭头

> ③ 正电荷附近的一对电子：1个弯曲箭头

> ④ 两个不同电负性原子之间的Pi键

> ⑤ 环状结构内的共轭π键

> ① 没有单一结构能够准确地代表共振结构。

> ② 这种单独的结构称为共振或共振贡献者。

> ③ 真实的分子被理解为这些结构的混合物。

> ②根据X射线研究，碳酸根离子中所有C-O键长相同；它们比双键长，比单键短。

⑷ 绘制共振结构

> ① 共振结构或贡献者用双头箭头 ↔ 表示。

>> ○ 化学平衡用⇌表示。

> ② 共振结构仅由电子运动形成。

>> ○ 1,2-氢化物位移不是共振现象。

> ③ 只有π键电子或非键电子对才能形成共振结构。

> ④ 分子内电子总数必须保持恒定。

> ⑤ 需要破坏西格玛键的共振结构是不合理的。

> ⑥ 第二周期元素不能容纳超过八个电子。

> ⑦ 氢原子最多只能有两个电子。

⑸ 贡献

> ① 共振贡献：每个结构在共振贡献者中受青睐的程度。

>> ○ 主要贡献者：与实际结构最相似的共振贡献者。

>> ○ 次要贡献者：与实际结构不太相似的共振贡献者。

> ② 第一<sup>第一</sup>。八位组规则：违反八位组规则的共振贡献者总是具有最低的贡献。

>> ○ 示例：碳阳离子的共振贡献者可能是次要贡献者。

>> ○ 对于第二周期原子，价层电子数不超过 8 个。

> ③ 第二<sup>第二</sup>。电荷分离：没有电荷分离的共振贡献者具有更高的贡献。

> ④ 第三<sup>第</sup>。负电荷处的电负性越高，贡献越大。

> ⑤ 第 4<sup></sup>。如果②~④之间没有差异，则键越多的共振贡献者的贡献就越高。

>> ○ **示例1.** [扎伊采夫规则](https://jb243.github.io/pages/1363)

>> ○ **示例2.** [给电子感应效应](https://jb243.github.io/pages/1355)

## **5。酸碱理论**

⑴ 所有反应分为酸碱反应（氧化数变化）和氧化还原反应（氧化数不变）。

⑵ 普通化学：【酸碱理论】(https://jb243.github.io/pages/1352)

⑶ 有机化学：路易斯酸和碱

> ①路易斯酸和碱：从热力学角度来说，亲核试剂和亲电子试剂的反应速率存在差异。

>> ○ 在有机化学中，路易斯酸被称为亲电子试剂，路易斯碱被称为亲核试剂。

> ② 亲核试剂（Nu）：喜欢原子核的物种。反应速率相关。

>> ○ 示例：富电子负离子、δ<sup>-</sup>、碳-碳 pi 键。

>> ○ 尽管碱性较强，但由于空间位阻，亲核性可能会降低。

>>> ○ 碱度：叔丁醇 > 乙醇

>>> ○ 亲核性：乙醇盐 > 叔丁醇盐（由于空间位阻）

> ③ 亲电体（E）：喜欢电子的物种。反应速率相关。

>> ○ 示例：缺电子正离子，δ+。主要是碳δ+。

>> ○ Br<sub>2</sub> 是亲电子试剂的原因。

<br>

![图片](https://github.com/JB243/jb243.github.io/assets/55747737/57048f9f-aee7-4157-a1fc-37c9b00f57be)

<br>

> ④ 碱度作为热力学概念的意义

>> ○ 热力学与平衡态（即最终状态）有关。

>> ○ 就平衡常数 K<sub​​>b</sub> 而言，较高的碱度对应于较低的 pK<sub​​>b</sub>。

>> ○ 也就是说，碱度表明哪种物质在最终状态下更有效地吸引质子。

> ⑤ 亲核性作为反应速率概念的意义

>> ○ 与哪个物种最初反应更快有关。

>> ○ 示例：由于空间位阻，t-BuOK 的反应速度比 EtOK 慢，因此亲核性较低。

⑷ **碱性：**接受H+的程度。酸度和碱度并不是绝对的。> ① 概述：反应性↑→稳定性↓→碱度↑

>> ○ 强酸的共轭碱是弱碱；弱酸的共轭碱是强碱。

<br>

![图片](https://github.com/JB243/jb243.github.io/assets/55747737/dc8418ed-b421-4823-ace4-000825aaa786)

<br>

>> ○ 酸性和碱性是热力学的（存在可逆反应），而亲核和亲电是动力学的（不可逆，无逆反应）。

>>> ○ 示例：t-BuOK 是强碱，但是弱亲核试剂。

>> ○ 碱度因素

>>> ○ **因素1：** 感应效应：感应效应、电负性、共振、氢键、尺寸效应。

>>> ○ **因素 2：** 共振效应：由于共振、极化、主链键合、芳香效应而稳定。

>>> ○ **因素 3：** 其他效应：空间位阻、溶剂化效应。

>>> ○ 引入考虑感应效应和谐振效应的[EDG（给电子基团）和EWG（吸电子基团）](https://jb243.github.io/pages/1370#7-substituent-effects)概念。

> ② **1-1.** **感应效应：** 通过σ键吸电子或给予电子。

>> ○ 定义：通过σ键影响电子密度的效应。

>> ○ 区分给电子和吸电子感应效应。

>> ○ 官能团越多，碳正离子越稳定，影响反应速率。

>> ○ 示例：

>>> ○ CH<sub>3</sub>COOH：CH<sub>3</sub>基团的给电子作用导致-COO<sup>-</sup>基团上的负电荷不稳定。

>>> ○ CF<sub>3</sub>COOH：CF<sub>3</sub>基团的吸电子作用稳定了-COO<sup>-</sup>基团上的负电荷。

>>> ○ CF<sub>3</sub>COO<sup>-</sup> 比 CH<sub>3</sub>COO<sup>-</sup> 更稳定，导致碱性更低，因此共轭酸更强。

>>> ○ 酸度比较：CF<sub>3</sub>COOH > CH<sub>3</sub>COOH

>>> ○ 实际 pKa 值：CF<sub>3</sub>COOH pK<sub>a</sub> = 0, CH<sub>3</sub>COOH pK<sub>a</sub> = 5

>>> ○ 酸度比较： (CH<sub>3</sub>)<sub>3</sub>COH (pK<sub>a</sub> = 18) < CH<sub>3</sub>CH<sub>2</sub>OH (pK<sub>a</sub> = 16) < H<sub>2</sub>O (pK<sub>a</sub> = 15.7)

> ③ **1-2.** **电负性：** 随着基团数量的增加而增加，导致稳定性增加和碱度降低。

>> ○ 对于同一周期的元素，电负性较大的原子带负电则稳定，带正电则不稳定。

>> ○ 碱度： F<sup>-</sup> < OH<sup>-</sup> < NH<sub>2</sub><sup>-</sup> < CH<sub>3</sub><sup>-</sup> （由于电负性）

>> ○ 酸度：HF (pK<sub>a</sub> = 3.2) > H<sub>2</sub>O (pK<sub>a</sub> = 15.7) > NH<sub>3</sub> (pK<sub>a</sub> = 38) > CH<sub>4</sub> (pK<sub>a</sub> = 50)

> ④ **1-3.** **混合效应**：随着 s 字符的增加，酸度增加

>> ○ **超共轭**

>>> ○ 定义：σ键向反π键提供电子，使化合物稳定的现象。

>>> ○ 按照 sp ＞ sp<sup>2</sup> ＞ sp<sup>3</sup> 的顺序，电负性较强，因此 sp<sup>3</sup> 碳向 sp 和 sp<sup>2</sup> 碳提供电子。

>> ○ 原因：按 sp ＞ sp<sup>2</sup> ＞ sp<sup>3</sup> 的顺序，具有 s 性质的球形轨道更强，更接近原子核 → 电负性增加。

>>> ○ sp 杂化：2 个杂化轨道中，1 s 轨道 → 50 % s 特征>>> ○ sp<sup>2</sup> 杂化：3个杂化轨道中，1 s轨道→ 33 % s-字符

>>> ○ sp<sup>3</sup> 杂化：4 个杂化轨道中，1 s 轨道 → 25 % s 字符

>> ○ **示例 1.**

>>> ○ 碱度：CH<sub>3</sub>CH<sub>2</sub><sup>-</sup> > C<sub>2</sub>H<sub>3</sub><sup>-</sup> > C<sub>2</sub>H<sup>-</sup>

>>> ○ 酸度：CH<sub>3</sub>CH<sub>3</sub> (pK<sub>a</sub> = ~50) < C<sub>2</sub>H<sub>4</sub> (pK<sub>a</sub> = 44) < C<sub>2</sub>H<sub>2</sub> (pK<sub>a</sub> = 25)

> ⑥ **1-5.** 尺寸效应

>> ○ 较大的极化（∝尺寸）导致更好的亲核性，使得在过渡态下更容易与 σ* 重叠。

>>> ○ 与 **2-2** 解释相反

>> ○ **实施例1.** 亲核性：H<sub>2</sub>O ＜ H<sub>2</sub>S ＜ H<sub>2</sub>Se

>> ○ **实施例2.** 亲核性：NH<sub>3</sub> ＜ PH<sub>3</sub>

> ⑦ **2-1.** **共振稳定效果：** 共振结构增加稳定性，降低碱度。

>> ○ 多个共振贡献者导致能量稳定。

>> ○ [盒子中的粒子](https://jb243.github.io/pages/1329#4-article-in-a-box-generally-article-in-one-Dimension-box)：电子离域会降低能量 (∝ 1 / L<sup>2</sup>)。

>>> ○ 示例：羧酸阴离子由于共振而稳定且无害。

>> ○ 碱度：CH<sub>3</sub>CH<sub>2</sub>O<sup>-</sup> > CH<sub>3</sub>COO<sup>-</sup>（由于共振稳定）

>> ○ 酸度：CH<sub>3</sub>CH<sub>2</sub>OH (pK<sub>a</sub> = 16) < CH<sub>3</sub>COOH (pK<sub>a</sub> = 5)

> ⑧ **2-2.** **极化：** 更多的电子壳层导致电子-电子排斥力降低，增加稳定性并降低碱度。

>> ○ 主要适用于HX化合物。

>> ○ 碱度：F<sup>-</sup> > Cl<sup>-</sup> > Br<sup>-</sup>

>> ○ 酸度：HF (pK<sub>a</sub> = 3.2) < HCl (pK<sub>a</sub> = -7) < HBr (pK<sub>a</sub> = -9)

> ⑨ **2-3.** **背键合：** p 轨道电子与空的 d 轨道相互作用，稳定分子。>> ○ 示例：CHF<sub>3</sub> 和 CHCl<sub>3</sub> 的比较

>> ○ 酸度：CHF<sub>3</sub> < CHCl<sub>3</sub>

>> ○ 共轭碱基稳定性：CF<sub>3</sub><sup>-</sup> < CCl<sub>3</sub><sup>-</sup>

>> ○ 解释：Cl具有可以放置电子的3d轨道，因此可以通过在Cl的3d轨道中离域来稳定来自碳2p轨道的多余电子。

>> ○ CF<sub>3</sub><sup>-</sup> 中的吸电子诱导效应比 CCl<sub>3</sub><sup>-</sup> 更强，这可能会误导共轭碱基的稳定性。

>> ○ 记住酸度的趋势：HF < HCl < HBr < HI，这使得理解背键合更容易。

> ⑩ **2-4.** **芳香效果**

>> ○ 一种共振稳定效应。

>> ○ 如果共轭碱具有芳香性，则由于共振而产生的特殊稳定作用会导致较高的酸度。

>> ○ 如果共轭碱是反芳香族的，则它非常不稳定，导致酸度较低。

> ⑪ **3-1.** **空间位阻**

>> ○ 酸度：乙醇 (pK<sub>a</sub> = 16) > 叔丁醇 (pK<sub>a</sub> = 18)

>> ○ 碱度: 乙醇盐 < 叔丁醇盐

>> ○ 如果亲核基团较大，叔丁醇的空间位阻可以克服其在溶剂中的稳定性，导致乙醇醇的亲核性更强。

> ⑫ **3-2.** **溶剂化效应**

>> ○ 定义：分离的共轭碱被周围溶剂稳定化的现象。

>> ○ 注意：碱度不受影响，仅影响亲核性。

>> ○ 介电常数：介电常数越高，极性越强，溶剂化效果增强。

>>> ○ 与极性非质子溶剂（例如 DMSO、DMF）相比，极性质子溶剂（例如水）具有更高的介电常数。

>> ○ 正质子溶剂：较小尺寸的负电性原子更容易稳定。较大的原子具有较小的电子密度，导致离子-溶剂相互作用较弱。

>>> ○ 碱度： I<sup>-</sup> < Br<sup>-</sup> < Cl<sup>-</sup> < F<sup>-</sup>

>>> ○ 亲核性： I<sup>-</sup> > Br<sup>-</sup> > Cl<sup>-</sup> > F<sup>-</sup>

>>> ○ 碱度：SeH<sup>-</sup> < SH<sup>-</sup> < OH<sup>-</sup>

>>> ○ 亲核性：SeH<sup>-</sup> > SH<sup>-</sup> > OH<sup>-</sup>

>>> ○ 碱度：叔丁醇 > 乙醇（由于空间位阻）

>>> ○ I<sup>-</sup> 是水中良好的离去基团和亲核试剂。

>>> ○ 对于同一周期的元素，电负性决定趋势。 CH<sub>3</sub><sup>-</sup> > H<sub>2</sub>N<sup>-</sup> > HO<sup>-</sup> > F<sup>-</sup>

>> ○ 非极性非质子溶剂：由于缺乏分子内氢键，亲核性遵循碱性趋势。

>>> ○ 碱度： I<sup>-</sup> < Br<sup>-</sup> < Cl<sup>-</sup> < F<sup>-</sup>

>>> ○ 亲核性： I<sup>-</sup> < Br<sup>-</sup> < Cl<sup>-</sup> < F<sup>-</sup>

>>> ○ 碱度：SeH<sup>-</sup> < SH<sup>-</sup> < OH<sup>-</sup>

>>> ○ 亲核性： SeH<sup>-</sup> < SH<sup>-</sup> < OH<sup>-</sup>

>> ○ 与非极性非质子溶剂相比，I<sup>-</sup> 在质子溶剂中显示出更高的碱性。

>>> ○ 溶剂氢键会增加碱度，但会降低稳定性。
