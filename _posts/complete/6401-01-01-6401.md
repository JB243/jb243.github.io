## 1\.氧化还原反应

 ⑴ 氧化剂：自身被还原并氧化其他物质的物质

> ① K<sub>2</sub>Cr<sub>2</sub>O<sub>7</sub>

> ② H<sub>2</sub>CrO<sub>4</sub>

> ③ CrO<sub>3</sub>

> ④ 高锰酸钾<sub>4</sub>

> ⑤ 琼斯试剂：CrO<sub>3</sub> / 水溶液。 H<sub>2</sub>SO<sub>4</sub>

> ⑥ PCC：温和氧化剂

>> ○ 伯醇 → 醛

>> ○ 仲醇 → 酮

>> ○ 叔醇 → 无反应

> ⑦ DMP（戴斯-马丁氧化剂）

> ⑧ 次氯酸

> ⑨ NaOCl / CH3COOH / 0 ℃

> ⑩ Swern氧化试剂

> ⑪ 柯林斯试剂

> ⑫ Magtrieve 氧化剂 (CrO<sub>2</sub>)

> ⑬ 节奏

> ⑭ Ag<sub>2</sub>CO<sub>3</sub>

> ⑮ NaIO<sub>4</sub>

⑵ 还原剂：自身被氧化并还原其他物质的物质

> ① 概述

>> ○ 还原剂通常是亲核试剂并改变氧化态

>> ○ NaH 和类似化合物是碱，不会改变氧化态

> ② H<sub>2</sub> / Pt、Pd 或 Ni

> ③ [Lindlar催化剂](https://jb243.github.io/pages/1363#:~:text=syn%20%EC%B2%A8%EA%B0%80%20%EB%B0%98%EC%9D%91%20%3A-,Lindlar,-%EC%B4%89%EB%A7%A4%20\(P%2D2)

>> ○ 催化剂由Pd、CaCO<sub>3</sub>（或BaSO<sub>4</sub>）、乙酸铅（Ⅱ）（（CH<sub>3</sub>COO<sup>-</sup>）<sub>2</sub>Pb<sup>2+</sup>）、喹啉等组成。

> ④ LiAlH<sub>4</sub> 和 NaBH<sub>4</sub>

>> ○ LiAlH<sub>4</sub> (LAH)：强还原剂

>>> ○ 羧酸 → 伯醇

>>> ○ 醛 → 伯醇

>>> ○ 酮 → 仲醇

>>> ○ 考虑还原反应中的空间位阻

>> ○ NaBH<sub>4</sub>：弱还原剂

>>> ○ 羧酸 → 无反应

>>> ○ 醛 → 伯醇

>>> ○ 酮 → 仲醇

>> ○ 为什么LiAlH<sub>4</sub>比NaBH<sub>4</sub>强

>>> ○ **原因1：** 与B不同，Al处于第3<sup>rd</sup>周期，使得氢更容易离开

>>> ○ **原因 2：** Al 的电负性较低，导致 H 的 δ<sup>-</sup> 较大

>>> ○ LiAlH<sub>4</sub> 与 H<sub>2</sub>O 反应剧烈，因此应避免同时添加 H<sub>2</sub>O：LAH 主要与 THF 溶剂一起使用

> ⑤ Na<sub>2</sub>SO<sub>3</sub>

## **2\.有机金属试剂**

⑴ 概述

> ① 有机金属试剂进行碳-碳偶联反应

> ② 化合物中的金属离子带δ<sup>+</sup>电荷，烷基带δ<sup>-</sup>电荷

⑵ 格氏试剂：RMgX

> ①制备：RX+Mg（催化剂：无水乙醚）→RMgX

> ② **反应**：在 RMgX 中，Mg 变成 δ<sup>+</sup>，导致 R 上带有强负电荷，与亲电碳连接

> ③ 格氏试剂与低空间位阻的碳发生反应

> ④ 有机卤化物的限制

>> ○ 有机溴化物和有机碘化物：易形成格氏试剂

>> ○ 有机氟化物：几乎不形成格氏试剂

> ④ 酒精[保护反应](https://jb243.github.io/pages/1372) (_例如_, 甲苯磺酰化, 甲磺酰化)

>> ○ 如果 RX 中存在酸性氢，Mg 将充当路易斯碱并与氢发生反应。例如，酒精就会产生这种行为。

>> ○ 因此，通过醇保护反应可以保护醇免受镁的影响

⑶ 烷基锂试剂：RLi

> ① 准备工作

> ② **反应**：在RLi中，Li变成δ<sup>+</sup>，导致R上带有强负电荷，与亲电碳连接

>> ○ 烷基锂试剂与格氏试剂和吉尔曼试剂不同，可以与羧酸反应

⑷ Gilman 试剂：R<sub>2</sub>CuLi

> ①制备：2RLi + CuI（催化剂：无水乙醚）→ R<sub>2</sub>CuLi

> ② **反应**：在R<sub>2</sub>CuLi中，CuLi变成δ<sup>+</sup>，导致R上带有强负电荷。将单个R连接到亲电碳。

>> ○ **1-1.** 伯烷基卤化物与S<sub>N</sub>2机理的反应

>> ○ **1-2.** 烯烃的卤素取代反应：保留立体构型Z、E

>> ○ Gilman 试剂不与羰基 (C=O) 发生反应

>> ○ **例外 1.** 由酰氯形成酮：酰氯具有反应性，R 基团连接到酮碳上，而 -Cl 基团释放

>> ○ **例外2.** 烯酮和1,4-加成反应（迈克尔加成）：对于β-卤代羰基E1cB，在远离酮的烯烃位置发生碳-碳偶联反应
