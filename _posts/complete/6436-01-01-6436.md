## **1\.病毒**

 ● 概述

> ○ 病毒不能分为革兰氏阳性或革兰氏阴性。

> ○ 革兰氏染色的概念仅适用于具有肽聚糖的细菌。

> ○ 病毒类型

<br>

**图 1.** 病毒类型

<br>

● 腺病毒（AdV）

> ○ 36 kb，双链DNA病毒，裸露病毒

> ○ 辅助依赖性细小病毒

> ○ 小（18-25 nm）二十面体衣壳

> ○ DNA结构：ITR-REP-CAP-ITR

>> ○ ITR（反向末端重复序列）：REP·CAP所需的独特顺式元件

>> ○ REP：负责复制

>> ○ CAP：负责包装

> ○ 腺病毒通过网格蛋白介导的内吞作用进入细胞。

● 噬菌体 phi-X174

> ○ 第一个公布基因组序列的生物体

<br>

> ○ 基因组：5,386 bp，ssDNA

> ○ 含有11种蛋白质，总共1,986个氨基酸

> ○ 第一个已知的重叠基因示例 (∵ 3 × 1,986 > 5,386)

 ● 冠状病毒（CoV）：巢病毒目成员

> ○ 特征：有包膜病毒

> ○ 遗传物质：正义单链RNA

> ○ 螺旋对称核衣壳

> ○ 机理：感染人体呼吸系统

> ○ 症状：通常无症状，但有时会引起严重症状

> ○ 分类：Alpha（Ⅰ组）、Beta（Ⅱ组）、Gamma（Ⅲ组）、Delta（Ⅳ组）

> ○ **示例 1.** **SARS**（严重急性呼吸系统综合症）

>> ○ 引起 SARS 的病毒称为 SARS-CoV。

>> ○ 发生于2002年冬季，中国。

> ○ **示例 2.** **MERS**（中东呼吸综合症）

>> ○ 新型冠状病毒（nCoV）引起的疾病

>> ○ 仅感染非睫状上皮细胞

>> ○ 2012年4月首次出现在沙特阿拉伯，据信通过骆驼或蝙蝠传播

>> ○ 2015年5月韩国首例

>> ○ 2018年9月17日世界卫生组织宣布MERS结束

> ○ **示例 3.** **新型冠状病毒 (COVID-19)**：请参阅下文

 ● 其他信息：**新型冠状病毒（2019-nCoV、COVID-19）**

> ○ 概述

>> ○ 由 SARS-CoV-2（严重急性呼吸系统综合症冠状病毒 2）引起

>> ○ 2019年12月首次亮相中国武汉

>> ○ 截至2020年7月9日，死亡人数已超过54万人

> ○ 特点

>> ○ 有包膜病毒

>> ○ 约80-120 nm

>> ○ 约 40,000 kDa

> ○ 零部件

>> ○ **成分 1：** 刺突 (S) 蛋白：约 100 个拷贝

>>> ○ 长度约20 nm。细分为S1亚基和S2亚基

>>> ○ 冠状病毒表面有 75 个尖峰

>> ○ **成分 2：** 包膜 (E) 蛋白：约 20 个拷贝

>> ○ **成分 3：** 膜 (M) 蛋白：约 2000 个拷贝

>> ○ **成分 4：** 核衣壳 (N) 蛋白：约 1000 个拷贝。具有围绕基因组的管状蛋白质壳

>>> ○ 螺旋对称核衣壳

>> ○ **1-1.** S1 亚基：三聚体

>>> ○ 进行宿主细胞特异性结合：与ACE2结合的Kd值约为15 nM（相当大）

>>> ○ S1亚基中的S1-NTD和S1-CTD是受体结合域>>> ○ S1-CTD：与 [ACE2](https://jb243.github.io/pages/87)、APN（氨肽酶 N）、DPP4（二肽基肽酶 4）结合

>> ○ **1-2.** S2 亚基

>>> ○ 连接S1单元和病毒细胞膜

>> ○ **1-3.** 冠状病毒的S1和S2亚基被宿主细胞蛋白酶分离

>>> ○ 组织蛋白酶家族和 TMPRSS2（跨膜蛋白酶丝氨酸 2）的参与

> ○ 生命周期（Hartenian 等人，2020）

>> ○ 第一阶段：通过膜融合或内吞作用结合和病毒进入

>> ○ 第2阶段：多肽翻译

>> ○ 第 3 阶段：多肽的自蛋白水解和共翻译切割，生成 nsps（非结构蛋白）

>> ○ 第 4 阶段：（正义）亚基因组转录和 RNA 复制

>> ○ 第五阶段：（+sense）亚基因组转录和RNA复制

>> ○ 第 6 阶段：亚基因组 mRNA 翻译为结构蛋白和辅助蛋白

>> ○ 第 7 阶段：核衣壳芽进入 ERGIC（内质网-高尔基体中间室），其中布满了 S、E 和 M 蛋白

>> ○ 第 8 阶段：病毒粒子的形成

>> ○ 第 9 阶段：胞吐作用

> ○ 遗传物质：正义单链RNA

>> ○ 基因组大小：30 kb

>> ○ 包含 5' 帽和 3' 聚 A 尾

> ○ 机理：感染人体呼吸系统

>> ○ 进入牢房：约10分钟

>> ○ 日食期间：10小时左右

>> ○ 爆发规模：约 1000 个病毒粒子

> ○ 诊断

>> ○ 类型 1：RT-qPCR

>>> ○ 准确度高、灵敏度高，常用作金标准

>>> ○ 专利已过期，价格实惠

>> ○ 类型 2：微滴式数字 PCR

>> ○ 类型 3：基于 LAMP 的测试

>>> ○ 在单链病毒RNA模板末端形成环后进行PCR。

>>> ○ 无需变性步骤，可在室温下进行

>> ○ 类型 4：基于 NEAR 的测试

>>> ○ 使用类似于限制性内切酶的酶，称为切口核酸内切酶

>>> ○ 最快的诊断方法：ID NOW 产品只需不到 15 分钟

>> ○ 5 型：COVID-19 免疫测定：使用抗原抗体反应。

>>> ○ **类型 5-1:** 抗原测试：附着抗体

>>> ○ **类型5-2：** 抗体检测：像抗原测试一样附加抗原 

>>> ○ **类型 5-3:** 中和

>>> ○ 缺点：无法区分 SARS-CoV-2 和 SARS-CoV

>> ○ 类型6：重组酶聚合酶扩增

>> ○ 类型 7：基于 CRISPR 的检测

>> ○ CO-RAD（COVID-19 报告和数据系统）

>>> ○ 类别 0：由于技术缺陷而无法解释

>>> ○ 第 1 类：非常低。正常与否感染

>>> ○ 第 2 类：低。典型感染，但不是 COVID-19

>>> ○ 第 3 类：模棱两可/不确定。可能是 COVID-19，也可能不是

>>> ○ 第 4 类：高。疑似 COVID-19

>>> ○ 第 5 类：非常高。典型的 COVID-19 症状

>>> ○ 第 6 类：通过 RT-PCR 证实的 COVID-19

> ○ **治疗：** 唯一成功的治疗方法是瑞德西韦，但它并没有显着降低死亡率

>> ○ 阿比朵尔：抑制刺突蛋白和 ACE2 相互作用以及膜融合

>> ○ Bamlanivimab + Etesevimab：与 S 蛋白的 RBD 结合

>> ○ 甲磺酸卡莫司他：抑制 TMPRSS2

>> ○ 氯喹：抑制膜融合

>> ○ EK1C4/HR2P：抑制膜融合

>> ○ 地塞米松：抗炎反应

>> ○ 法匹拉韦：抑制RNA依赖性RNA聚合酶

>> ○ 羟氯喹：抑制膜融合

>> ○ 洛匹那韦：抑制复制酶-转录酶复合物 3CLpro

>> ○ 重组 ACE2：结合 COVID-19 而不是 ACE2

>> ○ **瑞德西韦：** ATP 类似物。抑制 RNA 依赖性 RNA 聚合酶

>> ○ 利巴韦林：抑制RNA依赖性RNA聚合酶

>> ○ 利托那韦：抑制复制酶-转录酶复合物 3CLpro

>> ○ Sarilumab：特异性结合 IL-6，抑制 IL-6 信号传导

>> ○ 托珠单抗：特异性结合 IL-6，抑制 IL-6 信号传导> ○ **附加信息：** 为什么艾滋病治疗对冠状病毒有效

>> ○ RNA病毒的性质

>>> ○ 人类从 DNA 中产生 RNA，从 RNA 中产生蛋白质。蛋白质执行重要的生物学功能。

>>> ○ 病毒复制涉及将带有病毒遗传信息的DNA插入人类DNA中。

>>> ○ 因此，RNA病毒总是需要逆转录酶来从RNA产生DNA。

>> ○ 艾滋病病毒和冠状病毒

>>> ○ 艾滋病病毒和冠状病毒都是ssRNA病毒。

>>> ○ 不难推断，通过艾滋病疗法可以治疗冠状病毒。

>>> ○ 2002年，韩国用艾滋病疗法治疗SARS病毒（一种冠状病毒）。

> ○ **补充：** 冠状病毒和群体免疫

>> ○ 群体免疫的公式：群体免疫所需的感染个体比例取决于*n*，即一台感染宿主可以将感染传播到的宿主数量。但是，这可能会根据所使用的统计分布而有所不同。
