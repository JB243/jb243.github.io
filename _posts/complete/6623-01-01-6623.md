## **1.集合论** 

⑴ 案例（结果）**：**可以任意定义。标记为 ω<sub>i</sub>

⑵样本空间**：**设置满足以下条件的Ω

> ① Ω = { ω<sub>1</sub>, ω<sub>2</sub>, ···, ω<sub>n</sub> }

> ② ω<sub>i</sub> 和 ω<sub>j</sub> 不相交

> ③ Ω 包括所有可能的结果

> ④ ((1, 2, 3), (3, 4, 5, 6)) (×)

> ⑤ ((1, 2), (3, 4, 5, 6)) (○)

⑶ 域**:** Ω的所有子集的集合F

> ①假设**:** Ω = {1, 2, 3} 

> ② F = {∅, {1}, {2}, {3}, {1, 2}, {1, 3}, {2, 3}, {1, 2, 3}}

⑷ 事件：F的每个元素，_即_样本空间的**子集** 

> ①总事件：Ω 

> ② 空事件：∅

> ③ 互补事件：A<sup>c</sup> = Ω - A 

> ④ 工会事件**：** A ∪ B

> ⑤ 相交事件：A ∩ B

> ⑥ 互斥事件：满足 A ∩ B = ∅ 的事件 A 和 B 之间的关系

> ⑦ 当事例数为n时，事件数为2<sup>n</sup> 

> ⑧ 德摩根定律：(E ∪ F)<sup>c</sup> = E<sup>c</sup> ∩ F<sup>c</sup>, (E ∩ F)<sup>c</sup> = E<sup>c</sup> ∪ F<sup>c</sup> 

⑸ 概率

> ①直观的定义**：**特定事件的概率=特定事件的数量÷所有事例的数量

> ②公理化方法**：**由统计学之父柯尔莫哥洛夫提出

>> ○ 概率是一种函数： F → ℝ

>> ○ **公理 1**。 P(E)≥0

>> ○ **公理 2**。 P(Ω) = 1

>> ○ **公理 3.** 概率加法规则**：** 关于不相交的 E<sub>i</sub> 和 E<sub>j</sub>

>> ○ **定理1.** P(∅) = 0 

>>> ○ P(U) = P(U) + P(∅) ⇔ P(∅) = 0

>>> ○ 请注意，即使 E ≠ ∅，P(E) 也可以为 0

>> ○ **定理 2.** P(E<sup>c</sup>) = 1 - P(E) 

>>> ○ 1 = P(E ∪ E<sup>c</sup>) = P(E) + P(E<sup>c</sup>)

>> ○ **定理3.** E ⊂ F → P(E) ≤ P(F) 

>>> ○ F = E ∪ (E<sup>c</sup> ∩ F) → P(F) = P(E) + P(E<sup>c</sup> ∩ F) ≥ P(E)

>> ○ **定理 4.** P(E ∪ F) = P(E) + P(F) - P(E ∩ F)

>>> ○ E ∪ F = (E<sup>c</sup> ∩ F) ∪ (E ∩ F) ∪ (E ∩ F<sup>c</sup>)

>>> ○ P(E ∪ F) = (P(F) - P(E ∩ F)) + P(E ∩ F) + (P(E) - P(E ∩ F)) 

>> ○ **定理 5.** 德摩根定律

>>> ○ (A ∪ B)<sup>c</sup> = ((A<sup>c</sup> ∩ B) ∪ (A ∩ B<sup>c</sup>) ∪ (A ∩ B))<sup>c</sup> = (U - A<sup>c</sup> ∩ B<sup>c</sup>)<sup>c</sup> = A<sup>c</sup> ∩ B<sup>c</sup> 

>>> ○ 设 C = A<sup>c</sup>，D = B<sup>c</sup>，则 C ∩ D = (C<sup>c</sup> ∪ D<sup>c</sup>)<sup>c</sup> ⇔ (C ∩ D)<sup>c</sup> = C<sup>c</sup> ∪ D<sup>c</sup>

⑻ **示例** **1.** 诊断率

> ①癌症患者比例

>> P(A) = 1%

> ②诊断时的阳性概率（命中率）

>> P(乙)

> ③诊断癌症患者时的阳性概率

>> P(B | A) = 95%

> ④ 诊断非癌症患者时出现假阳性的概率

>> P(B | A<sup>c</sup>) = 5%

> ⑤ **问题：** 阳性者在诊断过程中真正患癌症的概率

>> P(A | B) 

> ⑥诊断阳性比例**：**采用全概率定律

>> P(B) = P(B | A) + P(B | A<sup>c</sup>) = 0.95 × 0.01 + 0.05 × 0.99 

> ⑦ 任何人是癌症患者且诊断呈阳性的概率

>> P(A ∩ B) = P(B | A) × P(A) = 0.95 × 0.01 

> ⑧ 回答

>> P(A | B) = P(A ∩ B) ÷ P(B) = **0.16**

> ⑨ **解读：** 由于非癌症患者较多，假阳性率相当高

⑼ **示例 2.** 蒙蒂霍尔问题

> ①情况

>> ○ 超级跑车站在三扇门之一后面

>> ○ 表演者选择三扇门中的一扇 

>> ○ 节目主持人打开了节目参与者未打开的任何门

>> ○ 展会参与者可以坚持或更改其现有选择

>> ○问题**：**哪种选择合理？

> ②前提

>> ○ 显示参与者首先选择门1

>> ○ 节目主持人开门3

> ③定义

>> ○ P(i): 超级跑车在门1后面的概率

>> ○ P(ii): 超级跑车在门 2 后面的概率

>> ○ P(iii): 超级跑车在门3后面的概率

>> ○ P(Ⅰ)：主持人选择1号门的概率

>> ○ P(Ⅱ)：主持人选择2号门的概率

>> ○ P(Ⅲ)**:** 主持人选择3号门的概率

>> ○ P(i)、P(ii) 和 P(iii) 是先验（原因）

>> ○ P(Ⅰ)、P(Ⅱ)、P(Ⅲ) 是后验（结果）

> ④条件概率

>> ○ <span>P(ⅰ | Ⅲ)</span>：当主持人选择3号门时，1号门后面有一辆超级跑车的概率

>> ○ <span>P(Ⅲ | ⅰ)</span>：如果1号门后面有一辆超级跑车，主持人选择3号门的概率
