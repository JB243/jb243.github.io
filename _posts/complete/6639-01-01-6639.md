**1.资本主义**

⑴ 定义

> ① 每个人都追求私人利益的经济制度  
> ② 市场机制运行的经济体系

⑵ 资本主义的阶段

> ① **第 1 阶段.** 自发订单  
> ② **第二阶段** 基于公共权力的国家  
> ③ **阶段3.** 市场一体化和全球化

⑶ 批判观点**：** 由反全球化运动主导

> ① 即使不能（或不应该）成为商品的东西也能变成商品的经济体系**：**环境、人权等。  
> ② 资本家剥削工人的经济制度  
> ③ 通过垄断市场威胁自给经济的经济体系

---

**2.第一阶段：自发秩序**

⑴ 概述

> ①自然形成的经济体系  
> ② 推动中世纪经济史

⑵ 长途贸易的兴起

> ① 概述  
>> ○ 长途贸易的需要 **：** 商品多样化  
>> ○ **缺点1.** 欺诈、违约行为频繁  
>> ○ **缺点2.** 难以惩罚合同违规行为  
>> ○ **缺点3.** 贸易习俗的差异

> ② 第一<sup>第一</sup>。当地贸易**：**自然发展  
> ③ 第二<sup>第二</sup>。随着当地贸易的发展，当地商业社区出现  
> ③ 第三<sup>第</sup>。当地商业社区缓冲了长途贸易的负面影响，从而促进了长途贸易的发展

⑶ 长途贸易→货币激活

> ①（参考）罗马时代的货币  
>> ○ 金币居中；银币和零钱是补充  
>> ○ 金币**：** 在朝贡（实物税）制度下使用  
>> ○ 罗马灭亡后，金银铜三金属体系消失

> ②（参考）银币的出现  
>> ○ 到中世纪后期，银币出现  
>> ○ 银币：季节性用于在收获季节销售农产品和支付租金/会费

> ③金银铜三金属系的出现  
>> ○ **出现1.** 随着长途贸易的扩大金币的流通  
>>> ○ 示例：金币“弗罗林”在佛罗伦萨推出（1252 年）  
>> ○ **出现2.** 小币的流通随着城市中下层经济的增长  
>> ○ **结果1.** 将多维理性转化为一维理性  
>>> ○ 鼓励消费者做出经济判断  
>>> ○ 结果 1-1。解决了各种纠纷  
>>> ○ 结果 1-2。加深对金钱运作方式的理解  
>> ○ **结果2.** 货币体系用于商品经济  
>>> ○ 罗马 **:** 货币体系被用来运作朝贡经济  
>> ○ **结果3.** 货币大量流通造成混乱**：**信用交易的发展

⑷ 货币激活→商业城市诞生

> ① 决定因素  
>> ○ 有利于商业发展的地区 **：** 有利于保护产权和商业权利的地区  
>> ○ 长途贸易利润最大化的领域

> ② 变化因素  
>> ○政权更迭**：**政权越专制，商业城市越衰落  
>> ○ 分销路线的变化 **：** 因战争、运输技术、区域专业化结构等而改变。

**3.第二阶段：以公共权力为基础的国家**

⑴ 概述

> ①国家控制市场  
> ② 推动近代早期的经济史⑵ 规模经济→国家的出现

> ①随着集团规模增大，享受规模经济，但总效益增速逐渐下降  
> ② **规模不经济 1.** 成员之间对于生产公共产品的分歧——自身利益、异质性等——增加了达成共识的成本  
> ③ **规模不经济 2.** 公共产品会产生额外的公共管理成本  
> ④ 边界形成适当的尺寸

⑶ 国家的出现

> ① **原因1.** 与教会的斗争  
>> ○ 驱动力 **：** 文艺复兴（强调人类理性）、宗教改革  
>> ○ 例子**：**在14世纪的法国，国王超越了教皇  
>> ○ 示例**：** 国王也能够对教会财产征税

> ② **事业2.** 对抗神圣罗马帝国  
>> ○ 神圣罗马帝国是教皇和皇帝的制度  
>> ○ 教皇权威衰落，神圣罗马帝国衰弱  
>> ○ 神圣罗马帝国影响下的国家，后来出现了中央集权国家  
>> ○ 例子 **：** 金牛（1356）削弱了皇帝的权威

> ③ **原因3.** 与贵族的斗争

> ④ 国家的出现是什么样子的  
>> ○ 因地区而异  
>> ○ 联合边界的形成、民族感情的形成、国王官僚机构的强化  
>> ○ 伴随着无数战争

⑷ 国家的出现→法律控制

> ①国家第三方执行体系  
>> ○ 优势**：**降低交易成本  
>> ○ 缺点**：**减缓市场经济的发展  
>> ○ 国家的性质是由小统治阶级的性质决定的

> ② 英国案例  
>> ○ 第一<sup>第一</sup>。大宪章  
>>> ○ 不仅适用于自由农民，也适用于国王  
>>> ○ 人权和财产权的发展  
>>> ○ 普通法：建立基于判例法的命令  
>>>> ○ 立法者制定法律时，往往会排除对自己不利的规则  
>>>> ○ 立法过程中的“黄金法则”

>> ○ 第二<sup>第二</sup>。光荣革命 (1688)  
>>> ○ 大宪章颁布后，英国王室通过出售各种特权来增加收入  
>>> ○ 由于这种出售特权的行为变得过度，议会在 1688 年削弱了王权

>> ○ 第三<sup>第</sup>。政府机构的组建  
>>> ○ 组织了议会，以便各阶层的代表能够在议会内达成协议  
>>> ○ 光荣革命后，议会中的商人代表夺取了国家最高权力

>> ○ 第 4<sup></sup>。发展寻租特权的公开竞争  
>>> ○ 通过公开竞争获得特权，实现了特权的合理分配  
>>> ○ 在某些情况下，由于利益集团之间的凝聚力，特权分配仍然失败

> ③ 法国案例  
>> ○ 政府机构的组建  
>> ○ 开发基于任人唯亲的特权分配系统  
>>> ○ 权限值不固定  
>>> ○ 即使是特权持有者也面临着特权被剥夺的风险 **:** 频繁隐瞒技术  
>>> ○ 行政官僚对工业技术和商业了解不够

> ④ 专利法的起源  
>> ○ 专利法的悖论  
>>> ○ 如果专利法薄弱 **:** 发明人的激励减弱  
>>> ○ 如果专利法很强大 **：** 会阻碍竞争，抑制商业  
>>> ○ 解决方案**：** 设置垄断权的有效期>> ○ 第一<sup>第一</sup>。威尼斯专利法 (1474)  
>> ○ 第二<sup>第二</sup>。英国伊普斯威奇纺织工人案  
>>> ○ 法院支持新发明项目垄断权  
>>> ○ 专利保护要求（1624）：任何缺乏新颖性和有限有效期的东西都是非法的  
>> ○ 第三<sup>第</sup>。美国专利法 (1790)  
>>> ○ 仅收集各国专利申请制度的优点  
>>> ○ 专利申请成本变得非常低

⑸ 法律控制→经济政策控制

> ① 第一<sup>第一</sup>。纳入商法  
>> ○ 商业法庭的消失  
>>> ○ 12世纪左右，欧洲各国政府系统地编纂了商法：他们采用了习惯法的很大一部分  
>>> ○ 英国《主食法》（1353）：即使在皇家法庭也可以根据商法做出判决  
>>>> ○ 吸引案件向皇家法院提起的政策  
>>>> ○ 商业法庭仍然存在

>>> ○ 爱德华·科克的裁决 (1606)  
>>>> ○ **裁决 1.** 商业法庭所做的是“寻求妥协”  
>>>> ○ **裁决2.** 皇家法院所做的是“确认正义”  
>>>> ○ 最终裁决权被授予皇家法院  
>>>> ○ 导致商业法庭消失

>> ○ 普通法中商法的演变  
>>> ○ 大陆民法比普通法更好地接受新的商业惯例  
>>> ○ 英国法院在某种程度上尝试接受新的商业惯例  
>>> ○ 限制**：** 要被承认为先例，必须首先存在长期确立的惯例，因此采用是不够的

> ② 第二<sup>第二</sup>。民族商人的出现  
>> ○ 随着商法的发展，保护国内商人的制度也随之发展  
>> ○（参考）汉萨商人时代  
>>> ○ 汉萨商人从外国统治者那里获得商业权利，在许多地方同时发生  
>>> ○ 有竞争力的英国商人感受到反向歧视**：**关税、各种特权

>> ○ 民族商人的崛起  
>>> ○ 从被动贸易国向主动贸易国转变  
>>>> ○ 从羊毛生产国向毛纺织品生产国转变  
>>>> ○ 冒险家商人发达，对外贸易扩大

>>> ○ 互惠要求**：** 颁布英国商人保护条例（1377）  
>>>> ○ 英国商人对汉萨商人在英国享有的权利不满  
>>>> ○ 要求互惠，以便英国商人在汉萨同盟城市享有同等权利  
>>>> ○ 随着专制国家的发展，汉萨单方面的特权贸易在 16 世纪末消失了

>>> ○ 旨在控制市场的市场规则  
>>>> ○ 随着民族商人的出现，汉萨商人感受到了威胁  
>>>> ○ 汉萨商人制定“外国人法”规范市场

> ③ 第三<sup>第</sup>。重商主义经济政策  
>> ○ 民族国家的经济政策  
>>> ○ 推动经济政策强化国家，提高经济实力  
>>> ○ 用国家力量推动经济增长

>> ○ 经济民族主义  
>>> ○ 国家之间争夺海外殖民地和贸易主导地位的竞争  
>>>> ○ 商家争取经营权的利益  
>>>> ○ 国家对从海外殖民地获取新收入来源的兴趣  
>>>> ○ 两种利益的结合

>>> ○ 分散在各地的经济控制权转移到中央政府  
>>>> ○ 关税> ④ 第 4<sup></sup>。引入国家货币管理体系  
>> ○ 硬币进出口管制  
>>> ○ 金银主义出现于 16 世纪  
>>> ○ 从殖民地获得大量金币的西班牙的垮台在很多方面都具有启发意义

>> ○ 纸币发行与国家干预 **：** 以英格兰银行为例  
>>> ○ 第一<sup>第一</sup>。试图建立一家与荷兰阿姆斯特丹银行竞争的银行  
>>> ○ 第二<sup>第二</sup>。光荣革命将国债制度化，与君主个人债务不同  
>>> ○ 第三<sup>第</sup>。需要一个机构来处理政府财政  
>>> ○ 第四<sup>th</sup>。由强大的伦敦商人和金融家创立（1694 年）  
>>> ○ 第 5<sup></sup>。纸币发行垄断→自由银行时代→纸币发行垄断  
>>> ○ 第六<sup>th</sup>。由于纸币仅在伦敦流通，因此不能用于小额交易

**4.第三阶段：市场一体化和全球化**

⑴ 概述

> ① 随着全球市场一体化，国家的重要性下降  
> ② 推动近代经济史

⑵ 工业革命的发生和扩散

> ① 英国工业革命  
>> ○ 长期且不可逆转的体制改革  
>> ○ 以伦敦为中心的全国市场一体化和社会分工取得进展  
>> ○ 农业和工业生产力的提高和资本积累的进步

> ②工业化在西欧国家的蔓延**：**19世纪中后期  
>> ○ 英国工业化紧张感蔓延  
>> ○ 传播支持工业化的意识形态  
>> ○ 工业化的两大支柱 **：** 国家的机构改革、银行和国家的资本配置  
>> ○ 重化工业成长为新的主导产业

> ③ 工业化在非欧洲国家的蔓延 **：** 20 世纪末  
>> ○ 新兴经济体 **：** 借助国际经济形势实现增长  
>> ○ 金砖国家**：**随着国际经济格局的变化而发展

⑶ 全球化的发展

> ① 交通通讯革命  
>> ○ 交通运输设备的发展  
>> ○ 通讯技术的发展

> ② 全球流动性增加**：** 19 世纪中叶  
>> ○ 货物流动**：**英国谷物法（1846年）、英法商业条约（1860年）的废除/修订  
>> ○ 货币流动 **：** 国际货币秩序的建立、19世纪金本位→20世纪末布雷顿森林体系  
>> ○ 人员流动 **：** 奴隶贸易、爱尔兰大饥荒、亚洲移民浪潮  
>> ○信息流动**：**商品交易所、国际金融市场、跨国公司的发展

⑷ 全球化的阶段

> ① 第一次全球化**：** 1870-1914  
>> ○ 产业间贸易发展  
>> ○ 运输/通讯成本下降和自由贸易很重要  
>> ○ 向新大陆大规模迁徙  
>> ○ 帝国区域集团导致全球化崩溃

> ② 第二次全球化**：**20世纪末  
>> ○产业内贸易的发展**：**实时通信网络的发展  
>> ○ 国际资本流动表现为外商直接投资  
>> ○ 区域主义促进区域交流进而促进自由贸易

⑸ 福利国家的兴衰> ① 第一<sup>第一</sup>。 19世纪末的长期萧条； 1930 年代的大萧条 **：** 国家自由放任观念的衰落  
> ② 第二<sup>第二</sup>。世界大战**：**国家主义的兴起；为建设福利国家的基础设施做出了贡献  
> ③ 第三<sup>第</sup>。世界大战导致全球化倒退  
> ④ 第 4<sup></sup>。福利国家简介  
> ⑤ 第 5<sup></sup>。福利国家的危机  
>> ○ 促进国际资本流出  
>> ○ 促进欠发达地区劳动力外流  
>> ○ 欧洲以外的经济发展**：** 日本、新兴经济体、东盟、金砖四国

⑹ 社会融合机制的解体

> ① 国家标准和法规**：** 贸易壁垒、社会融合机制  
> ② 1974年，非关税壁垒（各国的标准和规定）成为贸易协定的对象
