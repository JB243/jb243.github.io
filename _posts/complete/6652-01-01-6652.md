## **1.贸易法**

⑴ **贸易引力理论：** 两国之间的贸易量与其经济规模成正比，与两国之间的距离成反比。

> ① 它特别适合过去，当时陆路运输成本极高。
>
>> ○ 丝绸等奢侈品可以通过陆路运输，但谷物等大件物品则不能。
>
> ② 海运成本远低于陆运成本。
>
>> ○ 随着船舶运载能力不断增强，海上运输成本不断下降，海上贸易的可行距离大大扩展。
>>
>> ○ 这成为商业革命的基础。

⑵ **亨利·皮雷纳的边境效应：** 文化、宗教和法律差异对贸易的负面影响比距离更大。

> ① **示例：** 尽管朝鲜和韩国地理位置接近，但政治上截然不同，因此具有很强的边界效应。
>
> ② 与固定的地理距离不同，文化、宗教和法律差异是可变的。
>
> ③ **缓解 1.** 由于贸易促进共同语言和商法，边境影响往往会因贸易本身而减少。
>
> ④ **缓解措施2.** 在远距离交易中，买卖双方无法面对面见面或直接检验货物，因此出现了重量、体积、质量等共同标准的需求。
>
> ⑤ **缓解措施 3.** 乡镇提供法律服务，促进合同纠纷解决。
>
>> ○ 随着贸易的扩大，这些城镇被欧洲主要城市的永久贸易中心所取代。
>>
>> ○ 人口重新向这些城市集中。
>>
>> ○ 商船法院 → 被地方法院取代 **:** 海事法发展成为《海牙-维斯比规则》。
>
> ⑥ **缓解措施 4.** 语言标准化/统一
>
>> ○ 在日耳曼部落进入罗马世界期间，欧洲在语言上是异质的。
>>
>> ○ 贸易商和商人在每个地区都采用单一语言。

## **2.示例 1. 欧洲和贸易法**

⑴ **欧洲的地理和经济连续性：** 相似性、邻近性和没有边境效应刺激了贸易。

> ① **相似之处 1.** 罗马帝国总人口的 80% 居住在今天的欧盟境内。
>
> ② **相似性2.** 加洛林帝国的疆域与当今欧洲的边界颇为相似。
>
> ③ **相似点3.** 此后，直到19世纪帝国时代，自治城市一直存在，包括威尼斯等城邦。
>
> ④ 大多数欧盟国家的贸易发生在欧盟内部。
>
> ⑤ **邻近性：** 丹麦与较小的瑞典的贸易比与较大的英国或法国的贸易更多。
>
> ⑥ **没有边境效应：** 贸易密集的国家之间往往会进行更多的贸易。

⑵ **欧洲中心北移**> ① 8-9世纪阿拉伯人征服北非和伊比利亚半岛后，地中海贸易衰落。
>
>> ○ Henri Pirenne 将此解释为文化/宗教分裂造成了贸易的边境效应。
>>
>> ○ 最近的研究表明，下降幅度并没有那么严重，除了边境效应之外，收入减少也起到了一定作用。
>>
>> ○ 阿拉伯扩张后的几个世纪里，地中海世界的经济水平都没有恢复。
>
> ② 罗马晚期，优质陶器从法国南部的生产地运往整个帝国，但在德国莱茵河以北却找不到。
>
>> ○ **原因：** 贸易网络不发达，以河流为界的各个地区的品味和喜好不同。
>>
>> ○ 这些边界效应在日耳曼扩张和欧洲经济中世纪复兴期间得到了有效减少。
>>
>> ○ 由此，欧洲的中心向北转移。

## **3。示例 2. 俄罗斯的例外主义**

⑴ **俄罗斯的严重孤立**

> ○ 欧洲的文艺复兴对俄罗斯影响不大。
>
> ○ 与欧洲不同，俄罗斯的政治专制主义长期存在。
>
> ○ 欧洲其他地区在16世纪废除了农奴制，而俄罗斯却引入了农奴制。
>
> ○ 工业化、度量衡标准化、教育普及被推迟。
>
> ○ 日耳曼人迁入罗马世界后，当地字母被拉丁字母取代（拉丁化），但俄罗斯是个例外。
>
> ○ 俄罗斯没有参与 19 世纪末向自由贸易的转变。
>
> ○ 对19世纪西欧的科学觉醒没有贡献。
>
> ○ 反西方主义——反对商业化、个人主义和理性主义——作为 19 世纪的先锋意识形态出现。
>
> ○ 追赶和现代化始于19世纪末，但并没有朝着融入欧洲的方向进行。
>
> ○ 事实上，这种孤立主义依然存在**：** 2005年，俄罗斯在欧洲国家出口总额中所占份额仅为1-2%左右。

⑵ **俄罗斯为何未能并入欧洲**> ① **理由1.帝国扩张的限制**
>
>> ○ 帝国的形成往往会在语言、宗教和法律上产生同质性，但扩张是有限度的。
>>
>> ○ 例子包括维护边境地区安全的成本飙升以及低收入人口流入导致帝国收入减少。
>>
>> ○ 结果，罗马帝国和贸易的综合力量并没有到达俄罗斯的核心。
>>
>> ○ 由于俄罗斯没有经历过与欧洲一体化的历史，欧洲与俄罗斯之间的文化异质性增强（边界效应）。
>
> ② **理由2.贸易引力理论——俄罗斯的相对贫困**
>
>> ○ 俄罗斯幅员辽阔，人口密度低，难以产生足够规模的贸易。
>>
>> ○ 缺乏接触贸易带来的知识、文化、商品和制度 → 长期相对落后，孤立持续存在。
>
> ③ **原因3.皮雷纳边境效应的另一个方面——俄罗斯的严酷土地**
>
>> ○ 俄罗斯通过从外围开采资源并将其提供给沙皇来管理其帝国。
>>
>> ○ 这是由于俄罗斯独特的恶劣自然环境而自然形成的。
>>
>> ○ 其结果是，欧洲帝国以海外殖民开发和财富为基础，大力推进自由贸易，而独裁势力强大的俄罗斯，则实行不利于商人的政策，导致商业城市衰落。
>>
>> ○ 因此，未能发展自由贸易。
>
> ④ **理由4.反西方主义的历史背景**
>
>> ○ 俄罗斯曾一度并入蒙古帝国。
>>
>> ○ 看着蒙古帝国在波兰、立陶宛、匈牙利等强大的十字军势力下崩溃，俄罗斯对西方社会产生了恐惧。
>>
>> ○ 结果，反西方主义蔓延，压制个人主义和理性主义，封锁文艺复兴文化，减少与西方的自由贸易。
>>
>> ○ 事实上，社会主义意识形态的出现是对西方物质文明的抵制所推动的。

## **4。示例 3. 公制系统的传播历史**

⑴ **法国**

> ① 1789年，法国大革命瓦解了各种制度，并重新定义了几乎所有的度量衡体系。
>
> ② 1790年，法国采用公制并致力于在国际上标准化。

⑵ **英国及英联邦**

> ① **为什么英国较晚采用公制：** 因为它的竞争对手法国发明了它。
>
> ② 英国最终采用公制的原因
>
>> ○ 大多数与英国进行贸易的国家都采用公制。
>>
>> ○ 英国公众已经了解了公制。
>
> ③ 由于英国领导英联邦，其他英联邦国家也跟随英国过渡到公制。
>
>> ○ 印度 **：** 1954 年
>>
>> ○ 英国 **:** 1965
>>
>> ○ 爱尔兰 **:** 1967
>>
>> ○ 巴基斯坦 **:** 1967
>>
>> ○ 新西兰 **:** 1969
>>
>> ○ 澳大利亚 **:** 1970
>>
>> ○ 南非 **:** 1971
>>
>> ○ 马来西亚 **:** 1972
>>
>> ○ 加拿大 **:** 1973
>>
>> ○ 香港 : 1976
>>
>> ○ 牙买加 **:** 1998
>
>> ○ 迄今为止，利比里亚和缅甸是唯一使用码-磅系统的国家。

⑶ **俄罗斯**

> ① 由于俄罗斯的特殊性，直到1924年才采用公制。

⑷ **美国**> ① 依赖与世界各地科学家交流并严重依赖贸易的行业（例如制药业）采用公制。
>
> ② 主要面向国内市场生产的行业，如啤酒，仍采用惯例措施。
>
> ③ 可口可乐等国际品牌即使在美国也使用公制。
>
> ④ **原因1.** 3进制（三进制）系统已经建立了很长时间（历史的非理性）。
>
> ⑤ **原因2.** 由于美国幅员辽阔、实力雄厚，采用法国的爱国主义措施可能存在心理障碍。
>
> ⑥ **原因3.** 因为美国崇尚自由——独立战争、50个州等。
