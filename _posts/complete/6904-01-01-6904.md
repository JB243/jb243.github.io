## **1.垄断**（垄断）（例如市政供水、有线电视）

### ⑴ 概述

> ① **定义：** 市场上只有一家企业，该企业面临的（向下倾斜的）需求就是市场需求。

> ② **特点1.**单一供应商

> ③ **特点2.** 无相近替代品

> ④ **特点3.** 完备的准入门槛

### ⑵ 垄断的成因

> ① **规模经济**

>> ○ **自然垄断：** 规模经济产生的垄断

>> ○ **最小有效规模（MES）：**平均成本达到最低时的产出水平

>> ○ **当平均成本曲线持续下降时：** 如果 MES 仅在产量非常高的情况下发生，则往往会出现垄断/寡头垄断

>> ○ **原因：** 只有产量最大的企业才能以最低成本（即最高效）进行生产

> ② **政府政策**

>> ○ **示例1.** 专利

>> ○ **例2.** 独家销售权或国家垄断权：在韩国，例如烟草、人参等。

> ③ **竞争策略**

>> ○ **示例1.** 广告（广告）

>>> ○ **积极方面：** [1] 加剧竞争并降低市场价格。 [2] 提供信息

>>> ○ **负面方面：** [1] 虚假/夸大的广告误导消费者。 [2] 可以起到进入壁垒的作用

>> ○ **示例 2.** 产品差异化

>> ○ **示例 3.** 掠夺性定价：暂时降低价格以排除竞争对手

>> ○ **示例 4.** 搭售：将一种垄断产品与另一种产品捆绑在一起，以强化另一种产品的垄断力量

> ④ **缺乏信息**

### ⑷ 垄断市场中的均衡

> ① **市场需求曲线：** 因为只有一个供应商，所以企业面临的需求就是市场需求。这也是 AR（平均收入）曲线。

**数字。 1.** 完全竞争与垄断下的市场需求曲线

> ② **市场供给曲线**

>> ○ 在垄断市场中，供给曲线**未定义**

>> ○ **原因1.** 由于垄断者控制价格，因此价格与供应量之间的关系没有意义

>> ○ **原因 2.** 垄断者是价格制定者，并不接受给定的市场价格  
>> ○ 相反，价格和产量由 **MC = MR** 确定

> ③ **市场均衡：** 生产发生在利润最大化的产出水平*qₘ*。

> ④ 边际成本与边际收益的关系

> ⑤ **示例：** 假设垄断者的需求 **q(P) = 100 − P** （其中 0 ≤ P ≤ 100），成本 **C(q) = q²**。

### ⑸ 垄断市场中的福利

> ①（参考）**毛效益**

> ② **总剩余（净社会效益、总福利）：** 生产者和消费者的**总满意度**的衡量标准

> ③ **无谓损失（DWL）：**又称福利三角、超额负担、经济净损失等。

>> ○ **情况1.完全竞争：**总剩余最大化。从公司的角度来看，存在一些损失。

>>> ○ **P = MC**

>>> ○ 这种情况被称为**仁慈的社会规划者问题**。

>>> ○ 由此产生的市场结果是**社会最优结果**。

**数字。 2.** 完全竞争下的p-q曲线

>> ○ **情况 2. 垄断：** 垄断者的剩余增加，但总剩余减少了 DWL 的数量。>>> ○ **P > MC**

>>> ○ **为什么MR曲线位于需求曲线下方：**因为**价格效应**和**数量效应**

>>> ○ **数量效应：** 当多售出一件商品时，总收入将按该商品的售价增加

>>> ○ **价格效应：** 为了出售最后一个单位，垄断者必须降低所有已售单位的价格，从而减少总收入

>>> ○ 由于价格效应，MR 曲线位于需求曲线下方（因此低于 MC 满足需求的位置）

**图。 3.** 垄断下的 p-q 曲线

>> ○ **垄断的低效率**

>>> ○ 利润基本有保障，努力工作/创新的动力可能较弱

>>> ○ 可能会浪费开支来维持进入壁垒

>>> ○ 垄断会阻碍技术进步（尽管有些人认为它也可以促进创新）

### ⑹价格歧视（价格歧视）

> ① **定义：** 对同一产品向不同买家收取不同的价格

> ② **先决条件**

>> ○ 公司必须拥有**市场力量**（以便可以定价）

>> ○ **转售一定是不可能的**（如果转售是可能的，套利会使模型变得复杂）

>> ○ **不同买家/市场的需求价格弹性必须不同**

> ③ **价格歧视的方式**

>> ○ 高级购买限制

>> ○ 批量折扣

>> ○ 两部制关税

> ④ **情况1.纯垄断：**无价格歧视

> ⑤ **案例2.使用两种价格的价格歧视**

**图。 4.** 使用两种价格的价格歧视

**图。 5.** 歧视性垄断者的利润最大化

>> ○ **MRA = MRB：** 如果利润不同，公司为利润较高的一方生产更多产品

>> ○ **MRA = MRB = MC:** 供应商增加总产量直至总利润最大化

>> ○ 价格弹性较低的消费者对价格的敏感度较低，因此被收取的价格相对较高

>> ○ 与纯垄断相比，产出更高，福利损失更小

> ⑥ **案例3. 完全价格歧视（perfectpricediscrimination）：**也称为一级价格歧视

>> ○ **定义：** 垄断者知道每个消费者的最大支付意愿并准确收取该金额

>> ○ **特点1.** 无无谓损失：产量与完全竞争下相同

>> ○ **特征2.** 消费者剩余为零：所有剩余都被视为垄断者的利润

>> ○ 示例：跳蚤市场

>> ○ 实践中很难，因为公司不知道消费者的预订价格

### ⑺ 政府针对垄断的政策

> ① **国有化（国家所有）**

>> ○ 效率低下，因为没有动力最小化成本

> ② **价格监管**

>> ○ 对于自然垄断企业，平均成本向下倾斜； **当然**边际成本应低于平均成本

**图。 6.** 自然垄断企业的价格-需求曲线

>> ○ **边际成本定价：** 将价格设定为 **P = MC**

>>> ○ **优点：** 高效的资源配置

>>> ○ **缺点：** 出现赤字。对于具有规模经济的自然垄断企业来说，企业可能会遭受损失。

>> ○ **平均成本定价：** 将价格设定为 **P = AC**

>>> ○ **优点：** 无赤字（防止垄断者赔钱）

>>> ○ **缺点：** 资源分配效率低下

>> ○ **曼昆的观点：** 无论选择何种价格监管，都难以避免因监管捕获理论而造成的低效率

>>> ○ **捕捉监管理论：** 如果监管持续很长时间，监管者与被监管者之间就会形成密切的关系，监管者最终会按照被监管者所期望的方向制定政策

>> ○ **克鲁格曼的观点：** 只要价格上限不设太高就可以接受> ③ **鼓励竞争政策**

>> ○ 私有化鼓励竞争

>> ○ 美国电力基础设施产业被拆分、私有化后，停电、火灾频发

> ④ **反垄断法**

>> ○ **谢尔曼反垄断法 (1890)：** 将寡头垄断公司之间的协议定义为犯罪行为

>> ○ **《克莱顿法案》（1914 年）：** 进一步强化反垄断法，甚至可以针对串谋企图提起私人诉讼

## **2. （概念区分）垄断竞争市场**（垄断竞争市场）（例如小说、电影）

### ⑴ 特点

> ① **众多供应商：** 就卖家数量而言类似于完全竞争

> ② **产品差异化：** 个体企业不是价格接受者

>> ○ 他们面临着**向下倾斜**的需求曲线

>> ○ 每种产品都尝试以品牌名称实现差异化，获得一定程度的市场力量

>> ○ **非价格竞争：** 通过价格以外的方式进行竞争

> ③ **自由进入：** 企业进入和退出直至经济利润为零

### ⑵ 形式化

> ①利润最大化问题

> ② **需求弹性：** If |ε_{q,P}| > 1、需求是有弹性的

> ③ **勒纳指数：** 也称为百分比加成、市场势力

### ⑶ 短期市场均衡

**图。 7.** 垄断竞争中的短期均衡曲线

> ① 该图与垄断下的短期均衡曲线相同，但不同之处在于它代表的是单个企业的曲线

> ② 生产发生在 ε > 1 的情况下。获得正利润

### ⑷ 长期市场均衡

随着其他公司的进入，需求曲线继续从短期均衡向下移动。

**数字。 8.** 垄断竞争中的长期均衡曲线

> ① **特征1.** 产出低于长期平均成本曲线最低点：意味着产能过剩

> ② **特征2.** 为什么ATC曲线在Q处与需求曲线相切，其中MR = MC：因为利润在零时最大化
