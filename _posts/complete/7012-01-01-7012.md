### **1。经济学的基本特征**

⑴ 假设所有经济主体都是理性的（自利的）。

⑵ 经济学从根本上来说是一门“冷”学科**：**因为它研究的是自私的经济行为者（_homoeconomicus_）。

⑶ 经济学的血统（思想传统）

**图 1. 经济学的谱系**

> ① A. 史密斯 (1776) → D. 李嘉图 (1817) + T. 马尔萨斯 (1789)
>
> ② D. 李嘉图 (1817) → J. S. 密尔 (1847) + K. 马克思 (1867)
>
> ③ 看不见的手
>
>> ○ J. S. Mill (1847) → L. Walras, A. Marshall (1890)（新古典主义学派）
>>
>> ○ L. Walras、A. Marshall (1890)、T. Malthus (1789) → J. M. Keynes (1936)（凯恩斯主义）
>>
>> ○ J. M. 凯恩斯 (1936)、L. 瓦尔拉斯、A. 马歇尔 (1890) → 主流经济学、自由市场经济学
>
> ④劳动价值论
>
>> ○ K. 马克思 (1867) → V. 列宁 (1914) (社会主义) → 激进经济学

⑷经济学分类

> ①【完全竞争市场】(https://jb243.github.io/pages/1900)与不完全竞争市场：完全信息论（大规模）
>
> ②【契约论】(https://jb243.github.io/pages/1927)：同时不完全信息论（小规模）
>
> ③【Signaling】(https://jb243.github.io/pages/1935)：顺序不完全信息论（小规模）
>
> ④ [博弈论](https://jb243.github.io/pages/1914)提供了用于解释这些的数学原理。

### **2。经济学的主题**

⑴ 流量与存量

> ① 流量 **：** 必须在一段时间内衡量其数量的事物（例如需求量、收入、消费、投资等）
>
> ② 库存 **：** 必须在某个时间点衡量其数量的东西（例如，财富/资产、货币供应量）

⑵ 资源

> ①广义资源
>
> ②狭义资源**：**生产要素
>
>> ○ 劳工
>>
>> ○ 资本
>>
>> ○ 土地
>>
>> ○ 近年来，创业努力也越来越被视为生产要素。

⑶ 产品展示

> ① 定义**：**成为人们欲望的对象并在市场上买卖的物品
>
> ② 如果是可见的，则称为商品；如果它不可见，则称为服务。
>
> ③ 免费物品和经济物品
>
>> ○ 免费商品 **：** 无需刻意努力即可享受任意数量的商品
>>
>> ○ 经济商品 **：** 需要付出代价才能获得的商品
>
> ④ 消费品和资本品
>
>> ○ 消费品 **：** 日常生活中直接消费、满足人类需要的物品
>>
>> ○ 资本货物 **：** 用于生产消费品（机器、建筑物等）的货物

⑷ 成本> ① 会计成本**：**以货币形式支出的金额
>
> ② 机会成本 **：** 甚至包括那些具有成本特征的项目，尽管不是实际的货币支出
>
>> ○ 在做出选择而放弃的东西中，最有价值的东西
>>
>> ○ 示例**：** 如果选择上大学意味着放弃工作A（₩1,000,000/月）和工作B（₩2,000,000/月），机会成本为₩2,000,000/月。
>>
>> ○ 如果您开一家餐馆，您不仅要考虑会计利润，还要考虑您在其他工作中可以赚取的工资来评估损益。
>>
>> ○ 经济学之所以使用机会成本的概念，是因为“我应该为了这笔钱而做这件事吗？”的想法。可以限制某个行业的进入。
>
> ③沉没成本**：**一旦花费，无论之后如何选择都无法收回的成本
>
>> ○ 在现实中，人们常常因为关注沉没成本而做出非理性的选择。
>>
>> ○ 与沉没成本相关的机会成本为 0。
>>
>> ○ 示例 **：** 协和式飞机谬误

⑸ 价格

> ①价格的作用**：**传递信号、提供激励、分配收益
>
> ② 价格机制 **：** A. 史密斯认为，看不见的手决定价格。

### **3。国民经济基本结构**

⑴ 三大经济主体

> ① 家庭**：** 消费商品和供给生产要素
>
> ② 企业 **:** 供应商品并购买生产要素
>
> ③政府**：**征税并开展各种项目

⑵ 三大市场

> ① 产品市场
>
>> ○ 商品付款从家庭流向企业。
>>
>> ○ 商品和服务从企业流向家庭。
>
> ② 劳动力市场
>
>> ○ 劳动力从家庭流向企业。
>>
>> ○ 工资从企业流向家庭。
>
> ③资本市场
>
>> ○ 资本从家庭流向企业。
>>
>> ○ 租金、利息和利润从企业流向家庭。

⑶ 主要经济类型

> ①市场经济**：**以市场为一切经济活动中心的经济体
>
>> ○ 所有商品的价格均由市场决定。
>>
>> ○ 建立产权、允许自由竞争、保障自由贸易
>
> ...
>
> ②指令经济**：**主要由政府控制驱动的经济
>
> ③混合经济**：**政府干预补充市场不完善
>
>> ○市场不完善**：**市场经济未能有效配置资源
