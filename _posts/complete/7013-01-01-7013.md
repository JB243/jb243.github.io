## **3。国民收入统计**

⑴ 国内生产总值（GDP）

> ① **属地原则**

>> ○ 不包括海外国民的生产

>> ○ 包括外国人在国内生产的产品

> ② 生产

>> ○地下经济：民间借贷、炒房、走私、毒品、逃税。 **不计入GDP**

>> ○ 转让交易：继承、赠与。 **不计入GDP**

>> ○ 资本收益：股票价格波动、房地产价格波动。 **不计入GDP**

>> ○ 政府转移支付：失业救济金、灾害补偿金、补贴等。 **不计入GDP**

>> ○ 休闲、家务等 **不计入GDP**

>> ○ **例外**：估算租金、自消费农产品、国防和执法服务

>> ○ [补充说明](https://www.youtube.com/watch?v=vJ6CZO-rysw&list=PL11iCOlE6aYIL9z-DxtiG4oS7CZj1eva7&index=3)]

>>> ○ 新房购买价格计入GDP。二手房购买价格不计入GDP

>>> ○ 实物收入计入GDP。彩票奖金不计入GDP

>>> ○ 公司债券和银行利息计入GDP。政府债券利息不计入GDP

> ③ 最终成品

>> ○ 中间品不包括在内以避免重复计算

> ④转移支付虽不直接计入GDP，但通过消费间接计入GDP

>> ○ 转移支付不计入GDP的原因：不属于有市场价值的商品，不存在互惠义务

>> ○ 对 GDP 的贡献因消费者行为而异

> ⑤ 名义GDP和实际GDP

>> ○ 名义GDP：以货币形式表示的变量

>>> ○ 以比较年价格表示

>>> ○ 缺点：受通货膨胀影响

>> ○ 实际 GDP：以商品数量表示的变量

>>> ○ 以基准年价格表示

>>> ○ 在基准年，名义变量和实际变量始终相同

>>> ○ 优点：不受通货膨胀影响

>> ○ GDP平减指数

>>> ○ 名义GDP除以实际GDP所得值

>>> ○ 根据GDP统计追溯计算的最全面的物价指数（帕歇法）

> ⑥ 实际GDP和潜在GDP

>> ○ 实际GDP：一个经济体实际生产的所有最终产品的市场价值

>> ○ 潜在GDP：一国所有生产资源**正常**使用时可生产的最终产品的市场价值

>> ○ 潜在GDP = 充分就业产出 = 自然产出

>> ○（注）潜在GDP并非最高GDP

>> ○ GDP缺口=潜在GDP-实际GDP

>>> ○ GDP 差距 > 0：存在失业。需要增加总需求

>>> ○ GDP 差距 < 0：经济过热。需要抑制总需求

> ⑦ 各国GDP水平

>> ○ 韩国：每年2000万亿韩元

⑵ 分配GDP（GDI）：购买力的衡量标准

> ① GDI =（工资+利息+租金+利润）+间接税+固定资本消耗> ② 通过要素收入相加来估算国民收入的方法，要素收入是参与生产最终产品的生产要素的报酬

> ③ 名义GDP=名义GDI

> ④ 实际GDP < 实际GDI（贸易方面的改善）

> ⑤ 实际GDP > 实际GDI（贸易恶化）

> ⑥ 贸易条件似乎有与资本平衡类似的趋势

>> ○ 如果汇率上升，出口价格下降，导致贸易条件恶化

>> ○ 如果进口价格下降，贸易条件改善

> ⑦ 实际GDP不反映贸易条件变化：实际GDI = 实际GDP + 贸易条件变化导致的实际贸易损益

⑶ 支出国内生产总值（GDE）

> ① GDE = 私人消费支出(C) + 国内投资总额(I) + 政府支出(G) + 净出口(X-M)

⑷ 国民生产总值（GNP）

> ① 由于外来劳动力等因素，GNP 不如 GDP 受青睐

⑸ 国民总收入（GNI）

> ①国民在一定时期内通过提供国内外生产要素所获得的收入

> ② 用于衡量公民生活水平和福利水平的收入指标

> ③ 与GDI比较

>> ○ GDI遵循属地性原则，GNI遵循国籍性原则

>> ○ 要根据GDI计算GNI，必须加上来自国外的净要素收入

⑹（差异化概念）国民财富

> ① 一个国家拥有的总财富：一个经济体拥有的实物资产和国外净资产的总和

> ② 国内金融资产不包括在内，因为它们在国内相互抵消，导致净财富为零

> ③（参考）与国民收入的比较

>> ○ 国民财富是一个存量概念。国民收入是一个流量概念

>> ○ 国民财富是国民收入的源泉

>> ○ 国民收入是消费、储蓄和投资的源泉，是国民财富与劳动结合创造的

⑺ 其他经济指标

> ① 巨无霸指数

>> ○ 以麦当劳巨无霸价格为基础，比较约120个国家货币价值的关键指标

>> ○ 英国经济杂志《经济学人》季刊编辑出版

>> ○ 由于巨无霸的质量、尺寸和成分在全球范围内标准化，因此假设巨无霸的实际价格在任何地方都相同

> ② S&P 500 指数：跟踪美国证券交易所最大的 500 家上市公司业绩的股票市场指数。它被广泛认为是美国股市和整体经济的最佳代表之一。

> ③夏普比率：利用标准差评价基金业绩的指标

> ④【森索引】(https://jb243.github.io/pages/2044)

> ⑤【阿特金森指数】(https://jb243.github.io/pages/2044)：基于社会福利函数衡量收入不平等的方法

> ⑥ 恩格尔系数：食品支出占总消费支出的比重

> ⑦【基尼系数】(https://jb243.github.io/pages/2044)：收入不平等和贫富差距程度的数值表示，范围从0（完全平等）到1（完全不平等）

> ⑧ Nifty-50 指数：印度股票市场指数，代表在国家证券交易所上市的 50 家最大的印度公司的加权平均值

> ⑨ 痛苦指数：通过将年通货膨胀率与失业率相加计算得出的经济指标。它用于量化普通公民因生活成本上升和缺乏就业机会而经历的经济困境或“痛苦”程度。> ⑩ 恒生指数（HSI）：香港自由流通市值加权股票市场指数。它用于监测和记录香港股市最大公司的每日变化，并作为香港整体市场表现的主要指标。

> ⑪ Pizza Meter：Pizza Meter是一种非正式指标，来源于美国五角大楼附近的披萨店在工作量大的时候夜间销量激增的现象。通过衡量深夜销售额与平均水平相比的增幅，它被用作预测潜在国家危机或重大全球事件发生的间接指标。

>⑫巴菲特指标：=一国市值/国内生产总值（GDP）*100

## **6。宏观经济现象**

⑴ 帕累托法则（80-20法则）

> ① 20%的原因产生80%的结果的现象

> ② 意大利经济学家帕累托（V. Pareto）在研究收入与财富关系时发现的

> ③ 在任何国家，80%的财富掌握在20%的人口手中

> ④百货商店20%的顾客占总销售额的80%
