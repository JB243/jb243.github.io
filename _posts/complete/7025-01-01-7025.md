 ⑴凯恩斯的观点

> ①亚当·斯密的“看不见的手”**：**由于劳动力市场的灵活性，高失业率不可能持续下去。

> ② 1929年的大萧条导致失业率持续十多年，这一假设不成立。

...

> ③凯恩斯指出，总需求下降对高失业率很重要。

> ④ 特别是在消费（C）、投资（I）、政府支出（G）中，他强调了政府支出的作用。

> ⑤ 凯恩斯赞成大政府。

 ⑵ **假设1.** 价格变量的刚性

> ①凯恩斯基本上认为价格变量是刚性的。

> ② 特别是，他认为由于**雇佣合同期限**以及**工会**的影响等因素，工资具有**向下刚性**。

>> ○ 当劳动力需求过剩时，名义工资很容易上涨。

>> ○ 当劳动力供给过剩时，名义工资不会轻易下降。

 ⑶ **假设2.** 信息不完善

> ① 劳动力需求 **：** 实际工资的函数

> ② 劳动力供给 **：** 名义工资的函数

> ③（参考）也就是说，他认为工人主要以名义工资来思考，从而陷入金钱幻觉。

 ⑷ **假设3.** 存在闲置产能

> ①他认为**经济正处于衰退**，因此即使产出增加到充分就业水平，生产成本也不会上升。

> ② 即使总需求增加，总供给相应增加，**价格固定**的假设可能仍然有效。

> ③ 如果在极端情况下假设，则会得出**水平 AS 曲线**。

 ⑸ **假设4.** 有效需求原理

> ① 凯恩斯通论基本上假设经济衰退。

> ② 经济衰退时，由于总需求不足而无法实现充足的生产。

> ③ 他认为，如果总需求增加，闲置产能就会下降，国民收入（总产出）就会增加**而价格不会上涨**。

> ④（参考）繁荣时期，国民收入水平可以由总供给决定。

 ⑹ 凯恩斯适合衰退和短期，而古典理论适合繁荣和长期。

 ⑵ **步骤2.** 货币市场均衡

> ① 概述

>> ○（参考）IS-LM模型中，总需求曲线是通过分析商品市场和货币市场得出的。

>> ○ 在凯恩斯通论中，总需求曲线可以仅从商品市场推导出来。

> ② 利率的确定

>> ○（参考）古典学派的利率确定

>>> ○ **可贷资金理论** **：**利率由实体部门资金的供需决定（即储蓄和投资决定）。

>>> ○ IS曲线比较平坦； LM曲线比较陡。

>> ○ 凯恩斯学派的利率决定

>>> ○ [流动性偏好理论](https://jb243.github.io/pages/2026) **:** 凯恩斯认为利率是由货币的供求决定的。

>>> ○ 凯恩斯假定存在[流动性陷阱](https://jb243.github.io/pages/2026)。

>>> ○ IS曲线比较陡； **LM曲线相对平坦。**

 ⑶ **步骤3.**总供给曲线**：**满足**任一** **假设1**或**假设2**就足够了

> ① **假设1.** **货币幻觉** **：** 当预期价格（Pe）与实际价格（P）不同时，工人将名义工资W=P·w的变化误认为是实际工资的变化（即对价格水平的静态预期）。

>> ○ 劳动力供给 **：** 即使物价水平上涨，只要工人不修正预期，劳动力供给曲线就不会改变。

>> ○ 劳动力需求 **：** 企业作为劳动力需求者，面临的是实际工资，因此如果价格下跌，劳动力需求曲线左移。

>> ○ **结论1.** 劳动力供给是名义工资的函数；劳动力需求是实际工资的函数。

>> ○ **结论2.** 均衡就业完全由劳动力需求决定。

> ③ **假设2.** **名义工资的下行刚性**

**图。 1.** 名义工资的向下刚性

>> ○ **原因1.** 劳动合同不是连续签订的，而是有固定期限的。

>> ○ **原因2.** 在劳动力市场上，工人不接受名义工资的降低（劳动力市场的向下刚性）。

>> ○ **原因3.** 数量少时，产能闲置，所以即使数量增加，名义工资也不会增加（稍后修改）。

>> ○ 劳动力供给 **：** 由于名义工资的下行刚性，即使物价下跌，劳动力供给也不会发生变化。

>> ○ 劳动力需求 **：** 企业作为劳动力需求者，面临的是实际工资，因此如果价格下跌，劳动力需求曲线左移。

>> ○ **结论1.** 劳动力供给是名义工资的函数；劳动力需求是实际工资的函数。

>> ○ **结论2.** 均衡就业完全由劳动力需求决定。

> ④ **总供给曲线**（AS曲线）**：**为总需求水平E，

>> ○ 总供给曲线由 45° 线表示。

>>> ○ 原因**：** 如果有商品或服务的需求，供应就会自动得到满足。

>> ○ 一旦达到潜在国民收入水平（Y*），供给曲线就会变为垂直。

>>> ○ 潜在国民收入水平**：**充分就业时的国民收入水平。>>> ○ 原因**：**由于经济已经充分就业，即使总需求增加，产出也无法增加，因为**没有可用资源**。

 ⑸ 结论

> ① **解读1.** 繁荣与衰退

>> ○ 繁荣 **：** 价格上涨。过剩的有效需求称为通货膨胀缺口。

>> ○ 价格为何上涨 **：** 随着新增产能的扩大，单位供应成本增加。

>> ○ 经济衰退 **：** 有效需求的不足称为通货紧缩缺口。

>> ○ 凯恩斯基本上假设了经济衰退，并将原因诊断为需求不足。

>> ○ 如果政府支出增加，有效需求就会增加；因此，他认为通过通货紧缩缺口的大小来增加它将使经济恢复到正常状态。

>> ○（参考）凯恩斯认为政府支出是经济衰退的唯一解决方案。

> ② **解释2.** 乘数理论

>> ○ 乘数**：**（均衡国民收入增加）÷（自主支出增加）

>> ○ 乘数等于 Y 与 AD 曲线斜率的倒数。

>> ○ 代数解释 **:** G 增加 → Y 增加 → Yd 增加 → Y 增加 → Yd 增加 → ···

> ④ **解读4.** 节俭悖论

>> ○ 当经济不景气时，每个人都会增加储蓄 → 这会促进（加深）经济衰退。

>> ○ 个人理性与社会整体理性不一致的错误。

>> ○ 与乘数理论类似的逻辑，如果存在诱导投资，国民收入下降得更快。

>> ○ **案例1.**发展中国家**：**由于资金稀缺，储蓄充分转化为投资，所以储蓄在社会上是一种美德。

>> ○ **案例2.** 发达经济体/经济衰退期间**：** 由于缺乏投资机会，储蓄的增加不会导致投资的增加。节俭的悖论。
