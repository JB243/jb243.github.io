## **1.要素市场（一般）**

⑴ 背景

> ① 商品市场**：**家庭是需求方；企业是供给方  
> ②要素市场**：**企业是需求方；家庭是供给方  
> ③与新古典收入分配理论直接相关

⑵ 生产要素需求

> ① **解释1.** 具有衍生需求的特征  
>> ○ 含义**：**要素需求源自商品需求（MC）  
>> ○ 要素价格除以边际产量，即为边际成本（MC）

> ② **解释2.** 在新古典完全竞争市场中，价格代表边际产品**：**因为**边际收益产品为0**  
>> ○ 完全竞争企业的劳动力需求由名义工资等于劳动力边际产品价值决定  
>> ○ 边际收益产品（marginal Revenue Product；MRP）**：**多雇用一名工人所产生的额外收入  
>> ○ 边际产品价值 (value of marginproduct; VMP) **:** 要素的边际产品乘以产出的市场价格  
>> ○（注）VMP可以理解为完全竞争下的MRP  
>> ○ 劳动力价格=劳动力边际产量值（其中*w*表示名义工资）  
>> ○ 资本价格=资本边际产品价值（其中*v*表示名义租金）

> ③ 劳动力需求曲线  
>> ○ 假设 **：** 劳动力是唯一的可变投入  
>> ○ 向下倾斜 **:** 因为假设边际产量递减规律  
>> ○ 劳动力需求增加的因素  
>>> ○ 产品需求增加  
>>> ○ 生产要素生产率的提高  
>>> ○ 替代品的价格上涨  
>>> ○ 互补投入品的价格下降  
>>> ○ 生产者数量增加  
>>> ○（注）资本投入可以增加或减少劳动力需求

>> ○ 劳动力需求价格弹性**：** 也称为派生需求的希克斯-马歇尔定律  
>>> ○ 边际产量递减加快 ↑ → 劳动力需求曲线变得更陡 → **弹性 ↓**  
>>> ○ 与其他投入的替代性更大 ↑ → 就业变化更大 → 弹性 ↑  
>>> ○ 劳动力在总成本中的份额↑ → 就业变化更容易且变化幅度更大→ 弹性↑  
>>> ○ 产品需求的价格弹性↑ → 由于劳动力需求是衍生需求，弹性↑  
>>> ○ 资金供给弹性↑ → 弹性↑

⑶ 生产要素供给> ①劳动力供给**：**【消费者理论】(https://jb243.github.io/pages/1879)部分中消费者理论的新古典主义解释（劳动与休闲的选择）  
>> ○ 假设**：**工人可以自由选择工作时间  
>> ○ 目标函数 **：** 效用函数 *u*、休闲 *L* 和收入/预算 *M*  
>> ○ 约束 **：** 劳动 *H*、休闲 *L* 和收入/预算 *M*  
>> ○ **替代效应** **：**工资率↑→休闲机会成本↑→休闲消费↓→劳动力供给↑  
>> ○ **收入效应** **：**工资率↑→休闲消费↑（休闲是正常物品）→劳动力供给↓  
>> ○ 早期，替代效应占主导地位；随后，收入效应占主导地位，产生向后弯曲（backward-bending）的形状  
>> ○ 在后弯部分，劳动力供给的工资弹性为负  
>> ○（注）如果休闲是次品，则不会出现后弯形态  
>> ○（注）非劳动收入仅产生收入效应→劳动力供给减少

⑷ 要素市场均衡  

**图。 2.** 要素市场均衡

> ① 就业和工资的确定  
>> ○ 就业和工资由产品市场（需求方）和要素市场（供给方）的均衡决定  
>> ○ （注）如上图所示，有时会省略后弯部分  
>> ○ 由于均衡价格与消费者收入直接相关，因此也称为收入分配理论  
>> ○ 不同市场同一要素价格不同的原因

> ② 当不完全竞争市场变成完全竞争市场时  
>> ○ 均衡就业和均衡工资增长

> ③ 经济租金和转移收入

**图。 3.** 转移收入和租金的概念

>> ○ 转移收益**：**要素供应商的机会成本  
>>> ○ 供给曲线下方的面积  
>>> ○ (评论) 与“成本”类似

>> ○ **经济租金** **：**由于要素供给缺乏弹性而支付的额外费用  
>>> ○ 也称为非劳动收入 **：** 对应于商品市场中的 **生产者剩余**  
>>> ○ 换句话说，支付中超过转移收入的部分称为经济租金  
>>> ○ 劳动力供给越缺乏弹性，经济租金在总报酬中所占的比例就越大  
>>> ○ **示例1.** 著名艺人、运动员  
>>> ○ **示例 2.** 医疗或律师协会限制新许可证持有者数量的活动  
>>> ○ **示例 3.** 游说以获得政府许可/授权  
>>> ○ (评论) 类似于生产者剩余

>> ○ 准租  
>>> ○ 对短期供应固定的要素的补偿，例如资本设备  
>>> ○ 即短期具有租金性质，长期具有转移收益性质

⑸ 垄断与双边垄断

> ① 垄断  
>> ○ 定义**：** 一个有许多供应商但只有一个买家的市场  
>>> ○ 在商品市场上很难找到；主要见于要素市场  
>>> ○ 原因**：** 极其专业的因素、当地/区域特征  
>> ○ 边际要素成本曲线（边际要素成本曲线）**：**购买最后单位要素时的额外支出  
>> ○ 条件**：**边际要素成本=边际产品价值

>>> ○ 最终的结果取决于哪一方的议价能力更强

## **2.劳动力市场：**关注工资差异产生的原因

⑴ 当劳动力市场完全竞争时

> ① 工人之间的生产率差异  
> ② 工作便利或风险的差异 **：** 这里的工资差异称为补偿性工资差异

⑵ 当劳动力市场不完全竞争时

> ①市场暂时失衡  
> ② 按工龄计算工资  
> ③ 信息不完善  
> ④ 社会歧视

⑶ 熟练劳动力与非熟练劳动力

> ① 熟练劳动力的需求曲线位于非熟练劳动力的需求曲线之上（**∵**更高的边际生产率）  
> ② 熟练劳动力的供给曲线位于非熟练劳动力的供给曲线之上（**∵** 技能获取成本较高）  
> ③ 二元劳动力市场理论  
>> ○ 主要劳动力市场 **：** 管理岗位、专业岗位、技工岗位等。  
>> ○ 二级劳动力市场**：**简单工、家庭佣工等。  
>> ○（评论）将一级市场视为“第一线”/处于更有利地位的工人

⑷人力资本理论**：**受过教育的人比没有受过教育的人有更高的生产率，因此他们会获得更高的工资

⑸ 信号理论**：**教育只是一种区分能力强者和能力弱者的手段；提高教育水平与工资增长**无关**

⑹效率工资理论**：**支付高于行业工资以使工人发挥最大生产力的策略

⑺ 工会

> ①工会目标**：**增加工资收入，改善劳动条件，稳定就业  
> ② 结果  
>> ○ 不牺牲就业很难实现工资增长  
>> ○ 未成立工会的部门工资可能下降  
> ③限制工会权力的因素  
>> ○ 供给侧因素 **：** 对于非工会工人可以轻松提供劳动力的低工资劳动力，相关工会的权力减弱  
>> ○需求方因素**：**以机器替代劳动力、减产裁员、产品需求疲软

⑻ 劳动力市场歧视**：**种族歧视、性别歧视等。> ① 第一<sup>第一</sup>。歧视限制女性在高技能劳动力市场就业  
> ② 第二<sup>第二</sup>。随着高技能市场供应减少，女性工资上涨  
> ③ 第三<sup>第</sup>。随着普通技能市场供给的增加，女性工资下降  
> ④ 第 4<sup></sup>。如果这种歧视持续存在，就会被用作证明歧视合理性的“证据”，从而加剧歧视  
> ⑤ **政策1.** 同工同酬原则  
> ⑥ **政策2.** 可比价值原则  
> ⑦ **政策 3.** 自然选择 **：** 歧视性企业排除高生产力的人，因此随着时间的推移被淘汰  
> ⑧ 为什么歧视依然存在  
>> ○ 受歧视工人的生产力实际上较低  
>> ○ 根深蒂固的偏见导致公平就业政策难以落实

## **3。资本市场**

⑴ 资本概念

> ① 主要生产要素 **：** 劳动力和土地可以立即使用，无需额外的生产过程  
> ② 生产的生产要素 **：** 资本必须通过生产过程产生  
> ③ 资本货物 **：** 生产中使用的机器或设备  
>> ○ 折旧 **：** 随着资本货物的使用，其价值下降  
> ④ 资本服务**：**在一段时间内使用资本货物而获得的服务  
>> ○ 资本服务供给曲线向上倾斜

⑵ 投资决策过程

> ① 现值 **：** 以今天的条件评估的未来收入或成本的价值  
> ② 贴现率 **：** 将未来金额贴现为现值的比率  
> ③ 如果现值＞投资成本**：**投资  
> ④ 如果现值＜投资成本**：**不投资

⑶ 利率的确定

> ① 现实中为何存在多重利率  
>> ○ 风险  
>> ○ 借贷期限  
>> ○ 行政费用  
>> ○ 税务处理
