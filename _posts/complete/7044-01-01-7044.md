**1\.效率**（效率）

 ⑴ 概述

> ①定义**：**以最小的成本达到最大的效果

> ② 换句话说，一个经济体内存在的经济资源根本没有被浪费，而是得到了最好的利用

> ③效率的评价标准是帕累托效率

 ⑵ 可行分配（feasibleallocation）

> ① 定义**：** 不超过经济体资源禀赋的分配

> ②（商品 X 分配给 A）+（商品 X 分配给 B）≤（经济体中商品 X 的总量）

 ⑶ **帕累托效率** **:** 也称为帕累托最优

> ① 定义**：** 一种状态，在这种状态下，在不降低任何人的效用水平的情况下不可能转移到新的分配

>> ○ 即资源配置最有效的状态

>> ○ 原因**：** 因为所有的机会都在被利用

>> ○ 这里的“效用”是指序数效用 **:** 假设不同人的效用无法比较

> ② 帕累托改进**：** 一种至少使一个人受益而不伤害任何人的变化

> ③ **类型1.** 交换帕累托效率

>> ○ 定义**：** 在给定的货物数量内，合理分配，使两个人的效用最大化

>> ○ 埃奇沃斯盒（Edward 盒） **:** 也称为埃奇沃斯盒

**图。 1.**埃奇沃斯盒子]

>>> ○ Edgeworth 盒子内的每个点都使用（使用）所有给定的经济资源

**图。 2.**合约曲线]

>>> ○ Contract curve（契约曲线） **：** 连接帕累托有效分配点的曲线

>>> ○ 一般情况下，假设无差异曲线凸向原点

>>> ○ 合约曲线上的一点，例如 G 点，是 iA 和 iB 的公共切线的切点

> ④ **类型 2.** 生产中的帕累托效率

>> ○ 定义**：** 充分利用给定的生产要素，生产尽可能多的产出

>> ○ 正如交换中的帕累托效率一样，可以画出埃奇沃斯盒

>> ○ (参考) [生产可能性曲线](https://jb243.github.io/pages/1893) (生产可能性
曲线；竞价排名）

>>> ○ 通过在合同曲线上映射（移动）点，可以得到生产可能性曲线

>>> ○ 概述

>>>> ○ 简单定义 **：** 显示经济体中可最大限度生产的商品组合的曲线

>>>> ○ 精确定义**：** 显示社会充分就业下可以生产的两种商品的组合的图表

>>>> ○ 绘制在第一象限

>>> ○ 比 PPC 距原点更远的点 **:** 不可能

>>> ○ 比 PPC 更接近原点的点 **：** 效率低下

>>> ○ 机会成本递增法则（机会成本递增法则）

>>>> ○ **含义 1.** 实际上，PPC 相对于原点是凹的

>>>> ○ **含义 2.** 随着一种商品产量的增加，该商品的机会成本增加

>>>> ○ **含义3.** PPC的斜率增加（绝对值）

>>>> ○ **提示。** PPC 的斜率代表机会成本

>>>> ○（参考）如果机会成本增加→凹向原点；如果常数 → 直线；如果递减 → 凸向原点

>>>> ○ 根本原因**：** 随着产量的增加，效率较低的要素被带入生产，因此机会成本上升

> ⑤ **类型3.** 总体（综合）帕累托效率
([参考](https://www.youtube.com/watch?v=gC2vlpCwL9A&list=PL11iCOlE6aYJl63i5WZnWQmiLjSx7i5ka&index=90))]>> ○ 为了使整个经济体的资源配置更加有效，消费和生产都必须达到帕累托效率

>> ○ 第一<sup>第一</sup>。对于 PPC 上的每个点，都有一个对应的 Edgeworth 框

>> ○ 第二<sup>第二</sup>。对于每个埃奇沃斯盒子，可以绘制一条契约曲线

>> ○ 第三<sup>第</sup>。沿着每个埃奇沃斯盒子内的契约曲线，我们可以绘制一条效用可能性曲线

>> ○ 第 4<sup></sup>。如果所有效用可能性曲线都聚集并平滑连接（作为信封），则可以找到效用可能性边界

>> ○ 效用可能性边界（utility opportunity frontier；UPF）

>>> ○ 有效分配与低效分配的界限

>>> ○ 效用可能性曲线上的每个点都满足消费的帕累托效率

>>> ○ 由于合约曲线上只有一点满足 MRS = MRT，因此只有该点位于效用可能性边界上

> ⑥ 帕累托效率的局限性

>> ○ 使用帕累托效率会产生无限多个解决方案，因此需要另一个标准来选择“最佳”点

>> ○ 无法判断公平性

 ⑷福利经济学第一基本定理

> ① 假设

>> ○ 每个消费者的偏好系统都满足较强的单调性

>> ○ 经济不存在外部性

> ②结论**：**在竞争性（一般竞争）市场中，资源的帕累托有效配置是自动实现的

>> ○ 亚当·斯密“看不见的手”的现代诠释

> ③ 市场的局限性

>> ○ 市场不保证分配公平

>> ○ 有时甚至在资源配置效率方面也会出现问题 **：** 由于不完全竞争或外部性的存在

 ⑸ 福利经济学第二基本定理

> ① 假设

>> ○ 初始禀赋合理分配

>> ○ 每个人的偏好都是连续的、强单调的、凸的

> ② 结论 **:** 帕累托有效配置可以作为竞争性一般均衡来实现

>> ○ 这对应于第一定理的逆命题

**2\.股权**（股权）

 ⑴ 概述

> ① 经济资源如何在社会成员之间公平分配

> ② 无法找到客观的公平标准

 ⑵ 测量不平等

> ① Inequality Index（不平等指数）**：**基于平等意味着平等的前提

>> ○ Income decile（收入十分位数） **：** 由 10% 细分人群组成的收入群体

>>> ○ 平均家庭收入（平均家庭收入） **：** 所有家庭的平均收入

>>> ○ 家庭收入中位数（median family Income）**：**收入分布中中位数的家庭收入

>> ○ 十分位数分配比率（十分位数分配比率）

>>> ○ 定义**：**（底层40%的收入）除以（顶层20%的收入）

>>> ○ 范围从 0 到 2；值越小，分布越不均匀

>> ○ 五分位比

>>> ○ 定义**：**（收入最高的20%）除以（收入最低的20%）

>>> ○ 取 1 到 Infini 之间的值，如交换分子/分母的十分位数分配比

>> ○ 贫困率

>>> ○ 贫困线（贫困线）**：** 收入水平低于该水平的家庭被判定为绝对贫困

>>> ○ 贫困率（贫困率）**：**收入低于贫困线的家庭占总人口的比例

>>> ○ 贫困线也称为最低生活成本

>>> ○ 估算最低生活成本的方法 **：** Rowntree、Leyden 或平均或中位收入的固定比例

>>> ○ 贫困指数**：** 森指数等

 ⑶ 社会福利函数（social Welfare Function；SWF）**：**公平性的衡量标准

> ①定义**：**将社会成员的偏好聚合为社会偏好的函数

> ②功利主义价值观**：**社会福利函数定义为个体效用之和

> ③ 平等主义价值观 **：** 效用较高的人权重较低，效用较低的人权重较高

> ④罗尔斯（J. Rawls）**：**效用水平最低的成员的效用决定了社会的福利水平

> ⑤ 阿罗（K. Arrow）**：**通过不可能性定理证明不存在可以比较多种社会状态的理性标准

> ⑥ 次优理论**：**如果不是所有的效率条件都满足，满足更多的效率条件并不一定会增加社会福利

 ⑷ 收入再分配的政治哲学

> ① 平均主义（平均主义）

>> ○ 平等被解释为享有最低生活水平的平等权利

> ②功利主义（功利主义）

>> ○ 政府应实现社会全体成员总效用最大化的政治哲学

>> ○ 假设 **：** 每个人都有相同的效用函数，边际产量递减

>> ○ 由于边际产量递减，它认为财富应该从富人流向穷人

>> ○（注）似乎从根本上不相信市场这只看不见的手

> ③自由主义（自由主义）

>> ○ 政府应该选择“无知之幕”背后的公平第三方会选择的政策的政治哲学

>> ○ 基于约翰·罗尔斯的契约正义理论

>> ○ **原则1.** Maxi-min准则（maxi-min criteria）**：**最大化社会中最贫困人口的福利

>> ○ **原则2.** 社会保险（社会保险）**：**保护公民免受灾害风险

> ④ 自由主义（libertarianism）>> ○ 政府只应惩罚犯罪和执行契约，而不应重新分配收入的政治哲学

>> ○ 自由主义者认为机会平等（程序平等）比结果平等更重要

 ⑸ 福利项目

> ① 实物转移（in-kindtransfer）**：**以商品或服务而非现金的形式向贫困人口提供援助

> ② 经济状况调查计划（means-testedprogram）**：**当个人或家庭收入低于一定水平时提供福利

> ③ 累进税制

> ④ 负所得税（负所得税）

>> ○ 政府向低收入家庭提供补贴而不是征税的制度

>> ○ 优势**：** 受益人可以毫无耻辱地获得福利

>> ○ 缺点**：**成本过高；不是根本解决办法

> ⑤ 社会保障（社会保障）**：**通过工资税提供退休后的收入

> ⑥ 失业保险（失业保险）**：**为失业工人提供一部分工资，直至他们找到新工作

> ⑦ 健康保险

>> ○ 个人每年支付固定费用，并承诺支付以后需要的大部分医疗费用

>> ○ 如果完全自愿加入健康保险，健康人就不会加入，保费就会上涨

>> ○ 与私人健康保险相比，雇主健康保险的自愿性较低，因此保费较低

>> ○ 患者保护和平价医疗法案（奥巴马医改）

>>> ○ 也称为 PPACA（患者保护和平价医疗法案）或 ACA

>>> ○ 重点支持未参保人群并降低医疗费用

 ⑹ 福利计划的局限性

> ① 不能从根本上消除贫困

> ②经济效率整体下降**：**高收入者积极性降低、低收入者“福利依赖”等。
