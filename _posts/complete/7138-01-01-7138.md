## **1.快捷方式**

○ `fn`：特殊字符表

○ `fn + delete`：删除

○ `fn + f`：全屏查看浏览器

○ `caps lock`：韩语和英语输入之间切换

○ `control + command + f`：全屏

○ `command + q`：退出所有前台应用程序

○ `command + w`：关闭活动的前台应用程序

○ `command + r`：刷新

○ `command + z / c / v`：撤消/复制/粘贴

○ `command + tab`：在不同类型的程序窗口之间切换

○ `command + ~`：同一程序的窗口之间切换

○ `command + control + space`：特殊字符（字符查看器）

○ `command + shift + 5`：屏幕截图

○ `command + shift + return`：从 PowerPoint 中的第一张幻灯片开始播放幻灯片

○ `command + return`：从 PowerPoint 中的当前幻灯片播放幻灯片

○ `command + F3`：通过最小化所有窗口来显示桌面

○ `command + 1 (or 2, 3, ...)`：切换到网络浏览器中的选项卡 1（或 2、3、...）

○ 用两根手指在触摸板上拖动：滚动

○ 用两根手指在触摸板上单击：显示上下文菜单

○ `fn + ↑`：向上翻页

○ `fn + ↓`：向下翻页

○ `fn + ←`：主页

○ `fn + →`：结束

○ `option + return`：输入汉字。注意：韩文jamo+汉字不能输入特殊字符

## 3. **故障排除** 

 ⑴ 仅使用 Command + Shift + 5 进行屏幕录制时，可能无法录制音频。

> ① 下载 BlackHole 16ch（解决声音问题）：<https://underclub.tistory.com/m/5376>

> ② 配置音频 MIDI 设置 → 创建多输出设备 → 用英文命名多输出设备

> ③ 菜单栏中的声音 → 输出：Multi-Output Device → 输入：BlackHole 16ch

> ④ 使用 QuickTime Player 录制屏幕：使用 BlackHole 16ch 麦克风

> ⑤ 设置 Google 会议

> ⑥ 变焦设置

> ⑦ 卸载BlackHole：<https://github.com/ExistentialAudio/BlackHole>

⑵ 在 Windows 上将 PPT 转换为 PDF 时，有时可能无法拖动。但是，在 Mac 上，将 PPT 转换为 PDF 会以允许拖动的方式保存它。

⑶ 将 GIF 文件插入 PowerPoint 时，将其另存为图像可能会导致仅保存单帧的问题。在这种情况下，将文件扩展名从 `.pptx` 重命名为 `.rar` 并使用 [WinRAR](https://extract.me/) ([ref](https://www.reddit.com/r/powerpoint/comments/lbf7pj/how_do_i_save_a_gif_thats_already_in_a_powerpoint/)) 打开它。
