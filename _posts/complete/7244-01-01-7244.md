## **1\.概述**

⑴ 签证

> ① 签证只是一种旅行证件，而I-94只是表明一个人身份的文件。

> ② 存在签证过期后仍不出境而非法滞留在美国的情况。

⑵ 永久居留权（绿卡）：建议在特朗普上任前获得永久居留权。

> ① 程序

>> ○ **方法一：** 咨询律师事务所 

>> ○ **方法2：** 通过获得博士学位获得美国绿卡有一个程序：除了通过律师事务所之外，还有更便宜的方式获得绿卡。

>> ○ 申请绿卡时，您必须同时提交 I-140 和 I-485 表格。有多种策略，例如在 I-140 批准后提交 I-485 或同时提交。

>> ○ 获得绿卡后，之前获得的签证不会自动过期。

>> ○ 如果您希望放弃绿卡，您必须填写 I-407 表格。

>> ○ 使用信用卡可以为获得绿卡做出积极贡献。

>> ○ 与政府关门相关：美国移民局是一个收费机构，因此即使在联邦政府关门期间，它也将继续其核心业务，例如申请受理、裁决、生物识别和面试。然而，根据其他机构（例如，DOL、E-Verify、海外签证签发等），可能会在各个阶段发生延迟或暂停。

> ② **类型1.** 就业绿卡

>> ○ 通过基于就业的优惠 1 (EB-1) 至 3 (EB-3) 获得绿卡时： 非法居留时间必须少于 180 天，直至提交 I-485。

>> ○ 对于就业优先2（EB-2）和3（EB-3），也需要劳工证流程，劳工证的批准大约需要一年时间。

>> ○ 在寻求基于就业的移民程序之前，保持另一个有效的移民身份非常重要。

> ③ **类型 2.** EB-2 国家利益豁免 (NIW)

>> ○ 与顶级 EB-1 相比，EB-2 国家利益豁免 (NIW) 的资格要求不太严格。

>> ○ EB-2 需要硕士学位或学士学位并附有高度荣誉的主要奖项。

>> ○ EB-2 NIW 不需要劳工认证流程或担保人。

>> ○ 除了在中国和印度出生的个人外，优先日期是当前的，允许同时提交I-140和I-485，这是一个显着的优势。

> ④ 优势

>> ○ 长期居留和就业权：即使没有 H-1 或 O-1 签证也可以获得就业机会。

>> ○ 担保配偶和未婚子女的能力：有资格邀请配偶和未婚子女。配偶一旦收到 EAD（就业授权文件）即可自由寻求就业机会。

>> ○ 社会保障福利：有资格享受社会保障福利。

>> ○ 公民资格： 必须满足持绿卡居住一段特定期限的居住要求。

>> ○ 考试资格：有资格参加美国注册会计师（CPA）、美国专利律师、美国房地产经纪人考试等考试。

>>> ○ **例外 1.** ASA（精算师协会会员）、FSA（精算师协会会员）：无需永久居留权。>>> ○ **例外 2.** 美国注册营养师营养师 (RDN) 认证：足以进行相关研究生学习，然后进行考试和培训。

>>> ○ **例外 3.** CFA

>>> ○ **例外 4.** ACCA

>> ○ 作为研究生的优势：增加获得各种奖学金的机会，降低教授资助拥有永久居留权的研究生的成本。

>>> ○ 示例：秋季和冬季学期的学费合计（即年度总学费）为居民学生 28,508 美元至 35,810 美元，非居民学生为 53,586 美元至 63,122 美元。

> ⑤ 缺点

>> ○ 连续居留要求：如果您离开美国超过一年，可能会被视为缺乏移民意图，您的永久居留权可能会被撤销。 

>>> ○ 您可以通过每年前往关岛或其他附近的美国领土一次来维持您的永久居留权。

>> ○ 纳税义务：永久居民必须为其全球收入缴纳美国税款，并且可能需要在韩国和美国双重征税。

>> ○ 与美国公民的区别：没有投票权，也没有资格申请需要高安全级别的工作。

⑶ 公民身份（公民身份）

> ① 规范获得公民身份的要求。

>> ○ **例1：**获得永久居留权后，必须工作一定期限等。

>> ○ **示例 2：** 如果与美国公民结婚：您必须维持婚姻 2-3 年，直至获得美国政府的最终批准。

>> ○ **示例3：** 从美国大学或研究生院毕业并在美国军队服役的情况

>> ○ 满足要求后，必须参加【美国公民考试】(https://share.google/dXXqP3RTBYdSsM78z)。

> ② 国籍法第15条：如果公民自愿获得外国国籍（例如美国国籍），则立即失去韩国国籍。

>> ○ 补充信息：韩国不允许双重国籍，除非您超过 65 岁或原本是美国公民。

>>> ○ 最初，由于军事义务，这一规定仅适用于男性，但根据平等原则，这一规定扩展到女性。

>>> ○ 因此，一些人选择放弃韩国国籍或重新申请永久居留权。

>> ○ 取得外国国籍的效果：立即生效。

>>> ○ **效果1：** 您不能使用韩国护照。

>>> ○ **效果 2：** 您失去韩国公民身份带来的福利。

>>> ○ **效果 3：** 免除兵役义务。

> ③ 国籍法第16条第1项：丧失国籍时，必须向法务大臣报告。

>> ○ 未及时报告丧失国籍不会受到实质性处罚。

> ④ 公民身份的好处

>> ○ 美国公民有资格享受美国社会保障福利。

>> ○ 他们更容易获得商业用途的银行贷款。

>> ○ 对韩国的义务不再适用：例如纳税义务。

>> ○ 在美国投票的机会。

> ⑤ 出入境检查表

>> ○ 海外移民报告

>>> ○ 必须直接通过*海外同胞办公室（Jaeoe Dongpo Agency）填写

>>> ○ 稍后只能在线收到移民报告确认文件

>> ○ 外汇银行指定：完成移民报告后即可申请

>> ○ 国民年金一次性退还：完成移民报告后即可申请

>> ○ 指定外汇银行（在线或通过访问分行）- 在以下情况下需要：>>> ○ 汇款金额超过 5,000 美元且无证明文件时

>>> ○ 为海外居留或永久移民汇款时

>> ○ 通过移动应用程序或网站关闭未使用的银行账户

>>> ○ 可以使用名为 *Account Info* 的应用程序关闭

>>> ○ 如果无法通过应用程序关闭账户，您必须直接联系相关金融机构

>> ○ 剩余银行账户报告

>>> ○ 移民美国时，如果您的任何帐户的余额曾经超过10,000美元，则需要报告

>> ○ **区别5.** J1签证有5年限制，这使得从J1转H1变得困难

> ○ **H1签证和绿卡之间的区别**

>> ○ **区别一：** H1签证由特定雇主签发，取决于就业状况，而绿卡无论就业状况如何仍然有效。

 ○ H2：从事美国无法提供的农业或其他服务的临时工

 ○ H3：外星人实习生

> ○ H-3 签证的有效期最长可达 24 个月。然而，如果满24个月，有规定要求个人返回自己的祖国并停留6个月。因此，建议申请期限为23个月。 

 ○ H4：H1、H2 或 H3 的受抚养配偶或子女

 ○ HP：人道主义假释临时假释进入美国

 ○ I：国外信息媒体或家属代表

> ○ 按签证。持续时间：5年。

 ○ **J1**：交换访问国际学生（交换签证）

> ○ 美国国务院颁发

> ○ 对象：赴美从事博士后、研究人员等的外国博士毕业生，非旅游签证

> ○ **J1 和 OPT**

>> ○ 使用 OPT 后很难获得实习 J-1 身份，因为国务院对此有严格监管。

> ○ **J1与F1的区别**

>> ○ 从F1签证（如美国博士）转H1很容易，但从J1签证（如美国博士后）转H1就很难了

>>> ○ 原因：J1签证有**5年限制** + 签证到期后，必须回国2年>> ○ 从一所美国大学转SEVIS到另一所美国大学时，F1签证仍然可以使用，但J1签证必须更新。

>>> ○ 由于签证是旅行证件，如果您不打算在 J1 签证到期前离开美国，则无需重新申请。

> ○ **J1和H1之间的区别**
（[参考]（https://m.hibrain.net/braincafe/cafes/38/posts/208/articles/256986?pagekey=256986&listType=TOTAL&pagesize=10&sortType=RDT&limit=25&displayType=QNA&siteid=1&page=1））

>> ○ **区别1.** 大学博士后经常使用J1而不是H1的原因

>>> ○ **原因1.** J1进程更快

>>> ○ **原因2.** J1博士后需要支付的工资较少：J1的最低工资比H1低

>>> ○ **原因3.** 对于J1，健康保险或搬家费用的支持较少

>> ○ **区别2.** 申请永久居留权的可能性：虽然J1或H1签证持有者在技术上可以申请永久居留权，但由于它们是非移民签证，未能获得居留权可能会使未来的签证签发变得复杂

>>> ○ 对于J1→H1B或J1→永久居留权，需要J1豁免

>>> ○ 收到J1后有一段时间才能申请豁免

>>> ○ 如果在 J1 豁免后 H1B 发放或永久居留权被拒绝，则有可能成为非法移民

>> ○ **差异3.** J1不缴纳工资所得税

>> ○ **区别4.** J1到期后，有义务回国2年左右，然后才能再次出国

>> ○ **区别5.** J1签证有5年限制，这使得从J1转H1变得困难

> ○ 自2024年12月起，取消科学家和工程师的J1签证回国居住要求。但是，这不适用于获得政府奖学金的 J1 签证持有者。

○ J2：交流访问学者的配偶和子女

> ○ 居留期限与受益人J-1签证期限相同，不可延期居留

> ○ 如果有配偶，保证有在美国合法学习或工作的机会

 ○ K1：美国公民的未婚夫或未婚妻

 ○ K2：美国公民的未婚夫或未婚妻的孩子

 ○ K3：美国公民的配偶，是在美国提交的 I-130 申请（《生命法》）的受益人

 ○ M1：职业或非学术机构的学生，未参加语言培训课程

 ○ NATO 1-7：陪同北约成员的代表、工作人员、家属、专家雇员和平民

 ○ N8：特殊移民的父母

 ○ N9：特殊移民的孩子

 ○ OT：其他签证类型，包括国际学生具体身份未知

 ○ **O1**：在科学、艺术、教育、商业或体育方面具有非凡能力的外国人

> ○ 美国政府只发给科学、艺术、体育等特殊领域的人才的签证，范围很广，甚至包括厨师。我的一位朋友从斯坦福大学博士课程毕业，获得了 O1 签证并将其用作工作签证。

> ○ 仅需6个月即可申请绿卡

## **5。申请方法**（申请人↔︎入国管理局）

⑴ **方式一：** 使馆面试后签发

> ① **第一步：** 获得出国留学学校的【录取通知书（I-20）】(https://jb243.github.io/pages/2194#5-after-admission)

> ② **第 2 步：** 支付 I-901 SEVIS（学生和交流访问者信息服务）费用

> ③ **第三步：** 【签证面试】(https://jb243.github.io/pages/2193)

> ④ **步骤4：** 【签证签发确认】(https://ceac.state.gov/CEACStatTracker/Status.aspx)：面谈后1至3天即可确认签发

> ⑤ **步骤5：** 签证签发（[ref](https://ilyanglogis.co.kr/)）：面试后大约一周左右即可收到附有签证的护照

> ⑥ 拥有国立国际教育学院的经济担保证明，更容易获得签证

> ⑦ 全额资助的学生可能会更容易获得签证

⑵ **方法二：** 紧急申请后提交面试豁免文件并发出预约日期

> ① 紧急申请美国签证并不能免除您的面试

⑶ **方法三：** 旅游签证

> ① 代理机构：[https://cibtvisas.com/](https://cibtvisas.com/)（美国公民网站）

⑷ **方法4：** ESTA（Electronic System for Travel Authorization）：Visa Waiver Program（短期免签证入境计划）

> ① 概述

>> ○ 美国政府开发的在线申请系统

>> ○ 简单的旅行授权批准后即可进入美国

> ② 要求>> ○ 在美国停留少于90天

>> ○ 包括韩国在内的某些国家的公民无需申请签证即可进入美国旅游、商务或过境
